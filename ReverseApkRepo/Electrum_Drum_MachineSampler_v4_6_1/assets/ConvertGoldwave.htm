Converting samples to work with Electrum Drum using Goldwave

If we use Goldwave software, we can convert massive amounts of samples all at once using it's batch processing feature. The following is a small tutorial on how to do that.

The other nice thing about Goldwave is you can perform this batch processing using the unregistered version, it works just fine.

First, here we can see I have a folder with lots of subfolders with various samples in them. These samples are actually from July 2009 Computer Music magazine DVD.

So we start Goldwave and go to File->Batch Processing. You will then get this dialog:

 

Click the Add Folder button and add the root folder that contains all the samples you want to convert. I added the root of where all my samples were.

Click "Add Folder" and I selected my root folder, and also told it to include all subfolders.

On the "Convert" tab, change the settings to match what the settings show below

Now we also want to make sure that the samples are loud enough and that they don't have too much blank (empty) sound at their beginning or end. So select the "Process" tab, and then add the following two effects:

 

Now finally click the "Folder" tab and select the output directory. You can also preserve the input folder structure if you want.

 

And that's it! Now you can press "Begin" and it will convert all the files it finds, any format. The nice thing about this also is Goldwave handles any errors automatically for you. It will also convert MP3, AIFF, or whatever it finds, to the format that Electrum Drum wants.
