PicsPlay 사용설명서

PicsPlay 로 생생한 감동을 사진에 담아 보세요!
지금부터 PicsPlay 의 다양한 기능을 하나씩 알려드리겠습니다.

1. 메인 화면(Main Screen)


메인 화면에는 하단의 5개 ①메뉴 버튼-카메라(Camera) / 필터효과(Preset) / 도구(Tool) / 색상(Color) / 추가기능(Add-ons)과 ②되돌리기(Reset) 버튼이 있습니다. 메뉴 버튼을 누르면 각 메뉴의 세부 항목이 팝업으로 표시됩니다.

작업 중 되돌리기(Reset) 버튼을 눌러 언제든지 사진을 처음 상태로 되돌릴 수 있습니다.

모든 작업을 마친 후에는 하단 우측의 ③저장(Save) 버튼을 눌러 수정된 사진을 저장할 수 있습니다.

 

2. 카메라(Camera)메뉴

(1)사진 불러오기(Load a photo) : 갤러리에서 사진을 선택하여 불러올 수 있습니다.

(2)사진 촬영하기(Take a photo) : 카메라를 실행하여 사진을 촬영할 수 있습니다. 촬영한 사진은 자동으로 PicsPlay 로 불러옵니다.

※ 사진을 촬영하거나 불러오면 3가지 크기 중 용도에 맞는 사진 크기를 선택할 수 있습니다.

최소해상도 Fast (640x480) :모바일 기기 간 전송에 가장 최적화된 크기

표준해상도 Normal (1024x768) : 이메일, 블로그, 웹사이트 사용 시 최적화된 크기

최대해상도 Maximum (2048x1536) : 최고화질. 단, 처리 시간이 늘어나고 많은 메모리와 저장 공간을 사용

(3)사용설명서(User manual) : 본 사용설명서를 열어 봅니다.

(4)설정(Setting) : 사진 저장 위치 선택, 동영상 튜토리얼, PicsPlay 의 페이스북 페이지 및 개발사(JellyBus) 홈페이지 등을 선택할 수 있습니다.
