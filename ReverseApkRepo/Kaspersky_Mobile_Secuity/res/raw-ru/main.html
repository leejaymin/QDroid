Антивирус

Антивирус позволяет обнаруживать и устранять угрозы на вашем устройстве, используя антивирусные базы программы и дополнительно облачный сервис Kaspersky Security Network. В состав Антивируса входят следующие компоненты: защита, проверка и обновление.

Защита позволяет обнаруживать угрозы в открытых файлах, а также проверять новые программы и предотвращать заражение устройства в режиме реального времени.

Проверка запускается по требованию для всей файловой системы, оперативной памяти или папки. Полная проверка позволяет проверить на наличие вредоносных объектов всю файловую систему устройства, а проверка папки – конкретную папку. Полная проверка и проверка папки обнаруживают угрозы в файлах, которые установлены и не открыты, а также угрозы в файлах, которые открыты в данный момент. Проверка памяти позволяет обнаружить угрозы только в тех файлах, которые открыты в данный момент.

Обновление позволяет загружать новые антивирусные базы и версии программы.

Типовые задачи:

Проверка всей файловой системы устройства

В главном окне Kaspersky Mobile Security нажмите Антивирус > Проверка > Полная проверка.

Проверка папки

  1. В главном окне Kaspersky Mobile Security нажмите Антивирус > Проверка > Проверка папки.
  2. Выберите папку для проверки.

Проверка памяти устройства

  1. В главном окне Kaspersky Mobile Security нажмите Антивирус > Проверка > Проверка памяти.

Изменение режима защиты устройства

  1. В главном окне Kaspersky Mobile Security нажмите Антивирус > Настройка.
  2. Выберите режим защиты. Для этого выберите значение для параметра Режим защиты.

Проверка по расписанию

  1. В главном окне Kaspersky Mobile Security нажмите Антивирус > Настройка > Параметры проверки.
  2. Выберите периодичность запуска проверки. Для этого выберите значение для параметра Расписание.
  3. Выберите день и время запуска проверки. Для этого выберите значения для параметров День запуска и Время запуска.

Обновление антивирусных баз и версий программы

В главном окне Kaspersky Mobile Security нажмите Антивирус > Обновление.

Обновление баз программы и версий по расписанию

  1. В главном окне Kaspersky Mobile Security нажмите Антивирус > Настройка > Параметры обновления.
  2. Выберите периодичность запуска обновления. Для этого установите значение для параметра Расписание.
  3. Выберите день и время запуска обновления. Для этого установите значение для параметров День запуска и Время запуска.

Личные контакты

Личные контакты позволяют скрывать ваши конфиденциальные контакты, а также историю разговоров и SMS-переписку с этими контактами.

Конфиденциальные контакты хранятся в списке Скрываемые контакты.

Для конфиденциальных контактов программа позволяет скрывать следующую информацию:

  • записи в Контактах;
  • прочитанные входящие SMS, переданные и черновики SMS;
  • записи в журнале вызовов;
  • входящие вызовы и уведомления о новых SMS.

Когда скрытие информации включено, программа блокирует сигнал о получении нового SMS от конфиденциального номера и скрывает сообщение в списке входящих. Программа блокирует входящий вызов от конфиденциального номера и не отображает на экране информации о его приеме. Звонящий в этом случае получает сигнал "занято". Чтобы посмотреть вызовы и SMS, поступившие в тот период, когда скрытие конфиденциальной информации было включено, требуется отключить скрытие информации. Если вы повторно включаете скрытие информации, конфиденциальная информация снова становится скрытой.

Вы можете включить скрытие информации в программе или дистанционно с помощью SMS-команды, отправленной с другого мобильного устройства. Вы можете отправить стандартное SMS со специальным текстом или отправить SMS из Kaspersky Mobile Security, если программа установлена на другом устройстве.

На своем устройстве вы можете запретить или разрешить дистанционный запуск скрытия информации. Если дистанционный запуск скрытия информации запрещен, то эту функцию невозможно запустить дистанционно с помощью SMS-команды.

Вы можете отключить скрытие информации только в программе, установленной на вашем устройстве.

Типовые задачи:

Скрытие информации для контактов

  1. В главном окне Kaspersky Mobile Security нажмите Личные контакты.
  2. Сформируйте список скрываемых контактов, если он еще не сформирован. В списке хранятся контакты, о которых вы хотите скрывать информацию. Для этого нажмите Скрываемые контакты > Добавить.
  3. Вернитесь в главное окно Kaspersky Mobile Security > Личные контакты и установите переключатель Скрыть информацию в состояние включено.

Дистанционный запуск скрытия информации

На другом мобильном устройстве выполните одно из следующих действий:

  • Если на другом устройстве установлена программа Kaspersky Mobile Security, с помощью функции отправки SMS-команды сформируйте и отправьте команду на свое устройство.
  • Cоздайте и отправьте на ваше устройство SMS со специальным текстом hide:<код>, где <код> – это секретный код, заданный на вашем устройстве.

Отправка SMS оплачивается по тарифу оператора сотовой связи, используемого на другом мобильном устройстве.

Создание списка скрываемых контактов

  1. В главном окне Kaspersky Mobile Security нажмите Личные контакты > Скрываемые контакты.
  2. Выполните одно из следующих действий:
    • Чтобы добавить контакт из записной книги устройства, нажмите Добавить > Контакт.
    • Чтобы добавить телефонный номер, нажмите Добавить > Номер телефона.

Анти-Вор

Анти-Вор защищает от несанкционированного доступа вашу информацию на устройстве и помогает найти устройство в случае его кражи или потери.

С помощью SMS-команд, отправленных с другого мобильного устройства, Анти-Вор позволяет дистанционно заблокировать ваше устройство, удалить хранящуюся на нем информацию или получить его местоположение на карте. Также Анти-Вор позволяет заблокировать ваше устройство в случае смены SIM-карты или включении без нее.

Для запуска функций Анти-Вора используется специальная SMS-команда. С другого мобильного устройства вы можете отправить стандартное SMS со специальным текстом или, если на другом мобильном устройстве установлена Kaspersky Mobile Security, отправить SMS из программы. Для отправки SMS-команды требуется знать секретный код программы Kaspersky Mobile Security, установленной на вашем устройстве.

На своем устройстве вы можете запретить или разрешить дистанционный запуск функций Анти-Вора. Если запуск функции Анти-Вора запрещен, то эту функцию невозможно запускать дистанционно с помощью SMS-команды.

Типовые задачи:

Разрешение дистанционно блокировать устройство

  1. В главном окне Kaspersky Mobile Security нажмите Анти-Вор > Блокирование.
  2. Установите флажок Включить Блокирование.
  3. Если требуется, в поле Текст при блокировании введите текст сообщения, которое будет отображаться на экране заблокированного устройства.

Разрешение дистанционно удалять данные

  1. В главном окне Kaspersky Mobile Security нажмите Анти-Вор > Удаление данных.
  2. Выберите информацию, которая будет удалена после получения дистанционной команды:
    • Чтобы дистанционно удалить выбранные данные, сначала установите флажок Удалить выбранные данные. Чтобы удалить персональные данные (например, контакты, SMS-переписку и календарь), установите флажок Персональные данные. Чтобы дистанционно удалить папки, нажмите Добавить папку и выберите необходимые папки.
    • Чтобы дистанционно удалить все данные и вернуть устройство к заводским настройкам, установите флажок Удалить все данные.

Разрешение дистанционно определять местоположение устройства

  1. В главном окне Kaspersky Mobile Security нажмите Анти-Вор > Поиск.
  2. Установите флажок Включить Поиск.
  3. Если требуется, укажите адрес электронной почты, на который вы хотите получить местоположение вашего устройства.

Разрешение контролировать замену SIM-карты

  1. В главном окне Kaspersky Mobile Security нажмите Анти-Вор > SIM-Контроль.
  2. Установите флажок Включить SIM-Контроль.
  3. Укажите ваш дополнительный номер телефона или адрес электронной почты, чтобы получать новый номер телефона при замене SIM-карты по SMS или по электронной почте.
  4. Если требуется, установите флажок Блокировать, чтобы заблокировать устройство при замене SIM-карты или включении без нее. Укажите текст сообщения, которое появится на экране заблокированного устройства.

Дистанционный запуск функций Анти-Вора с другого устройства

На другом мобильном устройстве выполните одно из следующих действий:

  • Если на другом устройстве установлена программа Kaspersky Mobile Security, с помощью функции отправки SMS-команды сформируйте и отправьте команду на свое устройство. Для формирования SMS-команды требуется секретный код Kaspersky Mobile Security, установленной на вашем устройстве.
  • Отправьте SMS на ваше устройство со следующим специальным текстом:
    • lock: <код> – для блокирования устройства;
    • wipe: <код> – для удаления выбранных персональных данных с устройства;
    • fullreset: <код> – для удаления всех данных с устройства;
    • locate: <код> – для определения местоположения устройства.

    <Код> – это секретный код Kaspersky Mobile Security, заданный на вашем устройстве.

Отправка SMS оплачивается согласно тарифу оператора сотовой связи, используемого на другом мобильном устройстве.

Фильтр вызовов и SMS

Фильтр вызовов и SMS позволяет вам избавиться от нежелательных входящих вызовов и SMS. Программа фильтрует вызовы и SMS на основе разрешенных и запрещенных контактов и в соответствии с выбранным режимом фильтрации.

Типовые задачи:

Избавление от нежелательных входящих вызовов и SMS

  1. Определитесь, от каких контактов вы хотите блокировать вызовы и SMS, а от каких контактов разрешать. Для этого выберите режим фильтрации. В главном окне Kaspersky Mobile Security нажмите Фильтр вызовов и SMS > Режим: <текущий режим фильтрации>.
  2. Выберите значение для параметра Режим Фильтра вызовов и SMS:
    • Чтобы блокировать вызовы и SMS только от запрещенных контактов и пропускать от остальных номеров, выберите Запрещенные контакты.
    • Чтобы блокировать вызовы и SMS от всех контактов и пропускать только от разрешенных контактов, выберите Разрешенные контакты.
    • Чтобы блокировать вызовы и SMS от запрещенных контактов и пропускать от разрешенных контактов, выберите Стандартный.
  3. Если потребуется, включите дополнительные параметры фильтрации.

    Например, если вы выбрали режим фильтрации Разрешенные контакты или Стандартный, вы можете автоматически разрешить вызовы и SMS от контактов из телефонной книги устройства, независимо выбранного режима фильтрации. Для этого установите флажок Разрешать Контакты.

    Если вы выбрали режим фильтрации Запрещенные контакты или Стандартный, вы можете автоматически запретить SMS с номеров, содержащих буквы, независимо от выбранного режима фильтрации. Для этого установите флажок Блокировать нечисловые номера.

  4. В зависимости от выбранного режима сформируйте список разрешенных и запрещенных контактов:
    • Чтобы сформировать список запрещенных контактов, вернитесь в главное окно В главном окне Kaspersky Mobile Security > Фильтр вызовов и SMS > Запрещенные контакты.
    • Чтобы сформировать список разрешенных контактов, вернитесь в главное окно В главном окне Kaspersky Mobile Security > Фильтр вызовов и SMS > Разрешенные контакты.

Добавление контакта в список разрешенных контактов

  1. В главном окне Kaspersky Mobile Security нажмите Фильтр вызовов и SMS > Разрешенные контакты.
  2. Нажмите Добавить.
  3. Выберите тип тех событий, которые вы хотите получать от контакта.
  4. Введите номер телефона контакта, от которого вы хотите получать входящую информацию. Для этого заполните поле Разрешенный номер телефона.

    Вы можете добавить номер абонента в список одним из следующих способов: указать номер вручную в поле ввода, выбрать вызов от абонента из списка вызовов или выбрать SMS от абонента из списка входящих сообщений. Чтобы добавить номер из списка вызовов или из списка сообщений, нажмите .

    Номер может начинаться с цифры, с буквы или со знака "+" и может содержать цифры и (или) буквы. При вводе номера можно также использовать маски "*" и "?" (где "*" – любое количество символов, а "?" – один любой символ). Например, если номер *1234? добавлен в список разрешенных контактов, то Фильтр вызовов и SMS разрешает вызовы или SMS с номера, в котором после цифр 1234 следует любой символ.

  5. Если вы выбрали SMS как тип событий, введите ключевую фразу, которая указывает на то, что полученное SMS является желательным. Для этого заполните поле Разрешенный текст. Фильтр вызовов и SMS разрешает только те SMS от этого номера, которые содержат ключевую фразу, и запрещает все остальные SMS с этого номера.

    Если вы хотите получать SMS с любым содержанием от этого контакта, оставьте поле Разрешенный текст пустым.

Добавление контакта в список запрещенных контактов

  1. В главном окне Kaspersky Mobile Security нажмите Фильтр вызовов и SMS > Запрещенные контакты.
  2. Нажмите Добавить.
  3. Выберите тип тех событий, которые вы считаете нежелательными от этого контакта.
  4. Введите номер телефона запрещенного контакта. Для этого заполните поле Запрещенный номер телефона.

    Вы можете добавить номер абонента в список одним из следующих способов: указать номер вручную в поле ввода, выбрать вызов от абонента из списка вызовов или выбрать SMS от абонента из списка входящих сообщений. Чтобы добавить номер из списка вызовов или из списка сообщений, нажмите .

    Номер может начинаться с цифры, с буквы или со знака "+" и может содержать цифры и (или) буквы. При вводе номера можно также использовать маски "*" и "?" (где "*" – любое количество символов, а "?" – один любой символ). Например, если номер *1234? добавлен в список запрещенных контактов, то Фильтр вызовов и SMS запрещает вызовы или SMS с номера, в котором после цифр 1234 следует любой символ.

  5. Если вы выбрали SMS как тип событий, введите ключевую фразу, которая указывает на то, что полученное SMS является нежелательным. Для этого заполните поле Запрещенный текст. Фильтр вызовов и SMS блокирует только те SMS от этого номера, которые содержат ключевую фразу, и разрешает все остальные SMS с этого номера.

    Если вы не хотите получать SMS с любым содержанием от этого контакта, оставьте поле Запрещенный текст пустым.

Веб-Фильтр

Веб-Фильтр блокирует вредоносные веб-сайты, цель которых – распространить вредоносный код, а также поддельные (фишинговые) веб-сайты, цель которых – украсть ваши конфиденциальные данные, такие как пароли от онлайн-банков, онлайн-аукционов или систем оплаты, и получить доступ к вашим финансовым счетам.

Веб-Фильтр проверяет веб-сайт до его открытия, используя для этого облачный сервис Kaspersky Security Network (специальный онлайн-сервис "Лаборатории Касперского", который содержит информацию о надежности файлов, программ и интернет-ресурсов).

Веб-Фильтр проверяет веб-сайты только в стандартном браузере Android и не проверяет их в других браузерах.

При включении Веб-Фильтр автоматически проверяет, какой браузер установлен по умолчанию. Если Веб-Фильтр не поддерживает текущий браузер, установленный по умолчанию, Веб-Фильтр предложит вам изменить его на стандартный браузер Android.

Для постоянной проверки веб-сайтов рекомендуется установить стандартный браузер Android в качестве браузера по умолчанию.

Если вы не хотите изменять текущий браузер по умолчанию, для безопасной работы с персональными данными в сети запускайте стандартный браузер Android.

Вы можете запустить стандартный браузер Android из Kaspersky Mobile Security или из меню приложений вашего устройства. Если Веб-Фильтр включен, он будет блокировать поддельные веб-сайты до их открытия.

Типовые задачи:

Включение Веб-Фильтра для постоянной проверки веб-сайтов

  1. В главном окне Kaspersky Mobile Security нажмите Веб-Фильтр.
  2. Установите переключатель Включить Веб-Фильтр в состояние включено.
  3. НажмитеИзменить браузер (кнопка отображается, если Веб-Фильтр включен и не проверяет веб-адреса в текущем браузере по умолчанию).
  4. Следуйте шагам мастера по установке браузера по умолчанию.

Безопасная работа с персональными данными

В главном окне Kaspersky Mobile Security нажмите Веб-Фильтр > Открыть браузер.
