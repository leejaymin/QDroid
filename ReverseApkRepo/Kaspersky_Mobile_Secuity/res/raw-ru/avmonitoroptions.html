Параметры защиты

Вы можете выбрать типы тех файлов, которые программа проверяет на наличие вредоносных объектов, и определить действие над файлами, в которых программа обнаружила угрозы.

Блок параметров доступен, если для параметра Режим защиты выбран вариант Расширенный.

Тип проверяемых файлов

Выбор типа файлов, которые защита проверяет на наличие вредоносных объектов. Защита проверяет файлы выбранного типа при любом действии над ними (например, при их открытии, копировании, запуске, перемещении, сохранении или изменении). Вы можете выбрать один из следующих типов файлов:

  • Все файлы – программа проверяет файлы всех типов (например, музыкальные, текстовые, системные или исполняемые).
  • Только исполняемые – программа проверяет только исполняемые файлы (то есть программы), например, файлы формата APK.

Действие при обнаружении угрозы

Выбор действия, которое программа выполняет при обнаружении угрозы. Вы можете выбрать одно из следующих действий:

  • Удалить – программа удаляет зараженный файл.
  • Пропустить – программа не выполняет никаких действий над зараженным файлом и не удаляет его с устройства.

Независимо от выбранного действия программа уведомляет вас о том, что обнаружена угроза.
