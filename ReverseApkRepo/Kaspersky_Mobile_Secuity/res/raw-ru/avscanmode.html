Область проверки

Вы можете выбрать типы тех файлов, которые программа будет проверять на наличие вредоносных объектов.

Когда вы выбираете типы файлов для проверки, помните следующее:

  • Вредоносный код может быть в любых файлах и чаще всего он содержится в исполняемых файлах (например, файлах формата APK).
  • Вредоносные объекты могут быть скрыты в архивах. Чтобы обнаружить вредоносный код в архивах, программа должна их распаковать, что может снизить скорость проверки устройства.

Тип проверяемых файлов

Выбор файлов, которые программа анализирует во время полной проверки файловой системы устройства. Вы можете выбрать для проверки файлы следующих типов:

  • Все файлы – программа проверяет файлы всех типов (например, музыкальные, текстовые, системные или исполняемые). Параметр выбран по умолчанию.
  • Только исполняемые – программа проверяет только исполняемые файлы (то есть программы или установщики программ), например, файлы формата APK, и не проверяет другие файлы. При выборе этого варианта скорость проверки устройства увеличивается.

Проверять архивы

Включение / отключение проверки содержимого архивов на наличие вредоносного кода. Программа позволяет проверять архивы следующих форматов: ZIP, JAR, JAD, SIS, SISX, CAB и APK. Программа анализирует архивы тех форматов, которые применяются в Android и других мобильных операционных системах. Это позволяет, например, проверять карту памяти, которая используется в нескольких устройствах.

Когда программа проверяет архивы, она их распаковывает. Это может значительно снизить скорость выполнения проверки устройства.

Если вы отключили проверку архивов и выбрали для проверки программы или установщики программ (исполняемые файлы), то согласно выбранным настройкам Kaspersky Mobile Security проверяет только исполняемые файлы. Если они запакованы в архивы (например, архивы форматов ZIP), программа их не проверяет.
