Удаление данных

Удаление данных позволяет вам дистанционно при помощи SMS-команд удалить информацию с вашего устройства, чтобы она не попала в чужие руки. Вы можете удалить персональную информацию (например, контакты, переписку или данные об учетной записи Google), заранее выбранные папки или все данные, включая данные на карте памяти.

Для удаления данных на вашем устройстве вам нужно отправить следующие SMS-команды:

  • wipe: <код> – для удаления конкретных данных;
  • fullreset: <код> – для удаления всех данных.

<Код> – это секретный код Kaspersky Mobile Security на вашем устройстве.

Отправка SMS-команды на ваше устройство оплачивается согласно тарифу оператора сотовой связи, используемого на другом мобильном устройстве.

Вы можете разрешить или запретить дистанционный запуск удаления конкретных данных или всех данных на вашем устройстве, а также выбрать конкретную информацию для удаления.

Удалить выбранные данные

Разрешение / запрет удалять конкретные данные с устройства дистанционно (при помощи специальной SMS-команды). Вы можете выбрать для удаления персональную информацию, а также конкретные папки.

Если функция разрешена на устройстве, то после того, как программа получит SMS-команду удаления конкретных данных, она удалит конкретные выбранные данные на устройстве и сообщит об этом в ответном SMS.

Отправка ответной SMS оплачивается согласно тарифу вашего оператора сотовой связи.

Если функция запрещена на устройстве, то программа не сможет выполнить SMS-команду удаления конкретных данных.

Удалить все данные

Разрешение / запрет удалить все данные с устройства дистанционно при помощи SMS-команды fullreset: <код> (где <код> – это секретный код Kaspersky Mobile Security на вашем устройстве).

Если функция разрешена на устройстве, то после того, как программа получит SMS-команду удалить все данные, она выполнит ее и сообщит об этом в ответном SMS.

Отправка ответной SMS оплачивается согласно тарифу вашего оператора сотовой связи.

Если функция запрещена на устройстве, то программа не сможет выполнить команду удалить все данные.

Удаляемая персональная информация

Вы можете выбрать конкретную информацию, которая будет удалена после получения SMS-команды wipe: <код> (где <код> – это секретный код Kaspersky Mobile Security на вашем устройстве).

Выбор конкретных данных для удаления доступен, если установлен флажок Удалить выбранные данные.

Персональные данные

Удаление записей в Контактах и на SIM-карте, всей истории SMS, журнала вызовов, календаря, параметров подключения к интернету, учетных записей пользователя (кроме учетной записи Google) после получения SMS-команды wipe: <код> (где <код> – это секретный код Kaspersky Mobile Security на вашем устройстве)

Если флажок установлен, после получения специальной SMS-команды программа удалит ваши персональные данные. Если флажок снят, программа не удалит ваши персональные данные при получении SMS-команды.

Папки

Список папок, которые будут удалены после после получения SMS-команды wipe: <код> (где <код> – это секретный код Kaspersky Mobile Security на вашем устройстве).

Вы можете добавить папки, сохраненные на карте памяти. Также при помощи контекстного меню вы можете удалить папки из списка.
