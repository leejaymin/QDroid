Параметры проверки

Для проверки устройства по требованию вы можете выбрать типы тех файлов, которые программа будет проверять на наличие вредоносных объектов, включить распаковку архивов, выбрать действие программы при обнаружении угрозы, а также настроить запуск проверки файловой системы по расписанию.

Область проверки

Вы можете выбрать типы тех файлов, которые программа будет проверять на наличие вредоносных объектов.

Действие при обнаружении угрозы

Вы можете выбрать действия над файлами, в которых программа обнаружила угрозы.

Проверка по расписанию

Вы можете настроить автоматический запуск полной проверки устройства по расписанию: выбрать периодичность запуска проверки, день и время запуска (если это необходимо).

Чтобы программа автоматически запускала проверку, устройство в это время должно быть включено.

Программа запускает автоматическую проверку по расписанию в фоновом режиме. При обнаружении угрозы программа выполняет действие, выбранное в параметрах проверки.

Расписание

Периодичность, в соответствии с которой программа будет автоматически запускать полную проверку файловой системы. Предусмотрены следующие варианты:

  • Раз в неделю – программа автоматически запускает проверку раз в неделю. Для этого варианта вы можете выбрать день и время начала проверки.
  • Раз в день – программа автоматически запускает проверку каждый день в указанное время. Для этого варианта вы можете выбрать время начала проверки.
  • Отключено – программа не выполняет автоматическую проверку по расписанию. При этом вы можете запускать проверку в любое удобное для вас время.

День запуска

День запуска проверки. Параметр доступен, если для параметра Расписание выбрано значение Раз в неделю.

Время запуска

Время запуска проверки. Параметр доступен, если для параметра Расписание выбрано значение Раз в день или Раз в неделю.
