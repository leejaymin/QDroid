Anti-Phishing: additional

On the Anti-Virus: Additional screen you can select Protection mode, adjust Protection settings, scan the device, start an anti-virus database update, and configure automatic starting of scheduled updates.

The Protection section allows you to select the mode used by Protection to track changes in the file system, switch to configure Protection settings, and scan device on request.

Protection mode.

The mode used by Protection to track changes in the file system in background mode and to scan it for malicious objects. The following modes are available:

  • Full – mode in which Protection scans all files being opened, saved, or run (including those stored on a memory card) for viruses; it also scans applications installed. Protection scans files when the user accesses them. Protection scans new applications right after they are installed.
  • Recommended – mode in which Protection only scans installed applications for viruses. Protection scans new applications right after they are installed.
  • Disabled – mode in which Protection is disabled.

Protection settings.

Switch to selection of file types to be scanned by the application for malicious objects while Protection is on.

Scan settings.

Scan on request settings. You are able to configure scans of archives and select files to scan. Furthermore, you can set the action carried out by the program in respect of malicious objects (for instance, disinfect or delete).

The Anti-virus database update section allows you to update the application's anti-virus databases and configure automatic starting of scheduled updates.

Start update.

Start the update of the application databases. The application will download application databases from the Kaspersky Lab server. Update information will be displayed on the screen.

Automatic update.

Navigate to automatic scheduled update configuration.
