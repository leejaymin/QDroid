Threat detected

The application prompts the user to select an action on files in which a threat is detected.

The application notifies the user of detected threats in the following circumstances:

  • if Anti-Virus detects a threat in a newly installed program when protection is enabled or during a scan of the device.
  • if Anti-Virus detects a threat in a file, and Anti-Virus has the value of the If disinfection fails setting set to Ask user (main window of Kaspersky Mobile Security > Anti-Virus > Settings > Scan settings > Action on threat detection).

Delete

The application deletes the infected object.

Skip

The application does not perform any actions on the infected file and does not delete it from the device.
