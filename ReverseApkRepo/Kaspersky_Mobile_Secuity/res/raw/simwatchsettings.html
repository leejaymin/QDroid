SIM Watch

SIM Watch helps you obtain the current phone number of the SIM card in the event that it is replaced, and lock the device if the SIM card is replaced or the device is activated without a SIM card.

You can obtain a new telephone number in a message sent to the phone number or email address that you specified.

To send an email message, the application sends an SMS to the dedicated number of the MTS mobile carrier (Russia). The cost of sending an SMS to the dedicated number depends on the charges of your mobile carrier. The MTS mobile carrier (Russia) then sends the current phone number to your email.

You can enable or disable SIM Watch, specify a phone number or email address to receive a new phone number in the event the SIM card is replaced, and enable Device Lock.

Enable SIM Watch

Allow / block the selected actions in the event the SIM card is replaced or the device is activated without it (for example, send a new phone number by SMS or email, or lock the device).

If the function is allowed on the device, the application performs the selected actions in the event the SIM card is replaced or the device is activated without it.

If the function is blocked on the device, the application does not control actions on the SIM card.

Send new number

You can specify a phone number and email address to receive a message with a new phone number if the SIM card is replaced.

To send an email message, the application sends an SMS to the dedicated number of the MTS mobile carrier (Russia). The cost of sending an SMS to the dedicated number depends on the charges of your mobile carrier. The MTS mobile carrier (Russia) then sends the current phone number to your email.

Phone number

The number to which the application sends an SMS message with the current number of the phone if the SIM card is replaced. The phone number may begin with a digit or with a "+", and must contain digits only.

Email address

The email address to which the application sends a message with the current number of the phone if the SIM card is replaced.

Locking the device

You can enable automatic Device Lock in the event the SIM card is replaced or the device is activated without it.

Block

Block device if the SIM card is replaced or if the device is activated without a SIM card. If the check box is selected, the application blocks the device if the SIM card is replaced or the device is activated without a SIM card. The application secret code would have to be entered to unlock the device. If the check box is unselected, the application does not block the device if the SIM card is replaced.

Text when locked

The text that is displayed on the screen of the blocked device. A standard message is displayed by default.
