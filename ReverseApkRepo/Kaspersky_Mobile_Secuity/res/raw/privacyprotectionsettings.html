Privacy Protection settings

You can select information that the application hides for confidential contacts, configure Privacy Protection automatically when the device switches to power-saving mode, and configure remote enabling of Privacy Protection by SMS command from another mobile device.

Information to be hidden

Information and events that the application hides for confidential contacts. You can select types of information to be hidden:

  • Contacts. If the check box is selected, the application hides information about confidential contacts in the device's phonebook, and does not show their numbers for incoming calls or in call history. If the check box is unselected, the application displays information about confidential contacts in the device's phonebook, and shows their numbers for incoming calls or in the call history.
  • SMS logs. If the check box is selected, the application hides incoming and forwarded SMS messages that have been read, and SMS drafts related to confidential contacts. If the check box is unselected, the application displays incoming and forwarded SMS messages that have been read, and SMS drafts related to confidential contacts.
  • Incoming SMS. If the check box is selected, the application blocks the signal and notification on receipt of a new SMS message from a confidential contact, and hides the message in the inbox. To view incoming SMS messages for the period of time when Privacy Protection was enabled, disable Privacy Protection. If the check box is unselected, the application does not block the signal and notification on receipt of a new SMS message from a confidential contact, and displays the message in the SMS inbox.
  • Call logs. If the check box is selected, the application hides call history for confidential contacts (for example, accepted, outgoing, or missed calls). If the check box is unselected, the application displays the call history for confidential contacts.
  • Incoming calls. If the check box is selected, the application blocks incoming calls from confidential contacts and hides on-screen information about the call. As a result, the caller receives a busy signal. To view incoming calls for the period of time when Privacy Protection was enabled, disable Privacy Protection. The application hides information about skipped calls in the call history if the Call logs check box is selected. If the Incoming calls check box is unselected, the application allows incoming calls from confidential contacts.

Automatic hiding

Privacy Protection can be enabled automatically once a specific amount of time has elapsed after the device switched to power-saving mode. The following options are available:

  • Disabled – the application does not enable Privacy Protection automatically. You can enable Privacy Protection from the application (main window of Kaspersky Mobile Security > Privacy Protection > Hide information) or from another device by sending an SMS command to your device.
  • No delay – automatically hide information immediately after the device switches to power-saving mode.
  • After 5 minutes, After 10 minutes, After 15 minutes, After 30 minutes – automatically hide information after the set time interval once the device has switched to power-saving mode.

Hide by SMS command

Allow / block the remote start of Privacy Protection on your device using the SMS command hide:<code> (where <code> is the secret code to Kaspersky Mobile Security on your device).

If the function is allowed on the device, when the device receives an SMS command, it enables Privacy Protection. If the function is blocked on the device, the application cannot execute the SMS command to enable Privacy Protection.
