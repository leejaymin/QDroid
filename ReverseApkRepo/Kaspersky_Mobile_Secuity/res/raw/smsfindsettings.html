Locate

Locate lets you use the SMS command locate:<code> to locate your device remotely and receive its geographical coordinates in an SMS message or email (<code> is the secret code to Kaspersky Mobile Security on your device).

To send an email message, the application sends an SMS to the dedicated number of the MTS mobile carrier (Russia). The cost of sending an SMS to the dedicated number depends on the charges of your mobile carrier. The MTS mobile carrier (Russia) then sends the current coordinates to your email.

The SMS reply is billed according to the price plan of your mobile carrier.

The application can locate the device using GPS or the mobile networks.

To provide GPS positioning, you must allow GPS use in the device settings. When the application receives a special SMS command, the GPS module will be activated. If the application cannot find the coordinates of the device using GPS, it uses the mobile networks to determine them.

You can allow or block the function to locate the device remotely, and specify an email address to receive the geographical coordinates of the device.

Enable Locate

Allow / block remote location of the device by SMS command.

If the function is allowed on the device, on receipt of the corresponding SMS command, the application sends a reply with the coordinates of the device by SMS to the device that issued the command, and to the specified email address.

Outgoing SMS messages are billed according to the price plan of your mobile service provider.

If the function is blocked on the device, the application cannot execute the command to locate the device.

Send device coordinates

By default, the application sends the device coordinates in an SMS reply to the number that issued the SMS command. You can additionally specify an email address to receive the location of the device.

Email address

The email address to which the application sends the coordinates of your device.
