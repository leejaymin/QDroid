Folder selection

You can select folders and add them to the list of folders to be deleted. When the application receives the SMS command wipe:<code>, it deletes these folders (<code> is the secret code to Kaspersky Mobile Security on your device).

The window displays the list of folders on the memory card. To add a folder, select it in the folder tree and tap the button to the right of the folder name.
