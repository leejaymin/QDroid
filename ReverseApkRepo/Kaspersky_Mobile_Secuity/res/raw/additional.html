Additional

You can do the following:

  • configure the receipt of notifications and the display of prompts by the application or the widget.
  • change the secret code.
  • view reports on the operation of the application (for example, reports on scans performed, threats detected, SMS messages, calls or websites blocked).
  • view information about the application.

Advanced settings

Notifications

Enable/disable the display of notifications about the operation of the application in the status line. For example, notifications that a threat has been detected or that the license has expired.

Sound

Enable/disable sound notifications about application runtime events. For example, the application issues a sound notification on detection of a threat.

Hints

Enable/disable the display of brief information about the component before configuration.

Widget

The color indicator of the widget on the main screen informs you about the protection status of your device and the hide status of confidential information, and allows you to configure the application settings.

A green shield indicates that the recommended or extended protection mode is enabled. A gray shield indicates that Protection is disabled. A green background indicates that confidential information is hidden; a gray background indicates that confidential information is displayed. An exclamation mark in a yellow triangle means that the license has expired.

You can allow/block displaying of the widget on the main screen of the device.

Changing the secret code

The secret code is set when the application runs for the first time, and is used to:

  • Prevents unauthorized access to application settings
  • Enables the remote start of Anti-Theft and Privacy Protection functions from another device

You can change the secret code.

Uninstalling Kaspersky Mobile Security

To uninstall Kaspersky Mobile Security, you have to enter the secret code to the application.

General information

Reports

Anti-Virus, Web Protection, and Call&SMS Filter events are logged in reports.

Reports are grouped by time of creation. You can choose to display reports for each component separately. Reports can contain up to 50 entries. Once the number of entires exceeds 50, older entries are deleted and replaced by new ones.
