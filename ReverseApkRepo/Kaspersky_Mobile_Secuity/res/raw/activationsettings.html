License

On the License screen, you can manage the application license.

The following items are accessible:

  • About license. View the following information about the license: number, type, days left until the expiry date, activation date, and serial number of the device.
  • Renewal. Go to renew your license.
  • Unsubscribe. Cancel automatic license renewal. If the subscription is canceled, the application continues to operate in full functionality mode until the current license for which payment was made expires. After that, the application functionality becomes limited, the application anti-virus databases are no longer updated. If you have canceled your subscription, you can renew it.

    This menu item is available if subscription is activated.
