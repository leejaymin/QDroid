Incoming call / incoming SMS

The application prompts you to select an action to perform on receipt of incoming SMS messages or on completion of incoming calls.

The application prompts you to select an action if the following conditions are met concurrently: Standard mode is set for Call&SMS Filter, and the number is not contained in the lists of blocked or allowed contacts.

Always block

The application flags calls (or SMS messages) from the contact as unwanted. The application adds the number of the contact from whom the call (or SMS message) was received to the list of blocked contacts.

If a call is received from the contact, the application blocks all subsequent incoming calls from it. But, if an SMS message is received from the contact, the application prompts you to select an action to perform on the incoming message.

If an SMS message is received from the contact, the application blocks it and all subsequent SMS messages from this contact. But, if a call is received from the contact, the application prompts you to select an action to perform on the incoming call.

Always allow

The application flags this call (or SMS message) from the contact as wanted. The application adds the number of the contact from which the call (or SMS message) was received to the list of allowed contacts.

If a call is received from the contact, the application allows all subsequent incoming calls from it. But, if an SMS message is received from the contact, the application prompts you to select an action to perform on the incoming message.

If an SMS message is received from the contact, the application allows it and all subsequent SMS messages from this contact. But, if a call is received from the contact, the application prompts you to select an action to perform on the incoming call.

Skip

The application skips calls (or SMS messages) from this contact. The application does not add the number of the contact from which the call (or SMS message) was received to either of the lists.

When the next incoming call (or SMS message) is received from this contact, the application prompts you to select an action to perform on the incoming event.
