Scan area

You can choose which types of files the application will scan for malicious objects.

When choosing the types of files to be scanned, remember the following:

  • Malicious code can be located in all types of files, and most often it is contained in executable files (such as APK files).
  • Malicious objects can be hidden in archives. In order to detect malicious code in archives, the application has to unpack them, which can reduce the speed of the device scan.

Type of files to scan

Selection of files to be analyzed during a full scan of the device's file system. You can choose to scan the following types of files:

  • All files: the application scans all types of files (such as music, text, system, and executable files). This is the default setting.
  • Executables only: the application scans only executable files (programs and installers), such as APK files, and does not scan other files. By selecting this option, the device scan is quicker.

Scan archives

Enables/disables scanning of the content of archives for malicious code. The program scans the following archive formats: ZIP, JAR, JAD, SIS SISX, CAB and APK. The program analyzes archive formats used in Android and other mobile operating systems. This allows you to scan a memory card that is used in several devices, for example.

When the program scans archives, it unpacks them. This can significantly reduce the speed of the device scan.

If you have disabled the scanning of archives and enabled the scanning of programs or installers only (executable files), Kaspersky Mobile Security scans only executable files according to the settings configured. If they are packed in archives (such as ZIP archives), the application skips them.
