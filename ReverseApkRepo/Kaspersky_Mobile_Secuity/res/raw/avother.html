Anti-Virus settings

Anti-Virus protects your device from viruses and other malware. Anti-Virus includes the following components: protection, scan and update.

Protection is enabled when the operating system starts up, and is always found in the device's memory. Protection prevents the device in real time from being infected: it scans all files that are opened, saved and executed, and all new applications.

Scan is launched on demand to detect and eliminate threats on your device (such as infected files as they are opened, unopened files, and uninstalled files in archives). Kaspersky Mobile Security allows you to perform a full or partial scan of the device's content. You can choose to scan the device's whole file system, the content of the internal memory, or a specific folder.

Updates to Kaspersky Mobile Security's anti-virus databases and application version (henceforth "updates") ensure that your device protection is up to date.

Kaspersky Mobile Security uses anti-virus databases when scanning for malware. The anti-virus databases of the application contain descriptions of all currently known malware and ways to neutralize them, as well as descriptions of other undesirable malicious objects. For this reason, it is strongly recommended to update the anti-virus databases regularly. If more than 15 days have passed since the last update, the databases are regarded as out of date. Protection will then be less reliable.

As well as the anti-virus databases, Kaspersky Mobile Security allows you to update the application version. Application version updates remove vulnerabilities from Kaspersky Mobile Security, add new functions, or improve existing functions.

Regular updates require an active license to use the application. If the license has expired, updates are not available.

Kaspersky Mobile Security runs updates from Kaspersky Lab's update servers (special websites that contain updates for all Kaspersky Lab products).

Protection mode

Protection is enabled when the operating system starts up, and is always found in the device's memory. Protection allows scanning of all files that are opened, saved and launched, and all applications that are installed.

Protection mode

The mode used by Protection to track changes in the file system in background mode and to scan it for malicious objects. The following modes are available:

  • Extended. Protection scans for the presence of malicious objects when any action is performed on them (for example, when they are opened, run, copied, saved, moved, or modified). Protection scans new applications as soon as they are installed.
  • Recommended. Protection scans for the presence of malicious objects only in newly installed programs and does not scan other files or actions involving them. Protection scans new programs as soon as they are installed.
  • Disabled. Protection is disabled.

    You can temporarily disable Protection to reduce load on the device when running several tasks concurrently.

    The Kaspersky Lab specialists do not recommend disabling Protection, as this could lead to the infection of your computer and loss of data.

    If Protection is disabled, it does not affect other Anti-Virus tasks, such as the scanning of the file system for viruses or anti-virus database updates.

Cloud Security Scan

There is an additional scan of newly installed programs using the Kaspersky Security Network cloud service (a special Kaspersky Lab online service that contains information about the reliability of files, applications, and Internet resources). Scanning using the cloud service helps to protect your data from new and unknown threats.

If the box is checked, Kaspersky Mobile Security carries out additional checks on newly installed programs before they are run for the first time using the Kaspersky Security Network cloud service. If the box is unchecked, Kaspersky Mobile Security does not carry out additional checks.

This setting is available if Protection mode is set to either Recommended or Extended.

Adware, auto-dialers, and other 

Detection of adware and other applications that may harm your device. This category includes applications that attempt to access to your personal data, send SMS messages or your location coordinates.

If the check box is selected, Kaspersky Mobile Security detects such applications and alerts you when they are detected. If the check box is cleared, Kaspersky Mobile Security skips applications in this category.

Component settings

You can configure each Anti-Virus component individually.

Protection settings

You can select the types of files that the application scans for the presence of malicious objects, and specify the action for the application to perform on files in which it detects threats.

Scan settings.

You can specify files types that must be scanned. In addition, you can set an action to be performed on malicious objects (for example, disinfect or delete), and configure automatic scheduled scans.

Update settings

You can configure automatic scheduled updates.
