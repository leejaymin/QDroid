Call&SMS Filter

Call&SMS Filter blocks unwanted messages and calls in accordance with the selected mode.

In addition to the selected mode, you can allow incoming events from all numbers in the device's address book (Contacts) or block incoming events from numbers that contain letters.

Call&SMS Filter mode

Selecting this mode causes the application to block unwanted calls and SMS messages. The following options are available:

  • Disabled – the application does not filter calls and SMS messages.
  • Blocked contacts: the application blocks calls and SMS messages from blocked contacts and allows calls and SMS messages from numbers that are not in the list of blocked contacts. The application deletes blocked SMS messages, and logs information about them in the report.
  • Allowed contacts: the application allows calls and SMS messages from allowed contacts only and blocks calls and SMS messages from numbers that are not in this list. The application deletes blocked SMS messages, and logs information about them in the report.
  • Standard – the application delivers calls and SMS messages from allowed contacts, and blocks calls and SMS messages from blocked contacts. If a call or SMS message is received from a number that is not in either of the lists, the application delivers it and prompts you to add it to the list of blocked or allowed contacts.

Allow Contacts

Allow calls and SMS messages from all numbers in the address book (Contacts). The setting is available if the selected mode is Allowed contacts or Standard.

If the check box is selected, the application automatically allows calls and SMS messages from all numbers in Contacts. If the check box is unselected, the application handles calls and SMS messages from numbers in Contacts according to the selected Call&SMS Filter mode.

For example, the Allow Contacts check box is unselected. The selected filtering mode is Allowed contacts. In this mode, the application allows only events from allowed contacts, and blocks events from all other contacts. There is not a single contact from the address book in the list of allowed contacts. In this case, the application blocks incoming events from all contacts in the address book.

Block non-numeric numbers

Block calls and SMS messages from numbers that contain letters (non-numeric). The setting is available if the selected mode is Blocked contacts or Standard.

If the check box is selected, the application blocks calls and SMS messages from non-numeric numbers. If the check box is unselected, the application handles calls and SMS messages from non-numeric numbers according to the selected Call&SMS Filter mode.
