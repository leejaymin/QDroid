Send SMS command

You can run Anti-Theft functions remotely or hide confidential information from any device with Kaspersky Mobile Security installed. To do so, you need to send a special SMS command.

The SMS command is sent in encrypted form, and contains the secret code of the application installed on the other device. Receipt of the SMS command is not acknowledged on the other device.

The application executes the received SMS command on the other device if the following conditions are met: the application has passed verification successfully, the device picks up a mobile network signal, Kaspersky Mobile Security has been installed in the role of a device administrator, and the remote start of this function is allowed on the device.

Outgoing SMS commands are billed at your mobile service provider's current rate.

Telephone number receiving the SMS command

The telephone number of the device that the SMS command is sent to.

Secret code to the application on the device receiving the SMS command

The secret code for Kaspersky Mobile Security on the device receiving the SMS command.
