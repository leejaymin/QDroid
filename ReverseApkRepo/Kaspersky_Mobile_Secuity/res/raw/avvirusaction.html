Action on threat detection

You can select the actions for the application to perform on files in which it has detected threats.

Disinfect

The application attempts to disinfect the infected file it has detected. If the check box is selected, the application first attempts to disinfect an infected file that it has detected. If the application fails to disinfect the infected file, the application performs the action specified in the If disinfection fails section.

If the check box is cleared, on detecting a malicious file the application immediately performs the action specified in the If disinfection fails section.

If disinfection fails

The action that the application performs on the infected file if disinfection fails or if the Disinfect check box is cleared. You can select one of the following actions:

  • Ask user: the application prompts you to select the action to perform on the infected file.
  • Delete: the application deletes the infected file.
  • Skip: the application does not perform any actions on the infected file and does not delete it from the device.
