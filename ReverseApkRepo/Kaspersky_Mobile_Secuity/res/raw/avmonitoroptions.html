Protection settings

You can select the types of files that the application scans for the presence of malicious objects, and specify the action for the application to perform on files in which it detects threats.

The settings section is available if Protection mode is set to Extended.

Type of files to scan

Selection of the type of files that Protection scans for the presence of malicious objects. Protection scans selected types of files when any action is performed on them (for example, when they are opened, copied, run, moved, saved, or modified). You can select one of the following file types:

  • All files: the application scans all types of files (such as music, text, system, and executable files).
  • Executables only: the application scans only executable files (programs), such as APK files.

Action on threat detection

Selection of the action that the application performs on detection of a threat. You can select one of the following actions:

  • Delete: the application deletes the infected file.
  • Skip: the application does not perform any actions on the infected file and does not delete it from the device.

Regardless of the selected action, the application notifies you that a threat has been detected.
