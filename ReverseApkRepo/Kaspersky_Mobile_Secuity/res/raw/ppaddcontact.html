Contacts to hide

The Contacts to hide list contains contacts for which related information is hidden.

You can create a list of hidden contacts only when Privacy Protection is disabled.

You can add a number manually (for example, +12345678), or import one from Contacts. You can also change or delete contacts from the list using the context menu.

Numbers added manually can be edited in the list using the context menu. A contact number imported from Contacts can be changed only in Contacts. When you modify an entry in Contacts, the list of hidden contacts is automatically updated with the new information.
