Data Wipe

Data Wipe allows you to delete information from your device remotely by SMS command so that it does not fall into the wrong hands. You can delete personal information (for example, contacts, correspondence, Google account data) and selected folders, or wipe all data, including data on the memory card.

To delete data on your device, you need to send the following SMS commands:

  • wipe:<code> – to delete specific data.
  • fullreset:<code> – to delete all data.

<Code> – this is the secret code for Kaspersky Mobile Security on your device.

Outgoing SMS commands are billed according to the price plan of the other mobile device.

You can enable or disable remote deletion of specific data or all data on your device, and select specific information to be deleted.

Delete selected data

Enable / disable remote deletion of specific data from the device (by SMS command). You can select personal information and specific folders to be deleted.

If the function is allowed on the device, on receipt of the corresponding SMS command, the application deletes the selected data on the device and replies with an SMS notification.

Outgoing SMS messages are billed according to the price plan of your mobile service provider.

If the function is blocked on the device, the application cannot execute the SMS command to delete data.

Delete all data

Enable / disable remote deletion of all data from the device using the SMS command fullreset:<code> (where <code> is the secret code to Kaspersky Mobile Security on your device).

If the function is allowed on the device, on receipt of the corresponding SMS command, the application executes it and replies with an SMS notification.

Outgoing SMS messages are billed according to the price plan of your mobile service provider.

If the function is blocked on the device, the application cannot execute the SMS command to delete all data.

Personal data to be deleted

You can select specific information to be deleted on receipt of the SMS command wipe:<code> (where <code> is the secret code to Kaspersky Mobile Security on your device).

Specific data can be selected for deletion if the Delete selected data check box is selected.

Identity data

Deletion of Contacts and SIM card entries, SMS history, call log, calendar, Internet connection settings, user accounts (except Google accounts) on receipt of the SMS command wipe:<code> (where <code> is the secret code to Kaspersky Mobile Security on your device).

If this check box is selected, Kaspersky Mobile Security deletes your personal data on receipt of the SMS command. If the check box is unselected, the application cannot delete your personal data by SMS command.

Folders

The list of folders to be deleted on receipt of the SMS command wipe:<code> (where <code> is the secret code to Kaspersky Mobile Security on your device).

You are able to add folders saved on the memory card. You can also use the context menu to delete folders from the list.
