Scan settings

To scan the device on demand, you can select the types of files to be scanned for the presence of malicious objects, enable unpacking of archives, select an action to be performed on detection of a threat, and configure scheduled scans of the file system.

Scan scope

You can choose which types of files the application will scan for malicious objects.

Action on threat detection

You can select the actions for the application to perform on files in which it has detected threats.

Scheduled scans

You can configure a schedule in order to automatically launch a full device scan: choose how often the scan is to be launched, as well as the day and time of launch (if necessary).

For the application to launch the scan automatically, the device must be turned on.

The application launches the automatic scheduled scan in background mode. When a malicious object is detected, the application performs the action selected in the scan settings.

Schedule

The frequency with which a full scan of the file system is performed. The following options are available:

  • Weekly: the application automatically launches the scan once a week. For this option, you can choose the day and time to launch the scan.
  • Daily: the application automatically launches the scan every day at the specified time. For this option, you can choose the time to launch the scan.
  • Disabled: the application does not perform an automatic scheduled scan. This allows you to perform a scan at any time.

Start day

Day of scan. This setting is available if Weekly is selected for the Schedule setting.

Start time

Time of scan. This setting is available if either Daily or Weekly is selected for the Schedule setting.
