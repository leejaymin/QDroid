Entering the secret code

The secret code is installed when the application is run for the first time. It is used to prevent unauthorized access to the application settings, and to block or find the device, delete data from it, or hide confidential information remotely.

If you forget the secret code, you can recover it.

The secret code cannot be recovered if the secret code recovery function was disabled during the initial configuration of the application.

Standard tasks:

Recovery of secret code

  1. Open Kaspersky Mobile Security > Menu > Secret code recovery.
  2. Copy the device identification code.
  3. Press Go or open the following website http://mobile.kaspersky.com/recover-code on the device or on a computer.
  4. Enter the following information in the appropriate fields:
    • Email address that you specified during the initial configuration of the application for the purposes of secret code recovery
    • Device identification code
  5. An email with the recovery code will be sent to your email address.
  6. Open Kaspersky Mobile Security > Menu > Enter recovery code.
  7. Enter the recovery code provided in the email.
  8. Confirm the new secret code.
