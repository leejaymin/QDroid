Paramètres de la protection

Vous pouvez sélectionner plusieurs types de fichier dans lesquels l'application recherche la présence éventuelle d'objets malveillants et définir l'action à exécuter sur les fichiers dans lesquelles l'application a détecté une menace.

Ce groupe de paramètres est accessible si le paramètre Mode de protection possède la valeur Avancé.

Type de fichiers à analyser

Sélection du type de fichier dans lesquels la protection recherche la présence éventuelle d'objets malveillants. La protection analyse les fichiers du type sélectionné dans le cadre de n'importe quelle action (par exemple, ouverture, copie, lancement, déplacement, enregistrement ou modification). Vous avez le choix parmi les types de fichiers suivants :

  • Tous les fichiers : l'application analyse les fichiers de tous les types (par exemple des fichiers musicaux, texte, système ou exécutables).
  • Exécutables seulement : l'application analyse uniquement les fichiers exécutables (à savoir les applications), par exemple des fichiers au format APK.

Action si détection

Sélection de l'action que l'application exécute lors de la détection d'une menace. Vous avez le choix entre les opérations suivantes :

  • Supprimer : l'application supprime le fichier infecté.
  • Ignorer : l'application n'exécute aucune action sur le fichier infecté et ne le supprime pas de l'appareil.

Quel que soit l'action sélectionnée, l'application vous signale qu'une menace a été détectée.
