Envoi d'une instruction

Vous pouvez lancer la fonction Antivol ou masquer les informations confidentielles sur un autre appareil doté de Kaspersky Mobile Security à distance. Il faut pour ce faire envoyer des instructions spéciales par SMS.

L'instruction SMS est envoyée sous forme d'un SMS crypté qui contient le code secret de l'application, installée sur l'autre appareil. La réception de l'instruction passera inaperçue sur l'autre appareil.

Sur un autre appareil, l'application exécute l'instruction SMS reçue si les conditions suivantes ont été exécutées : L'application a passé une vérification d'authenticité, le signal de téléphonie mobile existe sur l'appareil, l'application Kaspersky Mobile Security est installée en tant que l'administrateur de l'appareil et le lancement à distance de cette fonction est autorisé sur l'appareil.

Le coût du SMS contenant l'instruction envoyé est celui de votre opérateur de téléphonie mobile.
