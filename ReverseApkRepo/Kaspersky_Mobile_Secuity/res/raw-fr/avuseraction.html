Menace détectée

L'application propose de sélectionner l'action à exécuter sur le fichier qui contient la menace.

L'application signale la présence d'une menace dans les cas suivants :

  • Si l'Anti-Virus a détecté une menace dans une nouvelle application directement après son installation dans le cadre de la protection ou lors de l'analyse de l'appareil.
  • Si l'Anti-Virus a découvert une menace dans un fichier alors que les paramètres de l'Anti-Virus relatifs au paramètre Si réparation impossible contiennent la valeur Confirmer (fenêtre principale de Kaspersky Mobile Security > Anti-Virus > Configuration > Paramètres d'analyse > Action si détection).

Supprimer

L'application supprime le fichier infecté.

Ignorer

L'application n'exécute aucune action sur le fichier infecté et ne le supprime pas de l'appareil.
