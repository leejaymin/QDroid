Sélection du dossier

Vous pouvez sélectionner les dossiers et les ajouter à la liste des dossiers à supprimer. Lorsque l'application reçoit l'instruction SMS wipe:<code>, elle supprime ces dossiers (<code> représente le code secret de Kaspersky Mobile Security sur votre appareil).

La liste des dossiers sur la carte mémoire s'affiche dans la fenêtre. Pour ajouter un dossier, sélectionnez-le dans l'arborescence, puis cliquez sur le bouton à droite du nom du dossier.
