Verrouillage

L'option de verrouillage permet d'envoyer l'instruction SMS lock:<code> pour verrouiller à distance votre appareil afin que personne ne puisse avoir accès à vos données (<code> représente le code secret de Kaspersky Mobile Security sur votre appareil).

Le coût de l'envoi du SMS contenant l'instruction est défini par le tarif de l'opérateur de téléphonie mobile de l'autre appareil nomade.

Vous pouvez activer ou désactiver le lancement à distance du verrouillage sur l'appareil.

Activer le Verrouillage

Autorisation/interdiction du verrouillage à distance de l'appareil à l'aide d'une instruction SMS.

Si la fonction est autorisée sur l'appareil, alors elle verrouille l'appareil dès la réception de l'instruction par SMS et confirme l'opération dans une réponse par SMS.

Le coût du SMS envoyé en réponse est celui pratiqué par votre opérateur de téléphonie mobile.

Si la fonction est interdite sur l'appareil, l'application ne peut exécuter l'instruction de verrouillage par SMS.

Texte si verrouillage

Le texte du message qui apparaît sur l'écran de l'appareil verrouillé. Un texte standard est saisi par défaut.
