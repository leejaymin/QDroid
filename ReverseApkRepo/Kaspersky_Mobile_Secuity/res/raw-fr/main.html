Kaspersky Mobile Security

Anti-Virus

L'Antivirus permet de détecter et de neutraliser les menaces sur votre appareil à l'aide des bases antivirus de l'application et du service complémentaire dans le nuage Kaspersky Security Network. L'Anti-Virus reprend les composants suivants : protection, analyse et mise à jour.

La protection permet de découvrir les menaces dans les fichiers ouverts, d'analyser les nouvelles applications et de prévenir l'infection de l'appareil en temps réel.

La fonction Protection est lancée à la demande pour tout le système de fichiers, la mémoire vive ou les dossiers. L'analyse complète permet de rechercher la présence éventuelle d'objets malveillants dans tous le système de fichiers de l'appareil tandis que l'analyse d'un dossier porte sur un dossier en particulier. L'analyse complète et l'analyse d'un dossier détectent les menaces dans les fichiers qui sont installés, mais ne sont pas ouverts, ainsi que les menaces dans les fichiers qui sont actuellement ouverts. L'analyse de la mémoire permet de détecter les menaces uniquement dans les fichiers qui sont actuellement ouverts.

La mise à jour permet de charger les nouvelles versions des bases antivirus ou de l'application.

Tâches types :

Analyse de l'ensemble du système de fichiers de l'appareil

Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Analyse > Analyse complète.

Analyse dossier

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Analyse > Analyser dossier.
  2. Sélectionnez le dossier à analyser

Analyse de la mémoire de l'appareil

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Analyse > Analyse de la mémoire.

Modification du mode de protection de l'appareil

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Paramètres.
  2. Sélectionnez le mode de protection. Pour ce faire, attribuez une valeur au paramètre Mode de protection :

Analyse programmée

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Paramètres > Analyse.
  2. Sélectionnez la fréquence de lancement de l'analyse. Pour ce faire, attribuez une valeur au paramètre Programmation.
  3. Sélectionnez le jour et l'heure de lancement de l'analyse. Pour ce faire, définissez les valeurs des paramètres Jour d'analyse et Heure d'analyse.

Mise à jour des bases antivirus et des versions de l'application

Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Mise à jour.

Mise à jour programmée des bases de l'application et des versions

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Anti-Virus > Paramètres > Paramètres de mise à jour.
  2. Sélectionnez la fréquence de lancement de la mise à jour. Pour ce faire, sélectionnez la valeur pour le paramètre Programmation.
  3. Indiquez le jour et l'heure de lancement de la mise à jour. Pour ce faire, définissez les valeurs des paramètres Jour de lancement et Heure de début.

Contacts personnels

Les contacts personnels permettent de masquer vos contacts personnels, ainsi que l'historique des conversations et des échanges SMS avec ces contacts.

Les contacts personnels sont conservés dans la liste Contacts masqués.

Pour les contacts personnels, l'application permet de masquer les informations suivantes :

  • les entrées dans les contacts ;
  • les SMS reçus lus, les SMS transmis et les brouillons de SMS ;
  • les entrées dans le journal des appels ;
  • les appels entrants et les notifications relatives aux nouveaux SMS.

Quand la dissimulation des informations est activée, l'application bloque le signal de réception d'un nouveau SMS en provenance du numéro confidentiel et masque le message dans la lise des messages reçus. L'applique bloque les appels entrants d'un numéro confidentiel et l'écran n'indiquera rien au sujet de ces appels. Dans ce cas, la personne qui appelle entendra la tonalité "occupé". Pour afficher les appels et les SMS reçus pendant la période où la dissimulation des informations confidentielles était activée, il faut désactiver la dissimulation des informations. Si vous activez à nouveau la dissimulation des informations, les informations confidentielles sont à nouveau masquées.

Vous pouvez activer la dissimulation des informations de l'application à distance à l'aide d'une instruction SMS envoyée depuis un autre appareil mobile. Vous pouvez envoyer un message SMS standard avec un texte spécifique ou envoyer un message SMS depuis Kaspersky Mobile Security si l'application est installée sur un autre appareil.

Vous pouvez interdire ou autoriser l'activation à distance de l'activation de la dissimulation des informations. Si l'activation à distance de la dissimulation des informations est interdite, alors la fonction ne peut être lancée à distance via une instruction SMS.

Vous pouvez désactiver la dissimulation des informations uniquement pour l'application installée sur votre appareil.

Tâches types :

Dissimulation des informations relatives aux contacts

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Contacts personnels.
  2. Rédigez une liste des contacts masqués si vous ne l'avez pas encore rédigée. Cette liste reprend les contacts dont vous souhaitez masquer les informations. Pour ce faire, cliquez sur Contacts masqués > Ajouter.
  3. Revenez à la fenêtre principale de Kaspersky Mobile Security > Contacts personnels et positionnez le bouton Dissimuler les informations sur activé.

Lancement à distance de la dissimulation des informations

Sur l'autre appareil mobile, réalisez une des actions suivantes :

  • Si l'application Kaspersky Mobile Security est installée sur l'autre appareil, rédigez et envoyez l'instruction à votre appareil via SMS.
  • Rédigez et envoyez à votre appareil un SMS contenant le texte hide:<code>, où<code> représente le code secret défini sur votre appareil.

Le coût du SMS envoyé est celui de l'opérateur de téléphonie mobile de l'autre appareil mobile.

Création d'une liste de contacts à masquer

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Contacts personnels > Contacts masqués.
  2. Exécutez une des actions suivantes :
    • Pour ajouter un contact du répertoire de l'appareil, appuyez sur Ajouter > Contact.
    • Pour ajouter un numéro de téléphone, appuyez sur Ajouter > Numéro de téléphone.

Antivol

Le module Antivol protège les données de l'appareil contre l'accès non autorisé et permet de localiser l'appareil en cas de vol ou de perte de celui-ci.

A l'aide des instructions SMS envoyées depuis l'autre appareil mobile, Antivol permet, à distance, de verrouiller l'appareil, de supprimer les informations qu'il contient ou de définir sa position sur une carte. De plus, l'Antivol permet également de verrouiller l'appareil en cas de remplacement de la carte SIM ou de mise sous tension de l'appareil sans cette celle-ci.

Les fonctions de l'Antivol sont lancées à l'aide d'une instruction SMS spéciale. Vous pouvez envoyer un SMS normal contenant un texte spécial depuis un autre appareil mobile ou envoyer un SMS depuis l'application, si l'autre appareil est équipé de Kaspersky Mobile Security. Pour pouvoir envoyer les instructions SMS, vous devez connaître le code secret de Kaspersky Mobile Security installé sur votre appareil.

Vous pouvez interdire ou autoriser le lancement à distance des fonctionnalités Antivol sur votre appareil. Si le lancement des fonctionnalités Antivol est interdit, vous ne pouvez pas les lancer à distance à l'aide d'une instruction SMS.

Tâches types :

Activation du verrouillage à distance de l'appareil

  1. Sur l’écran principal de Kaspersky Mobile Security, appuyez sur Antivol > Verrouillage.
  2. Cochez la case Activer le Verrouillage.
  3. Le cas échéant, saisissez dans le champ Texte si verrouillage le texte du message qui sera affiché sur l'écran de l'appareil verrouillé.

Activation de la suppression à distance des données

  1. Sur l’écran principal de Kaspersky Mobile Security, appuyez sur Antivol > Suppression.
  2. Sélectionnez les informations qui seront supprimées après la réception de l'instruction à distance :
    • Pour supprimer à distance les données sélectionnées, commencez par cocher la case Suppression des données sélectionnées. Pour supprimer les données personnelles (par exemple, les contacts, la correspondance SMS et le calendrier), cochez la case Données personnelles. Pour supprimer à distance des dossiers, appuyez sur Ajouter un dossier, puis sélectionnez les dossiers indispensables.
    • Pour pouvoir supprimer à distance toutes les données et rétablir la configuration par défaut de l'appareil, cochez la case Effacement Complet.

Autorisation de l'activation de la localisation à distance

  1. Dans l’écran principal de Kaspersky Mobile Security, appuyez sur Antivol > Localisation.
  2. Cochez la case Activer la Localisation.
  3. Le cas échéant, saisissez l'adresse de messagerie à laquelle vous souhaitez recevoir également les coordonnées de votre appareil.

Autorisation du contrôle du remplacement de la carte SIM

  1. Sur l’écran principal de Kaspersky Mobile Security, appuyez sur Antivol > SIM-Surveillance.
  2. Cochez la case Activer SIM-Surveillance.
  3. Indiquez votre numéro de téléphone complémentaire ou une adresse de messagerie pour recevoir le nouveau numéro de téléphone en cas de remplacement de la carte SIM par SMS ou par courrier électronique.
  4. Le cas échéant, cochez la case Verrouiller afin de verrouiller l'appareil lors du remplacement de la carte SIM ou en cas d'utilisation de celui-ci sans la carte SIM. Rédigez le texte du message qui apparaît sur l'écran de l'appareil verrouillé.

Lancement à distance des fonctions d'Antivol depuis un autre appareil

Sur l'autre appareil mobile, réalisez une des actions suivantes :

  • Si l'application Kaspersky Mobile Security est installée sur l'autre appareil, rédigez et envoyez l'instruction à votre appareil via SMS. Pour pouvoir rédiger les instructions SMS, vous devez connaître le code secret de Kaspersky Mobile Security installé sur votre appareil.
  • Envoyez à votre appareil un SMS contenant le texte suivant :
    • lock:<code> pour verrouiller l'appareil ;
    • wipe:<code> pour supprimer les données personnelles sélectionnées de l'appareil ;
    • fullreset:<code> pour supprimer toutes les données de l'appareil ;
    • locate:<code> pour localiser l'appareil ;

    <Code> : il s'agit du code secret de Kaspersky Mobile Security sur votre appareil.

Le coût du SMS envoyé est celui de l'opérateur de téléphonie mobile de l'autre appareil nomade.

Filtre des appels et SMS

Le Filtre des appels et SMS vous débarrasse des SMS et des appels entrants non sollicités. L'application filtre les appels et les SMS sur la base des contacts autorisés et interdits et selon le mode de filtrage sélectionné.

Tâches types :

Blocage des appels entrants et des SMS non sollicités

  1. Définissez les contacts dont vous souhaitez bloquer les appels et SMS et les contacts dont vous souhaitez autoriser les appels et SMS. Pour ce faire, sélectionnez le mode de filtrage. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Filtres des appels et SMS > Mode : <mode de filtrage actuel>.
  2. Sélectionnez la valeur pour le paramètre Filtrage des appels :
    • Pour bloquer les appels et SMS uniquement en provenance des contacts interdits et autoriser tous les autres appels et SMS, sélectionnez Contacts interdits.
    • Pour bloquer les appels et SMS en provenance de tous les contacts et autoriser uniquement les appels et SMS en provenance des contacts autorisés, sélectionnez Contacts autorisés.
    • Pour bloquer les appels et SMS en provenance des contacts interdits et autoriser uniquement les appels et SMS en provenance des contacts autorisés, sélectionnez Standard.
  3. Activez les paramètres avancés de filtrage si nécessaire.

    Par exemple, si vous avez sélectionné le mode de filtrage Contacts autorisés ou Standard, vous pouvez automatiquement autoriser les appels et SMS en provenance des contacts du répertoire de l'appareil quel que soit le mode de filtrage sélectionné. Pour ce faire, cochez la case Autoriser les contacts.

    Si vous avez sélectionné le mode de filtrage Contacts interdits ou Standard, vous pouvez automatiquement interdire des SMS en provenance des numéros contenant des lettres quel que soit le mode de filtrage sélectionné. Pour ce faire, cochez la case Bloquer N° sans chiffres.

  4. En fonction du mode sélectionné, rédigez une liste de contacts autorisés et interdits :
    • Pour établir une liste de contacts interdits, revenez à l’écran principal. Dans la fenêtre principale de Kaspersky Mobile Security > Filtre des appels et SMS > Contacts interdits.
    • Pour établir une liste de contacts autorisés, revenez à l’écran principal. Dans la fenêtre principale de Kaspersky Mobile Security > Filtre des appels et SMS > Contacts autorisés.

Ajout de contacts à la liste des contacts autorisés

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Filtres des appels et SMS > Contacts autorisés.
  2. Appuyez sur Ajouter.
  3. Sélectionnez les types d'événements que vous souhaitez recevoir de ce contact.
  4. Sélectionnez le numéro du contact en provenance duquel vous souhaitez recevoir les informations. Pour ce faire, saisissez les données dans le champ Numéro de téléphone autorisé.

    Vous pouvez ajouter des numéros d'une des manières suivantes : saisir manuellement le numéro dans le champ prévu, sélectionner le numéro de l'abonné dans la liste des appels ou sélectionner un SMS en provenance de l'abonné dans la liste des messages reçus. Pour ajouter un numéro depuis la liste des appels ou depuis la liste des messages, appuyez sur .

    Le numéro peut commencer par un chiffre, par une lettre ou par le signe "+" et doit contenir des chiffres et/ou des lettres Les caractères génériques "*" et "?" peuvent également être utilisés en guise de numéro (où"*" représente n'importe quelle série de caractère et "?", n'importe quel caractère unique). Par exemple, si le numéro *1234? est ajouté à la liste des contacts autorisés, le Filtre des appels et SMS accepte les appels et les SMS en provenance des numéros où la séquence 1234 est suivie de n'importe quel caractère.

  5. Si vous avez sélectionné le type d'événement SMS, saisissez l'expression clé qui indiquera que le message reçu est désirable. Pour ce faire, saisissez les données dans le champ Texte autorisé. Le Filtre des appels et SMS autorise uniquement les SMS en provenance de ce numéro qui contiennent l'expression clé et bloque tous les autres SMS en provenance de ce numéro.

    Si vous souhaitez recevoir depuis ce contact des SMS contenant n'importe quel texte, ne remplissez pas le champ Texte autorisé.

Ajout d'un contact à la liste des contacts interdits

  1. Dans la fenêtre principale de Kaspersky Mobile Security, appuyez sur Filtres des appels et SMS > Contacts interdits.
  2. Appuyez sur Ajouter.
  3. Sélectionnez les événements en provenance de ce contact que vous considérerez comme indésirables.
  4. Saisissez le numéro de téléphone du contact interdit. Pour ce faire, saisissez les données dans le champ Numéro de téléphone interdit.

    Vous pouvez ajouter des numéros d'une des manières suivantes : saisir manuellement le numéro dans le champ prévu, sélectionner le numéro de l'abonné dans la liste des appels ou sélectionner un SMS en provenance de l'abonné dans la liste des messages reçus. Pour ajouter un numéro depuis la liste des appels ou depuis la liste des messages, appuyez sur .

    Le numéro peut commencer par un chiffre, par une lettre ou par le signe "+" et doit contenir des chiffres et/ou des lettres Les caractères génériques "*" et "?" peuvent également être utilisés en guise de numéro (où"*" représente n'importe quelle série de caractère et "?", n'importe quel caractère unique). Par exemple, si le numéro *1234? est ajouté à la liste des contacts interdits, le Filtre des appels et SMS rejettera les appels et les SMS en provenance des numéros où la séquence 1234 est suivie de n'importe quel caractère.

  5. Si vous avez sélectionné le type d'événement SMS, saisissez l'expression clé qui indiquera que le message reçu est indésirable. Pour ce faire, saisissez les données dans le champ Texte interdit. Le Filtre des appels et SMS bloque uniquement les SMS en provenance de ce numéro qui contiennent l'expression clé et autorise tous les autres SMS en provenance de ce numéro.

    Si vous ne souhaitez pas recevoir depuis ces numéros des SMS contenant n'importe quel texte, ne remplissez pas le champ Texte interdit.

Protection Internet

Le module Protection Internet bloque les sites Internet malveillants dont l'objectif est de diffuser du code malveillant ainsi que les sites de phishing dont la fonction est de voler les données confidentielles des utilisateurs comme les mots de passe d'accès aux services de transactions bancaires en ligne, aux sites de ventes aux enchères en ligne ou aux systèmes de paiement et d'avoir ainsi accès à vos comptes financiers.

Le module Protection Internet analyse le site Internet avant son ouverture à l'aide du service Cloud de Kaspersky Security Network (service en ligne de Kaspersky Lab qui contient les informations relatives à la fiabilité des fichiers, des applications et des ressources Internet).

La Protection Internet analyse les sites Internet uniquement dans le navigateur standard d'Android et non pas dans les autres navigateurs.

Quand le module Protection Internet est activé, il détermine automatiquement quel est le navigateur installé par défaut. Si la Protection Internet n'est pas compatible avec ce navigateur, le module propose de le remplacer par le navigateur standard d'Android.

Pour garantir l'analyse permanente des sites Internet, il est conseillé d'installer le navigateur standard d'Android en tant que navigateur par défaut.

Si vous ne souhaitez pas modifier le navigateur par défaut actuel, il suffira de lancer le navigateur standard d'Android lorsque vous souhaitez garantir la sécurité de vos données personnelles pendant l'utilisation d'Internet.

Vous pouvez lancer le navigateur standard d'Android depuis Kaspersky Mobile Security ou depuis le menu des applications de votre appareil. Si la Protection Internet est activée, elle bloquera les sites de phishing avant leur ouverture.

Tâches types :

Activation de la Protection Internet pour l'analyse continue des sites Internet

  1. Sur l’écran principal de Kaspersky Mobile Security, appuyez sur Protection Internet.
  2. Placez le bouton Activer la Protection Internet en position activé.
  3. Appuyez sur Modifier le navigateur (ce bouton s'affiche si la Protection Internet est activée et ne vérifie pas l'URL dans le navigateur Internet par défaut actuel).
  4. Suivez les instructions de l'Assistant d'installation du navigateur par défaut.

Utilisation sécurisée des données personnelles

Sur l’écran principal de Kaspersky Mobile Security, appuyez sur Protection Internet > Lancer le navigateur.
