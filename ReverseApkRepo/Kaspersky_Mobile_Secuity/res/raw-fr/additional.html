Avancé

Vous pouvez réaliser les opérations suivantes :

  • configurer la réception des notifications, l'affichage des astuces ou le widget ;
  • modifier le code secret ;
  • consulter les rapports sur le fonctionnement de l'application (par exemple, les rapports sur l'analyse exécutée, sur les menaces détectées, sur les SMS, les appels ou les sites Internet bloqués) ;
  • consulter les informations relatives à l'application.

Paramètres avancés

Notifications

Activation/désactivation des notifications flottantes sur le fonctionnement de l'application. Ces notifications s'affichent dans la barre d'état. Par exemple, des notifications signalants qu'une menace a été détectée ou que la validité de la licence est écoulée.

Son

Activation/ désactivation des notifications sonores sur les événements survenus pendant le fonctionnement de l'application. Par exemple, l'application signale la détection d'une menace par un son.

Astuces

Activation ou désactivation de l'affichage de brèves informations sur le composant avant sa configuration.

Widget logiciel

L'indicateur de couleur du gadget logiciel de l'écran principal informe sur l'état de la protection de votre appareil et sur la dissimulation de vos données confidentielles et il permet d'accéder à la configuration des paramètres de l'application.

Le vert indique que le mode de protection sélectionné est le mode recommandé ou avancé. Le bouclier gris signifie que la protection est désactivée. La couleur verte du fond signifie que les informations confidentielles sont masquées ; la couleur grise - les informations confidentielles s'affichent. Le point d'exclamation dans le triangle jaune signifie que la licence a expiré.

Vous pouvez décider d'afficher ou non le widget sur l'écran principal de l'appareil.

Modification du code secret

Le code secret de l'application est défini lors de la première exécution de l'application et ses fonctions sont les suivantes :

  • empêcher l'accès non autorisé aux paramètres de l'application ;
  • lancer à distance depuis un autre appareil la fonction Antivol et la dissimulation des données.

Vous pouvez modifier le code secret.

Suppression de Kaspersky Mobile Security

Pour supprimer Kaspersky Mobile Security, vous devez saisir le code secret de l'application.

Informations générales

Rapports

Les événements qui surviennent pendant le fonctionnement de l'Anti-Virus, de la Protection Internet et du Filtre des appels et des SMS sont consignés dans des rapports.

Les rapports sont regroupés par la date et l'heure de création. Vous pouvez sélectionner l'affichage des rapports pour un composant de l'application. Les rapports peuvent conserver un maximum de 50 entrées. Quand le nombre d'entrées atteint 50, les rapports plus anciens sont supprimés et remplacés par de nouveaux.
