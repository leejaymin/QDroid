Saisie du code secret

Le code secret de l'application est défini lors de la première exécution de celle-ci. Il permet d'éviter l'accès non autorisé aux paramètres de l'application et de verrouiller à distance l'appareil ou de déterminer sa position, de supprimer les données qu'il contient ou de masquer les informations confidentielles.

Si vous avez oublié le code secret, vous pouvez le restaurer.

Il est impossible de restaurer le code secret si la fonction de restauration du code secret avait été désactivée lors de la configuration initiale de l'application.

Tâches types :

Restauration du code secret

  1. Ouvrez Kaspersky Mobile Security > Menu > Restauration du code secret.
  2. Copiez le code d'identification de l'appareil.
  3. Cliquez sur Restaurer ou ouvrez le site Internet http://mobile.kaspersky.com/recover-code sur l'appareil ou sur un ordinateur.
  4. Saisissez les informations suivantes dans les champs correspondants :
    • l'adresse de messagerie que vous avez indiquée lors de la configuration initiale de l'application pour la restauration du code secret.
    • code d'identification de l'appareil.
  5. Vous recevrez un message électronique contenant le code de restauration.
  6. Sur l'appareil, ouvrez Kaspersky Mobile Security > Menu > Saisie du code de restauration.
  7. Saisissez le code de restauration indiqué dans le message reçu.
  8. Saisissez un nouveau code secret.
