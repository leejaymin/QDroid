Zone d'analyse

Vous pouvez sélectionner les types de fichiers dans lesquels l'application va rechercher la présence éventuelle d'objets malveillants.

Au moment de sélectionner les types de fichier à analyser, n'oubliez pas les éléments suivants :

  • N'importe quel fichier peut contenir du code malveillant et souvent ce code malveillant se retrouve dans des fichiers exécutables (par exemple, des fichiers au format APK).
  • Des objets malveillants peuvent être dissimulés dans des archives. Pour détecter du code malveillant dans des archives, l'application doit les décompacter, ce qui peut ralentir l'analyse de l'appareil.

Type de fichiers à analyser

Le choix des fichiers analysés par l'application pendant l'Analyse complète du système de fichiers de l'appareil. Vous pouvez sélectionner les types de fichier suivants :

  • Tous les fichiers : l'application analyse les fichiers de tous les types (par exemple des fichiers musicaux, texte, système ou exécutables). Le paramètre est sélectionné par défaut.
  • Exécutables seulement : l'application analyse uniquement les fichiers exécutables (à savoir, les applications et les programmes d'installation du logiciel), par exemple les fichiers au format APK. Les autres fichiers ne sont pas analysés. La vitesse d'analyse de l'appareil augmente avec cette option.

Analyse des archives

Activation ou désactivation de la recherche de la présence éventuelle de code malveillant dans le contenu des archives. L'application permet d'analyser les archives aux formats suivants : ZIP, JAR, JAD, SIS, SISX, CAB et APK. L'application analyse les archives dans les formats pris en charge par Android et d'autres systèmes d'exploitation pour téléphone mobile. Il est ainsi possible, par exemple, d'analyser la carte mémoire utilisée sur plusieurs appareils.

Quand l'application analyse des archives, elle les décompacte. Ceci peut ralentir considérablement l'exécution de l'analyse de l'appareil.

Si vous avez désactivé l'analyse des archives et sélectionné l'analyse des applications et des programmes d'installation du logiciel (fichiers exécutables), Kaspersky Mobile Security analyse uniquement les fichiers exécutables conformément aux paramètres sélectionnés. Si ces fichiers sont compressés dans une archive (par exemple, les archives au format ZIP), l'application les exclut de l'analyse.
