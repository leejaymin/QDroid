Suppression

La fonction Suppression permet de supprimer à distance les informations sur votre appareil via une instruction SMS afin qu'elles ne tombent pas entre de mauvaises mains. Vous pouvez supprimer les informations personnelles (par exemple, les contacts, la correspondance ou les données relatives au compte utilisateur Google) ou supprimer tous les dossiers ou toutes les données, y compris les données de la carte mémoire.

Pour supprimer les données sur votre appareil, il faut envoyer les instructions SMS suivantes :

  • wipe:<code> : suppression de données particulières ;
  • fullreset:<code> : suppression de toutes les données.

<Code> : il s'agit du code secret de Kaspersky Mobile Security sur votre appareil.

Le coût de l'envoi du SMS contenant l'instruction est défini par le tarif de l'opérateur de téléphonie mobile de l'autre appareil nomade.

Vous pouvez autoriser ou interdire le lancement à distance de la suppression de données particulières ou de toutes les données sur l'appareil ainsi que sélectionner les données que vous souhaitez supprimer.

Effacer des données

Autorise/interdit la suppression de données spécifiques de l'appareil à distance (à l'aide d'une instruction SMS spéciale). Vous pouvez décider de supprimer les données personnelles ou des dossiers en particulier.

Si la fonction est autorisée sur l'appareil, alors elle verrouille les données spécifiques sélectionnées sur l'appareil dès la réception de l'instruction par SMS et confirme l'opération dans une réponse par SMS.

Le coût du SMS envoyé en réponse est celui pratiqué par votre opérateur de téléphonie mobile.

Si la fonction est interdite sur l'appareil, l'application ne peut exécuter l'instruction de suppression de données particulières par SMS.

Effacement Complet

Autorise/interdit la suppression de toutes les données de l'appareil à distance à l'aide d'une instruction SMS fullreset:<code> (où <code> représente le code secret de Kaspersky Mobile Security sur votre appareil).

Si la fonction est autorisée sur l'appareil, alors elle supprime toutes les données dès la réception de l'instruction par SMS et confirme l'opération dans une réponse par SMS.

Le coût du SMS envoyé en réponse est celui pratiqué par votre opérateur de téléphonie mobile.

Si la fonction est interdite sur l'appareil, l'application ne peut exécuter l'instruction de suppression de toutes les données particulières par SMS.

Informations personnelles à supprimer

Vous pouvez sélectionner les informations précises qui seront supprimées après la réception de l'instruction SMS wipe:<code> (où <code> représente le code secret de Kaspersky Mobile Security sur votre appareil).

La sélection d'informations spécifiques est accessible si la case Suppression des données sélectionnées est cochée.

Données personnelles

Suppression des entrées dans les contacts et sur la carte SIM, toute l'historique des SMS, le journal des appels, le calendrier, les paramètres de connexion à Internet, les comptes d'utilisateurs (sauf le compte Google) après l'obtention de l'instruction SMS wipe:<code> (où <code> représente le code secret de Kaspersky Mobile Security sur votre appareil).

Quand la case est cochée, l'application supprime les données personnelles après avoir reçu l'instruction spéciale par SMS. Si la case n'est pas cochée, l'application ne supprime pas vos données personnelles en cas de réception d'une instruction SMS.

Dossiers

La liste des dossiers qui seront supprimés après la réception de l'instruction SMS wipe:<code> (où <code> représente le code secret de Kaspersky Mobile Security sur votre appareil).

Vous pouvez ajouter des dossiers enregistrés sur une carte mémoire. Vous pouvez également supprimer les dossiers de la liste via le menu contextuel.
