Action si détection

Vous pouvez sélectionner les actions à réaliser sur les fichiers dans lesquels l'application a détecté des menaces.

Réparer

L'application tente de réparer le fichier infecté détecté. Quand la case est cochée, l'application tente d'abord de réparer le fichier infecté détecté. Si l'application ne parvient pas à réparer le fichier infecté, elle exécute l'action définie dans le groupe Si réparation impossible.

Si cette case n'est pas cochée, quand un fichier infecté est détecté, l'application exécute sans délai l'action définie dans le groupe Si réparation impossible.

Si réparation impossible

Action que l'application exécute sur le fichier infecté au cas où la réparation serait impossible ou si la case Réparer est décochée. Vous avez le choix entre les opérations suivantes :

  • Confirmer : l'application vous propose de sélectionner l'action à réaliser sur le fichier infecté.
  • Supprimer : l'application supprime le fichier infecté.
  • Ignorer : l'application n'exécute aucune action sur le fichier infecté et ne le supprime pas de l'appareil.
