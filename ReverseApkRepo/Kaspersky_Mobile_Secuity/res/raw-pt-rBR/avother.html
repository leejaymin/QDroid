Configurações do Antivírus

O Antivírus protege o dispositivo de vírus e outros malware. O Antivírus inclui os seguintes componentes: proteção, verificação e atualização.

A Proteção é ativada quando o sistema operacional é iniciado e é sempre encontrada na memória do dispositivo. A Proteção previne a infecção do dispositivo, em tempo real: verifica todos os arquivos que são abertos, salvos ou executados e todos os novos aplicativos.

A Verificação é inicializada a pedido, para detectar e eliminar ameaças ao seu dispositivo (tais como arquivos infetados ao serem abertos, arquivos não abertos e arquivos comprimidos não instalados). O Kaspersky Mobile Security permite a realização de uma verificação completa ou parcial do conteúdo do dispositivo. Você pode escolher verificar todo o sistema de arquivos do dispositivo, o conteúdo da memória interna ou uma pasta específica.

As atualizações dos bancos de dados e da versão do aplicativo do Kaspersky Mobile Security (daqui em diante referidas como "atualizações") asseguram que a proteção de seu dispositivo está atualizada.

O Kaspersky Mobile Security usa bancos de dados de antivírus ao verificar malware. Os bancos de dados do aplicativo contêm descrições de todo o software malicioso conhecido e métodos para seu tratamento, bem como descrições de outros objetos maliciosos indesejáveis. Por esse motivo, é vivamente recomendável que atualize os bancos de dados de antivírus regularmente. Se já passaram mais de 15 dias desde a última atualização, os bancos de dados são considerados desatualizados. A proteção será menos confiável.

Assim como os bancos de dados de antivírus, o Kaspersky Mobile Security permite atualizar a versão do aplicativo. As atualizações da versão do aplicativo removem vulnerabilidades do Kaspersky Mobile Security, adicionam novas funções ou melhoram funções existentes.

As atualizações regulares requerem uma licença ativa para usar o aplicativo. Caso a licença tenha expirado, as atualizações não ficam disponíveis.

O Kaspersky Mobile Security executa atualizações a partir dos servidores de atualização da Kaspersky Lab (sites especiais que contêm atualizações para todos os produtos da Kaspersky Lab).

Modo de Proteção

A Proteção é ativada quando o sistema operacional é iniciado e é sempre encontrada na memória do dispositivo. A Proteção permite verificar todos os arquivos que são abertos, salvos e inicializados e todos os aplicativos instalados.

Modo de Proteção

O modo utilizado pela Proteção para rastrear em segundo plano as alterações no sistema de arquivos e para verificar a presença de objetos maliciosos. Os seguintes modos estão disponíveis:

  • Expandido. A proteção verifica a presença de objetos maliciosos quando uma ação é executada neles (por exemplo, quando eles são abertos, executados, copiados, salvos, movidos ou modificados). A Proteção verifica os novos aplicativos logo depois de serem instalados.
  • Recomendado. A Proteção verifica a presença de objetos maliciosos somente em novos aplicativos instalados e não verifica outros arquivos ou ações que os envolvam. A Proteção verifica os novos programas logo depois de serem instalados.
  • Desativado. A Proteção está desativada.

    Você pode temporariamente desativar a Proteção para reduzir o volume de processos no dispositivo ao executar várias tarefas em simultâneo.

    Os especialistas da Kaspersky Lab não recomendam a desativação da Proteção, já que isto pode permitir uma infecção de seu computador e perda de dados.

    Se a Proteção estiver desativada, isso não afeta a execução de outras tarefas do Antivírus, como a verificação do sistema de arquivos quanto à presença de vírus ou a atualização dos bancos de dados de antivírus.

Cloud Security Scan

Existe uma verificação adicional dos novos aplicativos instalados usando o serviço do Kaspersky Security Network na nuvem (um serviço online especial da Kaspersky Lab que contém informações sobre a confiabilidade de arquivos, aplicativos e recursos da Internet). A verificação através do serviço na nuvem ajuda a proteger seus dados contra ameaças novas e desconhecidas.

Se a caixa for selecionada, o Kaspersky Mobile Security irá realizar verificações adicionais em novos aplicativos instalados antes que eles sejam executados pela primeira vez, usando o serviço do Kaspersky Security Network na nuvem. Se a caixa estiver desmarcada, o Kaspersky Mobile Security não realizará verificações adicionais.

Esta configuração está disponível se o modo de Proteção estiver configurado para Recomendado ou Expandido.

Adware, discadores automáticos e outros 

A detecção de adware e outros aplicativos que podem prejudicar seu dispositivo foi adicionada. Esta categoria inclui aplicativos que tentam acessar seus dados particulares, enviar mensagens de SMS ou as coordenadas da sua localização.

Se a caixa de seleção estiver selecionada, o Kaspersky Mobile Security detecta tais aplicativos e alerta você quando eles são detectados. Se a caixa de seleção estiver desmarcada, o Kaspersky Mobile Security ignora os aplicativos nesta categoria.

Configurações do componente

Você pode configurar cada componente do Antivírus individualmente.

Configurações de proteção

Você pode selecionar os tipos de arquivos que o aplicativo verifica quanto à presença de objetos maliciosos e especificar a ação que o aplicativo realiza nos arquivos em que detectar ameaças.

Configurações de verificação.

Você pode especificar os tipos de arquivos que devem ser verificados. Além disso, você pode configurar uma ação a ser executada em objetos maliciosos (por exemplo, desinfetar ou excluir) e configurar verificações automáticas agendadas.

Atualizar configurações

Você pode configurar atualizações automáticas agendadas.
