Limpeza de Dados

A Limpeza de Dados permite que você exclua remotamente informações do seu dispositivo através do comando SMS para que eles não caiam nas mãos erradas. Você pode excluir informações particulares (por exemplo, contatos, correspondência, dados de conta do Google) e pastas selecionadas ou limpar todos os dados, incluindo os dados no cartão de memória.

Para excluir dados no seu dispositivo, você precisa enviar os seguintes comandos SMS:

  • wipe: <código> – para excluir dados específicos.
  • fullreset: <código> – para excluir todos os dados.

<Código> – este é o código secreto do Kaspersky Mobile Security no seu dispositivo.

As mensagens de SMS enviadas serão cobradas de acordo com o plano de preços do outro dispositivo.

Você pode ativar ou desativar a exclusão remota de dados específicos ou de todos os dados no seu dispositivo e selecionar informações específicas para serem excluídas.

Excluir dados selecionados

Ativar/desativar a exclusão remota de dados específicos do dispositivo (por comando SMS). Você pode selecionar informações particulares e pastas específicas para serem excluídas.

Se a função estiver permitida no dispositivo, ao receber o comando SMS correspondente, o aplicativo exclui os dados selecionados do dispositivo e responde com uma notificação de SMS.

As mensagens de SMS enviadas serão cobradas de acordo com o plano de preços do seu provedor de serviço.

Se a função estiver bloqueada no dispositivo, o aplicativo não pode executar o comando SMS para excluir os dados.

Excluir todos os dados

Ativar/desativar a exclusão remota de todos os dados do dispositivo utilizando o comando SMS fullreset: <código> (onde <código> é o código secreto para o Kaspersky Mobile Security no seu dispositivo).

Se a função estiver permitida no dispositivo, ao receber o comando SMS correspondente, o aplicativo bloqueia o dispositivo e responde com uma notificação de SMS.

As mensagens de SMS enviadas serão cobradas de acordo com o plano de preços do seu provedor de serviço.

Se a função estiver bloqueada no dispositivo, o aplicativo não pode executar o comando SMS para excluir todos os dados.

Informações particulares a serem excluídas

Você pode selecionar informações específicas para serem excluídas no recebimento do comando SMS wipe: <código> (onde <código> é o código secreto para o Kaspersky Mobile Security no seu dispositivo).

Os dados específicos podem ser selecionados para exclusão se a caixa de seleção Excluir dados selecionados estiver selecionada.

Dados de identidade

Exclusão de entradas do cartão SIM e dos Contatos, histórico de SMS, log de chamadas, calendário, configurações de conexão com a Internet, contas do usuário (exceto as contas do Google) no recebimento do comando SMS wipe: <código> (onde <código> é o código secreto para o Kaspersky Mobile Security no seu dispositivo).

Se a caixa de seleção estiver selecionada, o Kaspersky Mobile Security exclui seus dados particulares no recebimento do comando SMS. Se a caixa de seleção estiver desmarcada, o aplicativo não pode excluir seus dados particulares por comando SMS.
