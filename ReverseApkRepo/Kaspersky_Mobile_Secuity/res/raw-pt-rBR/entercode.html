Inserir código secreto

O código secreto é instalado quando o aplicativo é executado pela primeira vez. Ele é usado para prevenir o acesso não autorizado às configurações do aplicativo e para bloquear ou encontrar o dispositivo, excluir dados ou para ocultar remotamente informações confidenciais.

Se você esquecer o código secreto, é possível recuperá-lo.

O código secreto não pode ser recuperado se a função de recuperação do código secreto foi desativada durante a configuração inicial do aplicativo.

Tarefas padrão:

Recuperação do código secreto

  1. Abra Kaspersky Mobile Security > Menu > Recuperação de código secreto.
  2. Copie o código de identificação do dispositivo.
  3. Pressione Ir ou abra o seguinte site http://mobile.kaspersky.com/recover-code no dispositivo ou em um computador.
  4. Insira as seguintes informações nos campos apropriados:
    • O endereço de email que você especificou durante a configuração inicial do aplicativo com a finalidade de recuperação do código secreto
    • Código de identificação do dispositivo
  5. Um email com o código de recuperação será enviado para o seu endereço de email.
  6. Abra Kaspersky Mobile Security > Menu > Inserir código de recuperação.
  7. Insira o código de recuperação fornecido no email.
  8. Confirme o novo código secreto.
