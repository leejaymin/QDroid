Localizar

Localizar permite que você utilize o comando SMS locate: <código> para localizar remotamente seu dispositivo e receber suas coordenadas geográficas em uma mensagem de SMS ou email (<código> é o código secreto para o Kaspersky Mobile Security no seu dispositivo.).

Para enviar uma mensagem de email, o aplicativo envia um SMS para o número dedicado do provedor móvel MTS (Rússia). O custo do envio de um SMS para o número dedicado depende dos custos do provedor. O provedor móvel MTS (Rússia) envia em seguida as coordenadas atuais para o seu email.

A resposta do SMS será cobrada de acordo com o plano de preços do seu provedor.

O aplicativo pode localizar o dispositivo usando GPS ou redes móveis.

Para fornecer o posicionamento GPS, você deve permitir a utilização do GPS nas configurações do dispositivo. Quando o aplicativo receber o comando SMS especial, o módulo GPS será ativado. Se o aplicativo não encontrar as coordenadas do dispositivo usando o GPS, usará as redes móveis para determiná-las.

Você pode permitir ou bloquear a função de localizar remotamente o dispositivo e especificar um endereço de email para receber as coordenadas geográficas do dispositivo.

Ativar a Localização

Ativar/bloquear a localização do dispositivo por comando SMS.

Se a função estiver permitida no dispositivo, no recebimento do comando SMS correspondente, o aplicativo envia uma resposta com as coordenadas do dispositivo por SMS para o dispositivo que emitiu o comando e para o endereço de email.

As mensagens de SMS enviadas serão cobradas de acordo com o plano de preços do seu provedor de serviço.

Se a função estiver bloqueada no dispositivo, o aplicativo não pode executar o comando para localizar o dispositivo.

Enviar coordenadas do dispositivo

Por padrão, o aplicativo envia as coordenadas do dispositivo em uma resposta de SMS para o número que emitiu o comando SMS. Você pode adicionalmente especificar um endereço de email para receber a localização do dispositivo.

Endereço de email

O endereço de email para o qual o aplicativo envia as coordenadas do seu dispositivo.
