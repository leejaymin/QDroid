Área de verificação

Você pode escolher os tipos de arquivos que o aplicativo irá verificar quanto à presença de objetos maliciosos.

Ao escolher os tipos de arquivos a verificar, lembre-se do seguinte:

  • O código malicioso pode estar localizado em todos os tipos de arquivos e mais frequentemente encontra-se em arquivos executáveis (como arquivos APK).
  • Os objetos maliciosos podem estar ocultos nos arquivos comprimidos. Para detectar código malicioso em arquivos comprimidos, o aplicativo tem de descompactá-los, o que pode reduzir a velocidade da verificação do dispositivo.

Tipo de arquivos para verificar

Seleção de arquivos a serem analisados durante uma verificação completa do sistema de arquivos do dispositivo. Você pode escolher verificar os seguintes tipos de arquivos:

  • Todos os arquivos: o aplicativo verifica todos os tipos de arquivos (como música, texto, arquivos do sistema e arquivos executáveis). Esta configuração é selecionada por padrão.
  • Somente executáveis: o aplicativo verifica somente arquivos executáveis (programas e instaladores), tais como arquivos APK, e não verifica outros arquivos. Ao selecionar essa opção, a verificação do dispositivo é mais rápida.

Verificar arquivos

Ativa/desativa a verificação do conteúdo de arquivos quanto à presença de código malicioso. O programa verifica os seguintes formatos de arquivo comprimido: ZIP, JAR, JAD, SIS SISX CAB e APK. O programa analisa formatos de arquivo comprimido usados no Android e em outros sistemas operacionais móveis. Isso permite verificar um cartão de memória que é usado em vários dispositivos, por exemplo.

O programa descompacta arquivos comprimidos quando os verifica. Isso pode reduzir de forma significativa a velocidade da verificação do dispositivo.

Se você desativou a verificação de arquivos e ativou a verificação de programas ou de instaladores apenas (arquivos executáveis), o Kaspersky Mobile Security verifica apenas arquivos executáveis de acordo com as configurações determinadas. Se estiverem comprimidos em arquivos (tais como arquivos ZIP), o aplicativo os ignora.
