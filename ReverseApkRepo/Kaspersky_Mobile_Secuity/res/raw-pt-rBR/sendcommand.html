Envio de um comando SMS

Você pode executar remotamente as funções do Anti-Roubo ou ocultar informações confidenciais a partir de qualquer dispositivo com o Kaspersky Mobile Security instalado. Para fazer isso, você precisa enviar um comando SMS especial.

O comando SMS é enviado de forma encriptada e contém o código secreto do aplicativo instalado no outro dispositivo. O recebimento do comando SMS não é acusado no outro dispositivo.

O aplicativo executa os comandos SMS no outro dispositivo se as seguintes condições forem atendidas: o aplicativo passou por verificação com sucesso, a licença não expirou, o dispositivo recebe sinal de rede, o Kaspersky Mobile Security foi instalado no papel de um administrador do dispositivo e o início remoto da função está permitida no dispositivo.

O envio dos comandos SMS serão cobrados com a taxa atual do seu provedor de serviço móvel.

Número de telefone recebendo o comando SMS

O número do telefone do dispositivo para o qual o comando SMS é enviado.

Código secreto para o aplicativo no dispositivo que recebe o comando SMS

O código secreto do Kaspersky Mobile Security no dispositivo que recebe o comando SMS.
