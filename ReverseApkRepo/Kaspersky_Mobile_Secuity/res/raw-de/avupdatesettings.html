Update-Einstellungen

Sie können den automatischen Start eines Updates nach Zeitplan anpassen: Wählen Sie ein Intervall für den Start aus, sowie den Tag und die Uhrzeit des Starts.

Das Programm führt das automatische Update nur dann nach Zeitplan aus, wenn das Gerät eingeschaltet ist.

Zeitplan

Frequenz, mit der das Programm automatisch das Update startet. Folgende Modi sind vorgesehen:

  • 1x wöchentlich – das Update wird einmal pro Woche ausgeführt. Für diese Variante können Sie den Tag und die Uhrzeit des Updatestarts bestimmen.
  • 1x täglich – das Update wird täglich ausgeführt. Für diese Variante können Sie die Uhrzeit des Updatestarts bestimmen.
  • Aus – das Update wird nicht automatisch nach Zeitplan gestartet. Dabei können Sie das Update zu einem beliebigen, für Sie günstigen Zeitpunkt starten.

Starttag

Tag des Updatestarts. Der Parameter ist verfügbar, wenn für den Parameter Zeitplan der Wert 1x wöchentlich gesetzt ist.

Startzeit

Uhrzeit für den Updatestart. Der Parameter ist verfügbar, wenn für den Parameter Zeitplan der Wert 1x täglich oder 1x wöchentlich ausgewählt wurde.
