Scan-Einstellungen

Für einen auf Befehl zu startenden Scan des Geräts können Sie den Typ der vom Programm auf schädliche Objekte zu untersuchenden Dateien auswählen, das Entpacken von Archiven aktivieren, eine Aktion für gefundene Bedrohungen auswählen, und den Start einer Untersuchung des Dateisystems nach Zeitplan anpassen.

Untersuchungsbereich

Sie können die Dateitypen wählen, die das Programm auf das Vorhandensein schädlicher Objekte prüfen soll.

Aktion beim Fund einer Bedrohung

Sie können bestimmen, wie mit den Dateien verfahren werden soll, in denen das Programm eine Bedrohung entdeckt hat.

Scan nach Zeitplan

Sie können den automatischen Start einer vollständigen Untersuchung des Geräts nach Zeitplan anpassen: Wählen Sie das Intervall für den Start der Untersuchung sowie den Tag und die Uhrzeit des Starts (wenn nötig).

Damit das Programm den Scan automatisch starten kann, muss das Gerät zu dieser Zeit in Betrieb sein.

Das Programm startet den automatischen Scan im Hintergrundmodus nach Zeitplan. Wenn eine Bedrohung gefunden wird, führt das Programm die in den Untersuchungseinstellungen festgelegte Aktion aus.

Zeitplan

Frequenz, mit der das Programm automatisch einen vollständigen Scan des Dateisystems startet. Folgende Modi sind vorgesehen:

  • 1x wöchentlich – Das Programm startet die Untersuchung automatisch einmal pro Woche. Für diese Variante können Sie den Tag und die Uhrzeit des Untersuchungsstarts bestimmen.
  • 1x täglich – Das Programm startet die Untersuchung täglich automatisch zu einer festgelegten Uhrzeit Für diese Variante können Sie die Uhrzeit des Untersuchungsstarts bestimmen.
  • Aus – Das Programm startet keine Untersuchung automatisch nach Zeitplan. Dabei können Sie die Untersuchung zu einem beliebigen, für Sie günstigen Zeitpunkt starten.

Starttag

Tag für den Untersuchungsstart. Der Parameter ist verfügbar, wenn für den Parameter Zeitplan der Wert 1x wöchentlich gesetzt ist.

Startzeit

Uhrzeit für den Untersuchungsstart. Diese Einstellung ist verfügbar, wenn für den Parameter Zeitplan der Wert 1x täglich oder 1x wöchentlich gewählt ist.
