Anti-Virus

Anti-Virus kann Bedrohungen auf Ihrem Gerät erkennen und beseitigen. Dazu werden die Antiviren-Datenbanken des Programms und zusätzlich der Cloud-Dienst Kaspersky Security Network verwendet. Im Lieferumfang von Anti-Virus sind folgende Komponenten enthalten: Schutz, Scan und Update.

Der Schutz kann Bedrohungen in geöffneten Dateien erkennen, untersucht neue Programme und verhindert im Echtzeitmodus eine Infektion des Geräts.

Eine Untersuchung des gesamten Dateisystems, des Arbeitsspeichers oder bestimmter Ordner wird auf Befehl gestartet. Bei einer vollständigen Untersuchung wird das gesamte Dateisystem des Geräts auf schädliche Objekte untersucht. Eine Ordneruntersuchung bezieht sich auf einen konkreten Ordner. Die Funktionen Alles scannen und Ordner scannen finden Bedrohungen in Dateien, die installiert, aber nicht gestartet wurden, sowie Bedrohungen in Dateien, die im Moment ausgeführt werden. Bei einer Untersuchung des Speichers können nur Bedrohungen in jenen Dateien erkannt werden, die momentan ausgeführt werden.

Über die Funktion Update können Sie neue Antiviren-Datenbanken und aktuelle Programmversionen herunterladen.

Typische Aufgaben:

Untersuchung des gesamten Dateisystems des Geräts starten

Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Scan > Alles scannen.

Ordner scannen

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Scan > Ordner scannen.
  2. Wählen Sie einen Ordner, der untersucht werden soll.

Gerätespeicher scannen

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Scan > Speicher scannen.

Anpassen des Schutzmodus des Geräts

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Einstellungen.
  2. Bestimmen Sie den Schutzmodus. Wählen Sie dazu einen Wert für Schutzmodus aus.

Scan nach Zeitplan

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Einstellungen > Scan-Einstellungen.
  2. Wählen Sie ein Intervall für den Start der Untersuchung. Wählen Sie dazu einen Wert für Zeitplan.
  3. Bestimmen Sie einen Tag und eine Uhrzeit für den Untersuchungsstart. Wählen Sie Werte für Starttag und Startzeit aus.

Update der Antiviren-Datenbanken und Programm-Upgrade

Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Update.

Update der Datenbanken und Programmversionen nach Zeitplan

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anti-Virus > Einstellungen > Update-Einstellungen.
  2. Wählen Sie ein Intervall für den Start des Updates. Legen Sie dazu einen Wert für Zeitplan fest.
  3. Bestimmen Sie einen Tag und eine Uhrzeit für den Start des Updates. Wählen Sie Werte für Starttag und Startzeit aus.

Privatsphäre

Die Privatsphäre kann Ihre vertraulichen Kontakte sowie den Anrufverlauf und die SMS-Nachrichten für diese Kontakte verbergen.

Vertrauliche Kontakte werden in der Liste Verborgene Kontakte gespeichert.

Das Programm kann für vertrauliche Kontakte folgende Informationen verbergen:

  • Einträge in den Kontakten
  • gelesene eingehende SMS, gesendete SMS und SMS in den Entwürfen
  • Einträge in der Anrufliste
  • eingehende Anrufe und Benachrichtigungen über neue SMS

Wenn die Datenausblendung aktiviert ist, blockiert das Programm das Signal, das über den Empfang einer neuen SMS von einer vertraulichen Nummer informiert, und verbirgt die SMS im Eingangsordner. Das Programm blockiert einen Anruf, der von einer vertraulichen Nummer eingeht, und zeigt auf dem Display keine Informationen über den Anruf an. Der Anrufer hört in diesem Fall das "Besetzt"-Zeichen. Um die Anrufe und SMS-Nachrichten anzuzeigen, die eingegangen sind, während das Verbergen sensibler Daten aktiviert war, muss die Datenausblendung deaktiviert werden. Wenn Sie die Datenausblendung erneut aktivieren, werden die vertraulichen Informationen wieder verborgen.

Sie können die Datenausblendung vom Programm aus oder ferngesteuert mit einem SMS-Befehl aktivieren, der von einem anderen mobilen Gerät aus geschickt wird. Sie können entweder eine Standard-SMS mit einem speziellen Text schicken oder eine SMS aus Kaspersky Mobile Security schicken, falls das Programm auf dem anderen Gerät installiert ist.

Sie können auf Ihrem Gerät den ferngesteuerten Start der Datenausblendung erlauben oder verbieten. Wenn der ferngesteuerte Start der Datenausblendung verboten ist, kann diese Funktion nicht ferngesteuert durch einen SMS-Befehl gestartet werden.

Die Datenausblendung kann nur in dem Programm deaktiviert werden, das auf Ihrem Gerät installiert ist.

Typische Aufgaben:

Informationen für private Kontakte ausblenden

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Privatsphäre.
  2. Erstellen Sie eine Liste mit auszublendenden Kontakten (falls diese noch nicht vorhanden ist). Die Liste enthält Kontakte, für die Informationen verborgen werden sollen. Klicken Sie dazu auf Verborgene Kontakte > Hinzufügen.
  3. Kehren Sie zurück ins Hauptfenster von Kaspersky Mobile Security > Privatsphäre und stellen Sie den Schalter Infos verbergen auf aktiviert.

Verbergen von Informationen ferngesteuert starten

Führen Sie auf dem anderen mobilen Gerät eine der folgenden Aktionen aus:

  • Wenn auf dem anderen mobilen Gerät das Programm Kaspersky Mobile Security installiert ist, erstellen Sie mithilfe der SMS-Befehlsfunktion einen Befehl und senden Sie diesen an Ihr Gerät.
  • Erstellen Sie eine SMS mit dem speziellen Text hide:<Code> und senden Sie die SMS an Ihr Gerät. Dabei steht <Code> für den Geheimcode, der auf Ihrem Gerät festgelegt wurde.

Für das Senden einer SMS fallen Gebühren gemäß des Mobilfunktarifes an, der auf dem anderen mobilen Gerät gilt.

Liste für zu verbergende Kontakte anlegen

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Privatsphäre > Verborgene Kontakte.
  2. Führen Sie eine der folgenden Aktionen aus:
    • Klicken Sie auf Hinzufügen > Kontakt, um einen Kontakt aus dem Telefonbuch des Geräts hinzuzufügen.
    • Klicken Sie auf Hinzufügen > Telefonnummer, um eine Telefonnummer hinzuzufügen.

Diebstahlschutz

Diebstahlschutz schützt die Daten auf Ihrem Gerät vor unberechtigtem Zugriff und hilft Ihnen, das Gerät bei Verlust oder Diebstahl wiederzufinden.

Der Diebstahlschutz kann Ihr Gerät ferngesteuert sperren, auf dem Gerät gespeicherte Informationen löschen und das Gerät orten. Dazu werden von einem anderen mobilen Gerät aus entsprechende SMS-Befehle gesendet. Außerdem kann der Diebstahlschutz Ihr Gerät blockieren, falls die SIM-Karte gewechselt oder das Gerät ohne SIM-Karte eingeschaltet wird.

Die Diebstahlschutz-Funktionen werden durch einen speziellen SMS-Befehl gestartet. Sie können entweder von einem anderen Gerät aus eine Standard-SMS mit einem speziellen Text schicken, oder, falls Kaspersky Mobile Security auf dem anderen mobilen Gerät installiert ist, eine SMS aus diesem Programm schicken. Zum Senden des SMS-Befehls muss der Geheimcode des Programms Kaspersky Mobile Security bekannt sein, das auf Ihrem Gerät installiert ist.

Sie können auf Ihrem Gerät den ferngesteuerten Start der Diebstahlschutz-Funktionen erlauben oder verbieten. Wenn der Start der Diebstahlschutz-Funktion verboten ist, kann diese Funktion nicht ferngesteuert durch einen SMS-Befehl gestartet werden.

Typische Aufgaben:

Erlaubnis der ferngesteuerten Gerätesperre

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Diebstahlschutz > Gerätesperre.
  2. Aktivieren Sie das Kontrollkästchen Gerätesperre aktivieren.
  3. Geben Sie, falls erforderlich, im Feld Text beim Blockieren einen Text an, der auf dem Display des blockierten Geräts angezeigt werden soll.

Erlaubnis zur ferngesteuerten Datenlöschung

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Diebstahlschutz > Datenlöschung.
  2. Wählen Sie die Informationen aus, die nach dem Empfang eines Remotebefehls gelöscht werden sollen.
    • Aktivieren Sie zuerst das Kontrollkästchen Ausgewählte Daten löschen, damit die ausgewählten Daten ferngesteuert gelöscht werden können. Aktivieren Sie das Kontrollkästchen Persönliche Daten, damit persönliche Daten (wie Kontakte, SMS-Nachrichten und Kalender) gelöscht werden. Klicken Sie auf Ordner hinzufügen und wählen Sie die entsprechenden Ordner aus, wenn bestimmte Ordner ferngesteuert gelöscht werden sollen.
    • Aktivieren Sie das Kontrollkästchen Alle Daten löschen, um ferngesteuert alle Daten zu löschen und die Werkseinstellungen wiederherzustellen.

Erlaubnis zur ferngesteuerten Ortung des Geräts

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Diebstahlschutz > Ortung.
  2. Aktivieren Sie das Kontrollkästchen Ortung aktivieren.
  3. Geben Sie, falls erforderlich, eine E-Mail-Adresse an, unter der Sie über den Standort Ihres Geräts informiert werden möchten.

Erlaubnis zur Kontrolle über den Wechsel der SIM-Karte

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Diebstahlschutz > SIM-Kontrolle.
  2. Aktivieren Sie das Kontrollkästchen SIM-Kontrolle aktivieren.
  3. Geben Sie eine andere Telefonnummer oder Ihre E-Mail-Adresse an, an die die neue Telefonnummer beim Wechsel der SIM-Karte per SMS mitgeteilt werden soll.
  4. Aktivieren Sie bei Bedarf das Kontrollkästchen Blockieren, damit das Gerät gesperrt wird, wenn die SIM-Karte gewechselt oder das Gerät ohne SIM eingeschaltet wird. Geben Sie einen Meldungstext an, der auf dem Display des blockierten Geräts angezeigt werden soll.

Diebstahlschutz-Funktionen von einem anderen Gerät aus ferngesteuert starten

Führen Sie auf dem anderen mobilen Gerät eine der folgenden Aktionen aus:

  • Wenn auf dem anderen mobilen Gerät das Programm Kaspersky Mobile Security installiert ist, erstellen Sie mithilfe der SMS-Befehlsfunktion einen Befehl und senden Sie diesen an Ihr Gerät. Zum Erstellen des SMS-Befehls muss der Geheimcode des Programms Kaspersky Mobile Security bekannt sein, das auf Ihrem Gerät installiert ist.
  • Senden Sie eine SMS mit folgendem Text an Ihr Gerät:
    • lock:<Code> – zum Sperren des Geräts.
    • wipe:<Code> – zum Löschen von ausgewählten persönlichen Daten von dem Gerät.
    • fullreset:<Code> – zum Löschen aller Daten vom Gerät.
    • locate:<Code> – zur Ortung des Geräts.

    <Code> – Geheimcode für Kaspersky Mobile Security, der auf Ihrem Gerät festgelegt wurde.

Für das Senden einer SMS fallen für das sendende Gerät Gebühren entsprechend Ihres Mobilfunkgerätes an.

Anruf- und SMS-Filter

Der Anruf- und SMS-Filter verhindert unerwünschte eingehende Anrufe und SMS. Das Programm filtert Anrufe und SMS gemäß dem ausgewählten Filtermodus. Dazu dienen Listen mit erlaubten und verbotenen Kontakten.

Typische Aufgaben:

Blockieren von unerwünschten eingehenden Anrufen und SMS

  1. Legen Sie fest, von welchen Kontakten Anrufe und SMS gesperrt bzw. erlaubt werden sollen. Wählen Sie dazu einen Filtermodus aus. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anruf- und SMS-Filter > Modus: <aktueller Filtermodus>.
  2. Wählen Sie einen Wert für Modus für Anruf- und SMS-Filter aus:
    • Wählen Sie Verbotene Kontakte aus, damit nur Anrufe und SMS von verbotenen Kontakten gesperrt und von allen übrigen Nummern erlaubt werden.
    • Wählen Sie Erlaubte Kontakte aus, damit Anrufe und SMS von allen Kontakten gesperrt und nur von erlaubten Kontakten erlaubt werden.
    • Wählen Sie Standard aus, damit Anrufe und SMS von verbotenen Kontakten gesperrt und von erlaubten Kontakten erlaubt werden.
  3. Aktivieren Sie bei Bedarf die zusätzlichen Filtereinstellungen.

    Wenn Sie beispielsweise den Filtermodus Erlaubte Kontakte oder Standard ausgewählt haben, können Sie unabhängig vom ausgewählten Filtermodus automatisch Anrufe und SMS von Kontakten aus dem Telefonbuch des Geräts erlauben. Aktivieren Sie dazu das Kontrollkästchen Kontakte erlauben.

    Wenn Sie den Filtermodus Verbotene Kontakte oder Standard ausgewählt haben, können Sie unabhängig vom ausgewählten Filtermodus automatisch Anrufe und SMS von Nicht-Ziffern-Nummern verbieten. Aktivieren Sie dazu das Kontrollkästchen Nicht-Ziffern-Nummern blockieren.

  4. Erstellen Sie abhängig vom ausgewählten Modus eine Liste für erlaubte und verbotene Kontakte:
    • Um eine Liste für verbotene Kontakte zu erstellen, kehren Sie zurück ins Hauptfenster von Kaspersky Mobile Security > Anruf- und SMS-Filter > Verbotene Kontakte.
    • Um eine Liste für erlaubte Kontakte zu erstellen, kehren Sie zurück ins Hauptfenster von Kaspersky Mobile Security > Anruf- und SMS-Filter > Erlaubte Kontakte.

Kontakt zur Liste der erlaubten Kontakte hinzufügen

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anruf- und SMS-Filter > Erlaubte Kontakte
  2. Klicken Sie auf Hinzufügen.
  3. Wählen Sie einen Typ für die Ereignisse aus, die für einen Kontakt erlaubt werden sollen.
  4. Geben Sie die Telefonnummer eines Kontakts an, von dem Sie eingehende Informationen erhalten möchten. Füllen Sie dazu das Feld Erlaubte Telefonnummer aus.

    Eine Telefonnummer kann folgendermaßen hinzugefügt werden: Nummer manuell im Eingabefeld angeben, Anruf aus der Anrufliste oder SMS aus der Liste für eingehende Nachrichten auswählen. Klicken Sie auf , um eine Nummer aus der Anrufliste oder aus der SMS-Liste hinzuzufügen.

    Die Nummer kann mit einer Ziffer, einem Buchstaben oder dem Zeichen "+" beginnen und darf Ziffern und (oder) Buchstaben enthalten. Zur Angabe von Nummern sind auch die Masken "*" und "?" möglich (wobei "*" für eine beliebige Anzahl von Zeichen steht und "?" für ein beliebiges Zeichen). Wenn der Liste für verbotene Kontakte beispielsweise die Nummer *1234? hinzugefügt wurde, erlaubt der Anruf- und SMS-Filter die Anrufe und SMS von einer Nummer, in der auf die Ziffern 1234 ein beliebiges Zeichen folgt.

  5. Wenn Sie den Ereignistyp SMS ausgewählt haben, geben Sie eine Schlüsselphrase an, die darauf hinweist, dass eine empfangene SMS erwünscht ist. Füllen Sie dazu das Feld Erlaubter Text aus. Der Anruf- und SMS-Filter erlaubt von dieser Nummer nur SMS, die eine Schlüsselphrase enthalten, und verbietet alle übrigen SMS von dieser Nummer.

    Wenn Sie die SMS-Nachrichten von diesem Kontakt unabhängig von ihrem Inhalt erhalten möchten, lassen Sie das Feld Erlaubter Text leer.

Kontakt zur Liste der verbotenen Kontakte hinzufügen

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Anruf- und SMS-Filter > Verbotene Kontakte
  2. Klicken Sie auf Hinzufügen.
  3. Wählen Sie einen Typ für die Ereignisse aus, die Sie von diesem Kontakt für unerwünscht halten.
  4. Geben Sie die Telefonnummer eines verbotenen Kontakts an. Füllen Sie dazu das Feld Verbotene Telefonnummer aus.

    Eine Telefonnummer kann folgendermaßen hinzugefügt werden: Nummer manuell im Eingabefeld angeben, Anruf aus der Anrufliste oder SMS aus der Liste für eingehende Nachrichten auswählen. Klicken Sie auf , um eine Nummer aus der Anrufliste oder aus der SMS-Liste hinzuzufügen.

    Die Nummer kann mit einer Ziffer, einem Buchstaben oder dem Zeichen "+" beginnen und darf Ziffern und (oder) Buchstaben enthalten. Zur Angabe von Nummern sind auch die Masken "*" und "?" möglich (wobei "*" für eine beliebige Anzahl von Zeichen steht und "?" für ein beliebiges Zeichen). Wenn der Liste für verbotene Kontakte beispielsweise die Nummer *1234? hinzugefügt wurde, verbietet der Anruf- und SMS-Filter die Anrufe und SMS von einer Nummer, in der auf die Ziffern 1234 ein beliebiges Zeichen folgt.

  5. Wenn Sie den Ereignistyp SMS ausgewählt haben, geben Sie eine Schlüsselphrase an, die darauf hinweist, dass eine empfangene SMS unerwünscht ist. Füllen Sie dazu das Feld Verbotener Text aus. Der Anruf- und SMS-Filter blockiert von dieser Nummer nur SMS, die eine Schlüsselphrase enthalten, und erlaubt alle übrigen SMS von dieser Nummer.

    Wenn Sie unabhängig vom Nachrichteninhalt keine SMS-Nachrichten von diesem Kontakt erhalten möchten, lassen Sie das Feld Verbotener Text leer.

Web-Filter

Der Web-Filter blockiert schädliche Webseiten, deren Ziel darin besteht, schädlichen Code zu verbreiten, sowie gefälschte Webseiten (Phishing-Seiten), die Ihre persönlichen Daten wie PIN-Nummern für Online-Banking und Passwörter für Online-Auktionen oder Zahlungssysteme erschleichen und sich Zugang zu Ihren Finanzkonten verschaffen.

Der Web-Filter prüft eine Website, bevor sie aufgerufen wird. Dazu wird der Cloud-Dienst Kaspersky Security Network genutzt (ein spezieller Online-Service von Kaspersky Lab, der Informationen über die Zuverlässigkeit von Dateien, Programmen und Internetressourcen bereitstellt).

Web-Filter untersucht Webseiten nur im standardmäßigen Android-Browser. Andere Browser werden derzeit nicht unterstützt.

Bei seiner Aktivierung prüft Web-Filter automatisch, welcher Browser standardmäßig installiert ist. Wenn Web-Filter den installierten Standard-Browser nicht unterstützt, wird Ihnen vorgeschlagen, den Android-Browser als Standard-Browser festzulegen.

Um eine durchgehende Überprüfung von Webseiten zu gewährleisten, wird empfohlen, den standardmäßigen Android-Browser als Standard-Browser festzulegen.

Wenn Sie Ihren Standard-Browser nicht ändern möchten, verwenden Sie den standardmäßigen Android-Browser immer dann, wenn Sie persönliche Daten eingeben müssen und diese im Netz schützen möchten.

Starten Sie den standardmäßigen Android-Browser aus Kaspersky Mobile Security oder aus dem Programmmenü Ihres Geräts. Wenn Web-Filter aktiviert ist, blockiert er gefälschte Webseiten, noch bevor sie aufgerufen werden.

Typische Aufgaben:

Aktivierung des Web-Filters für eine durchgehende Überprüfung der Websites

  1. Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Web-Filter.
  2. Setzen Sie für das Optionsfeld Web-Filter aktivieren den Wert aktiviert.
  3. Klicken Sie auf Browser ändern (die Schaltfläche ist sichtbar, wenn der Web-Filter aktiviert ist und er die Webadressen im aktuellen Standardbrowser nicht untersucht).
  4. Befolgen Sie die Schritte des Assistenten zum Festlegen des Standard-Browsers.

Sichere Nutzung persönlicher Daten

Klicken Sie im Hauptfenster von Kaspersky Mobile Security auf Web-Filter > Browser öffnen.
