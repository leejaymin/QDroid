Erweitert

Sie können folgende Aktionen ausführen:

  • Empfang von Benachrichtigungen, Anzeige von Tooltips oder des Widgets anpassen
  • Geheimcode ändern
  • Berichte über das Programm anzeigen (beispielsweise Berichte über eine ausgeführte Untersuchung, über gefundene Bedrohungen, über blockierte SMS, Anrufe oder Webseiten)
  • Informationen über das Programm anzeigen

Erweiterte Einstellungen

Meldungen

Aktivieren / Deaktivieren von Pop-up-Meldungen zur Arbeit des Programms, die in der Statuszeile angezeigt werden. Dies sind zum Beispiel Meldungen, die Sie über den Fund einer Bedrohung oder den Ablauf der Gültigkeitsdauer der Lizenz benachrichtigen.

Ton

Tonsignale für Ereignisse aktivieren / deaktivieren, die bei der Arbeit des Programms eintreten. Beispiel: Das Programm informiert durch ein Tonsignal darüber, wenn eine Bedrohung gefunden wird.

Tooltips

Aktivieren / Deaktivieren der Anzeige von kurzen Informationen über eine Komponente, bevor die Komponente angepasst wird.

Widget

Der Farbindikator des Startbildschirm-Widgets signalisiert den Status, den der Schutz Ihres Geräts und die Datenausblendung besitzen. Außerdem dient er als Link zu den Programmeinstellungen.

Ein grüner Schild bedeutet, dass der empfohlene oder erweiterte Schutzmodus ausgewählt wurde. Ein grauer Schild bedeutet, dass der Schutz deaktiviert wurde. Grüner Hintergrund bedeutet, dass vertrauliche Informationen verborgen werden. Bei grauem Hintergrund werden vertrauliche Informationen angezeigt. Ein in ein gelbes Dreieck eingeschlossenes Ausrufezeichen bedeutet, dass die Lizenz abgelaufen ist.

Sie können erlauben / verbieten, dass das Widget auf dem Startbildschirm angezeigt wird.

Geheimcode ändern

Der Geheimcode des Programms wird beim ersten Programmstart festgelegt und dient folgenden Zwecken:

  • um einen unautorisierten Zugriff auf die Programmeinstellungen zu verhindern.
  • um die Diebstahlschutz-Funktionen und das Ausblenden von Kontakten ferngesteuert zu starten.

Sie können den Geheimcode ändern.

Kaspersky Mobile Security deinstallieren

Um Kaspersky Mobile Security zu entfernen, müssen Sie den Geheimcode des Programms eingeben.

Allgemeine Informationen

Berichte

Die Ereignisse, die in Anti-Virus, im Web-Filter und im Anruf- und SMS-Filter auftreten, werden in Berichten protokolliert.

Die Berichte sind nach dem Erstellungszeitpunkt angeordnet. Sie können die Berichte für eine bestimmte Programmkomponente anzeigen. Es werden maximal 50 Berichtseinträge gespeichert. Wenn die Anzahl der Einträge in einem Bericht die Zahl 50 übersteigt, werden ältere Berichte gelöscht und durch neue ersetzt.
