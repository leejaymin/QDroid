Datenlöschung

Mit der Datenlöschung können Sie durch einen SMS-Befehl ferngesteuert Informationen von Ihrem Gerät löschen, damit sie nicht in fremde Hände geraten. Sie können persönliche Daten (beispielsweise Kontakte, Nachrichten oder die Daten Ihres Google-Benutzerkontos), zuvor festgelegte Ordner oder alle Daten einschließlich des Inhalts der Speicherkarte löschen.

Um Daten auf Ihrem Gerät zu löschen, müssen Sie folgende SMS-Befehle senden:

  • wipe:<Code> – zum Löschen von konkreten Daten.
  • fullreset:<Code> – zum Löschen aller Daten.

<Code> – Geheimcode für Kaspersky Mobile Security auf Ihrem Gerät.

Für das Senden eines SMS-Befehls an Ihr Gerät fallen Gebühren gemäß des Mobilfunktarifes an, der auf dem anderen mobilen Gerät gilt.

Sie können den ferngesteuerten Start für das Löschen konkreter Daten oder aller Daten auf Ihrem Gerät erlauben oder verbieten. Außerdem können Sie konkrete Daten zum Löschen auswählen.

Ausgewählte Daten löschen

Erlaubnis / Verbot für das ferngesteuerte Löschen von konkreten Daten vom Gerät (mithilfe eines speziellen SMS-Befehls). Sie können persönliche Informationen oder konkrete Ordner zum Löschen auswählen.

Wenn die Funktion auf dem Gerät erlaubt ist, löscht das Programm die ausgewählten Daten vom Gerät und sendet eine Antwort-SMS, nachdem es einen SMS-Befehl zum Löschen konkreter Daten erhalten hat.

Für das Senden einer Antwort-SMS fallen Gebühren gemäß Ihres Mobilfunktarifes an.

Wenn die Funktion auf dem Gerät verboten ist, kann das Programm den SMS-Befehl zum Löschen konkreter Daten nicht ausführen.

Alle Daten löschen

Erlaubnis / Verbot für das ferngesteuerte Löschen aller Daten vom Gerät, wenn der SMS-Befehl fullreset:<Code> gelöscht werden sollen (<Code> ist der Geheimcode für Kaspersky Mobile Security auf Ihrem Gerät).

Wenn die Funktion auf dem Gerät erlaubt ist, löscht das Programm alle Daten und sendet eine Antwort-SMS, nachdem es einen SMS-Löschbefehl für alle Daten erhalten hat.

Für das Senden einer Antwort-SMS fallen Gebühren gemäß Ihres Mobilfunktarifes an.

Wenn die Funktion auf dem Gerät verboten ist, kann das Programm den SMS-Befehl zum Löschen aller Daten nicht ausführen.

Zu löschende persönliche Informationen

Sie können konkrete Informationen auswählen, die nach dem Empfang des SMS-Befehls wipe:<Code> gelöscht werden sollen (<Code> ist der Geheimcode für Kaspersky Mobile Security auf Ihrem Gerät).

Konkrete Daten können zum Löschen ausgewählt werden, wenn das Kontrollkästchen Datenlöschung aktivieren aktiviert ist.

Persönliche Daten

Nach dem Empfang des SMS-Befehls wipe:<Code> werden folgende Elemente gelöscht: Einträge in den Kontakten und auf der SIM-Karte, gesamter SMS-Verlauf, Anrufliste, Kalender, Einstellungen der Internetverbindung, Benutzerkonten (außer Google-Konto) (<Code> ist der Geheimcode für Kaspersky Mobile Security auf Ihrem Gerät).

Wenn dieses Kontrollkästchen aktiviert ist, löscht das Programm Ihre persönlichen Daten, nachdem ein spezieller SMS-Befehl empfangen wurde. Wenn dieses Kontrollkästchen deaktiviert ist, löscht das Programm Ihre persönlichen Daten nicht auf SMS-Befehl.

Ordner.

Liste der Ordner, die nach dem Empfang des SMS-Befehls wipe:<Code> gelöscht werden sollen (<Code> ist der Geheimcode für Kaspersky Mobile Security auf Ihrem Gerät).

Sie können Ordner hinzufügen, die sich auf einer Speicherkarte befinden. Außerdem können Sie mithilfe des Kontextmenüs Ordner aus dieser Liste löschen.
