Aktion beim Fund einer Bedrohung

Sie können bestimmen, wie mit den Dateien verfahren werden soll, in denen das Programm eine Bedrohung entdeckt hat.

Desinfizieren

Das Programm versucht, die gefundene infizierte Datei zu desinfizieren. Wenn dieses Kontrollkästchen aktiviert ist, versucht das Programm beim Fund einer infizierten Datei zunächst, sie zu desinfizieren. Wenn das Programm die infizierte Datei nicht desinfizieren kann, wird die Aktion ausgeführt, die im Block Wenn Desinfektion nicht möglich festgelegt wurde.

Wenn das Kontrollkästchen nicht aktiviert ist, führt das Programm beim Fund einer infizierten Datei sofort die Aktion aus, die im Block Wenn Desinfektion nicht möglich festgelegt wurde.

Wenn Desinfektion nicht möglich

Aktion, die das Programm mit einer infizierten Datei ausführt, wenn eine Desinfektion nicht möglich ist oder das Kontrollkästchen Desinfizieren deaktiviert ist. Folgende Aktionen stehen zur Auswahl:

  • Aktion erfragen – Das Programm fordert Sie auf, eine Aktion für die infizierte Datei auszuwählen.
  • Löschen – Das Programm löscht die infizierte Datei.
  • Überspringen – Das Programm führt keine Aktionen mit einer infizierten Datei aus und löscht sie nicht vom Gerät.
