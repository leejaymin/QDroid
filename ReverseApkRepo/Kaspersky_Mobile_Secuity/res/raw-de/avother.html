Anti-Virus-Einstellungen

Anti-Virus schützt das Gerät vor Viren und anderen Schadprogrammen. Im Lieferumfang von Anti-Virus sind folgende Komponenten enthalten: Schutz, Scan und Update.

Der Schutz wird beim Hochfahren des Betriebssystems aktiviert und befindet sich permanent im Arbeitsspeicher des Geräts. Der Schutz verhindert im Echtzeitmodus eine Infektion des Geräts. Er dient der Untersuchung neuer Programme sowie aller Dateien, die geöffnet, gespeichert und ausgeführt werden.

Der Scan wird auf Befehl gestartet und hilft, Bedrohungen auf Ihrem Gerät zu finden und zu beseitigen (beispielsweise geöffnete, nicht geöffnete oder und nicht installierte, in ein Archiv gepackte Dateien, die Bedrohungen enthalten). Mit Kaspersky Mobile Security können Sie den Inhalt Ihres Geräts vollständig oder teilweise untersuchen. Wählen Sie für die Untersuchung das gesamte Dateisystem des Geräts, den Inhalt des Arbeitsspeichers des Geräts oder einen spezifischen Ordner.

Das Update der Antiviren-Datenbanken und der Version von Kaspersky Mobile Security (im Weiteren "Update" genannt) gewährleistet, dass der Schutz Ihres Geräts sich immer auf dem neuesten Stand befindet.

Kaspersky Mobile Security verwendet bei der Suche nach schädlichen Programmen die Antiviren-Datenbanken. Die Antiviren-Datenbanken des Programms enthalten eine Beschreibung aller momentan bekannten schädlichen Programme und entsprechende Desinfektionsmethoden sowie eine Beschreibung sonstiger unerwünschter Objekte. Deshalb wird ausdrücklich empfohlen, die Datenbanken regelmäßig zu aktualisieren. Die Antiviren-Datenbanken des Programms gelten als veraltet, wenn seit dem letzten Update mehr als 15 Tage vergangen sind. In diesem Fall sinkt das Schutzniveau.

Neben der Aktualisierung der Antiviren-Datenbanken ermöglicht Kaspersky Mobile Security auch ein Upgrade der Programmversion. Updates der Programmversion beheben Schwachstellen von Kaspersky Mobile Security, fügen neue Funktionen hinzu und optimieren bestehende Funktionen.

Für ein regelmäßiges Update ist eine gültige Programmlizenz erforderlich. Wenn die Programmlizenz abgelaufen ist, steht das Update nicht mehr zur Verfügung.

Kaspersky Mobile Security führt das Update über die Updateserver von Kaspersky Lab durch (spezielle Webseiten, auf denen die Updates aller Produkte von Kaspersky Lab veröffentlicht werden).

Schutzmodus

Der Schutz wird beim Hochfahren des Betriebssystems aktiviert und befindet sich permanent im Arbeitsspeicher des Geräts. Der Schutz dient der Überprüfung aller installierten Programme sowie aller Dateien, die geöffnet, gespeichert und ausgeführt werden.

Schutzmodus

Modus, in dem der Schutz das Dateisystem im Hintergrundmodus auf Veränderungen überwacht und das Dateisystem auf schädliche Objekte untersucht. Folgende Modi sind vorgesehen:

  • Erweitert Der Schutz untersucht alle Dateien auf schädliche Objekte. Eine Untersuchung erfolgt bei jeder Aktion, die mit Dateien ausgeführt wird (beispielsweise beim Öffnen, Starten, Kopieren, Speichern, Verschieben oder Ändern einer Datei). Außerdem untersucht der Schutz neue Programme sofort nach ihrer Installation.
  • Empfohlen. Der Schutz untersucht nur neue Programme auf das Vorhandensein schädlicher Objekte, während andere Dateien und Dateivorgänge nicht geprüft werden. Neue Programme werden vom Schutz nur einmal untersucht, und zwar sofort nach der Installation.
  • Aus. Schutz deaktiviert.

    Sie können den Schutz für einige Zeit deaktivieren, um die Belastung Ihres Geräts während der Durchführung mehrerer Aufgaben zu reduzieren.

    Die Kaspersky-Lab-Spezialisten warnen davor, den Schutz zu deaktivieren, weil dies zur Infektion Ihres Computers und zu Datenverlust führen kann.

    Die Deaktivierung des Schutzes beeinträchtigt nicht die Ausführung anderer Aufgaben von Anti-Virus (beispielsweise Untersuchung des Dateisystems auf Viren oder Update der Antiviren-Datenbanken des Programms).

Cloud-Untersuchung

Zusätzliche Untersuchung neuer Programme, bevor sie zum ersten Mal gestartet werden. Dazu wird der Cloud-Dienst von Kaspersky Security Network verwendet (spezieller Online-Service von Kaspersky Lab, der Informationen über die Zuverlässigkeit von Dateien, Programmen und Internetressourcen bereitstellt). Die Untersuchung über den Cloud-Dienst dient dazu, Ihre Daten vor neuen, unbekannten Bedrohungen zu schützen.

Wenn dieses Kontrollkästchen aktiviert ist, überprüft Kaspersky Mobile Security neue Programme vor ihrem ersten Start zusätzlich mit dem Cloud-Dienst Kaspersky Security Network. Wenn dieses Kontrollkästchen deaktiviert ist, führt Kaspersky Mobile Security keine zusätzliche Untersuchung durch.

Diese Einstellung ist verfügbar, wenn für Schutzmodus die Variante Empfohlen oder Erweitert ausgewählt wurde.

Adware, Autodialer usw 

Erkennen von Adware sowie anderen Programmen, die Ihr Gerät beschädigen können. Zu diesem Programmtyp gehören beispielsweise Programme, die Zugriffsrechte für persönliche Informationen, für das Senden von Nachrichten oder für die Ortung Ihres Standorts anfordern.

Wenn dieses Kontrollkästchen aktiviert ist, erkennt Kaspersky Mobile Security Schwachstellen dieses Typs und benachrichtigt Sie darüber. Wenn dieses Kontrollkästchen deaktiviert ist, überspringt Kaspersky Mobile Security Bedrohungen dieses Typs.

Komponenten anpassen

Sie können die Einstellungen für die Komponenten von Anti-Virus separat anpassen.

Schutz-Einstellungen

Sie können die Dateitypen auswählen, die das Programm auf das Vorhandensein schädlicher Objekte prüfen soll, und festlegen, wie mit Dateien verfahren werden soll, in denen das Programm eine Bedrohung gefunden hat.

Untersuchungseinstellungen

Sie können festlegen, welche Dateitypen untersucht werden sollen. Außerdem können Sie eine Aktion festlegen, die das Programm mit schädlichen Objekten ausführen soll (beispielsweise Desinfizieren oder Löschen), und den automatischen Start der Untersuchung nach Zeitplan anpassen.

Update-Einstellungen

Sie können den automatischen Start des Updates nach Zeitplan anpassen.
