Untersuchungsbereich

Sie können die Dateitypen wählen, die das Programm auf das Vorhandensein schädlicher Objekte prüfen soll.

Bei der Wahl des Dateityps zur Untersuchung sollten Sie Folgendes beachten:

  • Schädlicher Code kann sich in jeder Datei befinden, kommt aber am häufigsten in ausführbaren Dateien vor (beispielsweise in Dateien des Formats APK).
  • Schädliche Objekte verbergen sich manchmal in Archiven. Um einen schädlichen Code in einem Archiv entdecken zu können, muss das Programm dieses entpacken, was die Geschwindigkeit der Überprüfung des Geräts verringern kann.

Typ der zu untersuchenden Dateien

Auswahl der Dateien, die das Programm bei einer vollständigen Untersuchung des Dateisystems des Geräts analysieren soll. Sie können folgende Dateitypen für die Untersuchung auswählen:

  • Alle Dateien – Das Programm prüft alle Dateitypen (beispielsweise Audio-, Text-, Systemdateien oder ausführbare Dateien). Dieser Parameter ist standardmäßig ausgewählt.
  • Nur ausführbare – Das Programm untersucht nur ausführbare Dateien (d.h. Programme oder Software-Installer), wie beispielsweise Dateien im Format APK, und überspringt alle anderen Dateien. Wenn Sie diese Variante wählen, kann die Geräteprüfung mit einer höheren Geschwindigkeit durchgeführt werden.

Archive untersuchen

Aktivieren / Deaktivieren der Überprüfung der Archivinhalte auf das Vorhandensein von schädlichem Code. Das Programm ist in der Lage, Archive folgender Formate zu untersuchen: ZIP, JAR, JAD, SIS, SISX, CAB und APK. Das Programm analysiert die Archive aller Formate, die unter Android und anderen Betriebssystemen für Mobilgeräte verwendet werden. Auf diese Weise können Sie zum Beispiel eine Speicherkarte überprüfen, die für mehrere Mobilgeräte genutzt wird.

Während der Überprüfung werden die Archive vom Programm entpackt. Das kann die Geschwindigkeit der Durchführung der Überprüfung erheblich verringern.

Wenn Sie die Untersuchung von Archiven deaktiviert und Software-Installer (ausführbare Dateien) für die Untersuchung ausgesucht haben, untersucht Kaspersky Mobile Security nur ausführbare Dateien. Wenn diese in Archive gepackt sind (beispielsweise Archive im ZIP-Format), werden sie nicht vom Programm untersucht.
