Einstellungen für das Widget

Sie können das Widget zum Startbildschirm des Geräts hinzufügen. Außerdem können Sie erlauben, dass der Status für das Verbergen von persönlichen Kontakten und dazugehörenden Informationen im Widget angezeigt werden soll (durch eine Änderung der Hintergrundfarbe im Widget).

Widget anzeigen

Erlaubnis / Verbot für das Hinzufügen des Widgets zum Startbildschirm des Geräts. Wenn dieses Kontrollkästchen aktiviert ist, wird das Widget in der Liste der verfügbaren Widgets angezeigt. Wenn dieses Kontrollkästchen deaktiviert ist, wird das Widget nach einem Neustart des Geräts nicht mehr in der Liste der verfügbaren Widgets angezeigt und kann nicht zum Startbildschirm des Geräts hinzugefügt werden.

Status der Datenausblendung anzeigen

Erlaubnis / Verbot der Statusanzeige für das Verbergen von vertraulichen Informationen im Startbildschirm-Widget.

Wenn dieses Kontrollkästchen aktiviert ist, wird durch die Farbe des Widgets der Status signalisiert, den das Verbergen von vertraulichen Daten besitzt. Grüner Hintergrund bedeutet, dass vertrauliche Informationen verborgen werden. Bei grauem Hintergrund werden vertrauliche Informationen angezeigt.

Wenn dieses Kontrollkästchen deaktiviert ist, informiert das Widget nicht über den Status der Datenausblendung. Die Hintergrundfarbe des Widgets ist unabhängig vom Status der Datenausblendung immer grau.
