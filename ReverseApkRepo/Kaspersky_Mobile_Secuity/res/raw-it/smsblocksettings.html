Blocco Dispositivo

Blocco Dispositivo consente di utilizzare il comando SMS lock: <password> per bloccare il dispositivo in remoto e impedire l'accesso non autorizzato ai dati (<password> è la password segreta di Kaspersky Mobile Security nel dispositivo).

Per i comandi SMS in uscita vengono applicate le tariffe previste per l'altro dispositivo mobile.

È possibile abilitare o disabilitare il blocco remoto del dispositivo.

Consenti Blocco Dispositivo

Consente di abilitare o disabilitare il blocco in remoto del dispositivo mediante un comando SMS.

Se la funzione è consentita nel dispositivo, alla ricezione del comando SMS corrispondente, l'applicazione blocca il dispositivo e risponde con una notifica via SMS.

Per i messaggi SMS in uscita vengono applicate le tariffe previste dall'operatore di telefonia mobile.

Se la funzione è disabilitata nel dispositivo, l'applicazione non può eseguire il comando SMS per l'abilitazione di Blocco Dispositivo.

Testo quando bloccato

Testo del messaggio che verrà visualizzato sullo schermo del dispositivo bloccato. Per impostazione predefinita, viene visualizzato un messaggio standard.
