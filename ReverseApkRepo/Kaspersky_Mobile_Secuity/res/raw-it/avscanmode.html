Ambito della scansione

È possibile scegliere i tipi di file da esaminare alla ricerca di oggetti dannosi.

Durante la scelta dei tipi di file di cui eseguire la scansione, tenere presenti i seguenti aspetti:

  • Il codice dannoso può essere presente in tutti i tipi di file e nella maggior parte dei casi è contenuto nei file eseguibili (come i file APK).
  • Gli oggetti dannosi possono essere nascosti all'interno di archivi. Allo scopo di rilevare il codice dannoso negli archivi, l'applicazione deve decomprimerli, il che riduce la velocità della scansione del dispositivo.

Tipi di file per la scansione

Selezione di file da analizzare durante una scansione completa del file system del dispositivo. È possibile scegliere di esaminare i seguenti tipi di file:

  • Tutti i file: l'applicazione esamina tutti i tipi di file (ad esempio, file di musica, di testo, di sistema ed eseguibili). Rappresenta l'impostazione predefinita.
  • Solo eseguibili: l'applicazione esamina solo i file eseguibili (programmi e pacchetti di installazione), come i file APK, e non esegue la scansione di altri file. Selezionando questa opzione, la scansione del dispositivo è più veloce.

Scansione archivi

Consente di abilitare o disabilitare la scansione del contenuto degli archivi alla ricerca di codice dannoso. Il programma esamina i seguenti formati di archivi: ZIP, JAR, JAD, SIS SISX, CAB e APK. Il programma analizza i formati di archivio utilizzati in Android e altri sistemi operativi per dispositivi mobili. Questo consente ad esempio di eseguire la scansione di una scheda di memoria utilizzata in diversi dispositivi.

Quando il programma esamina gli archivi, li decomprime. Questo può ridurre in modo significativo la velocità di scansione del dispositivo.

Se è stata disabilitata la scansione degli archivi ed è stata abilitata solo la scansione di programmi e pacchetti di installazione (file eseguibili), Kaspersky Mobile Security esamina solo i file eseguibili in base alle impostazioni configurate. Se tali file sono compressi in archivi (come archivi ZIP), essi vengono ignorati dall'applicazione.
