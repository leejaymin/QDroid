Impostazioni di protezione

È possibile selezionare i tipi di file esaminati dall'applicazione alla ricerca di oggetti dannosi e specificare le azioni da eseguire sui file in cui vengono rilevate minacce.

La sezione delle impostazioni è disponibile se Modalità Protezione è impostata su Estesa.

Tipi di file per la scansione

Selezione dei tipi di file esaminati dal componente Protezione alla ricerca di oggetti dannosi. Il componente Protezione esamina i tipi di file selezionati quando su tali file viene eseguita un'azione (ad esempio, quando vengono aperti, copiati, eseguiti, spostati, salvati o modificati). È possibile selezionare uno dei seguenti tipi di file:

  • Tutti i file: l'applicazione esamina tutti i tipi di file (ad esempio, file di musica, di testo, di sistema ed eseguibili).
  • Solo eseguibili: l'applicazione esegue solo i file eseguibili (programmi), come i file APK.

Azione se viene rilevata una minaccia

Consente di selezionare l'azione eseguita dall'applicazione in caso di rilevamento di una minaccia. È possibile selezionare una delle seguenti azioni:

  • Elimina: l'applicazione elimina il file infetto.
  • Ignora: l'applicazione non esegue nessuna azione sul file infetto e non lo elimina dal dispositivo.

Indipendentemente dall'azione selezionata, l'applicazione segnala che è stata rilevata una minaccia.
