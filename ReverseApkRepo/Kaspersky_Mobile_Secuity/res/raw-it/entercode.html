Immissione della password segreta

La password segreta viene installata durante la prima esecuzione dell'applicazione. Viene utilizzata per impedire l'accesso non autorizzato alle impostazioni dell'applicazione, bloccare o localizzare il dispositivo, eliminare i dati presenti in esso o nascondere le informazioni riservate in remoto.

Se si dimentica la password segreta, è possibile ripristinarla.

Non è possibile eseguire il ripristino della password segreta se la funzione di ripristino della password segreta è stata disabilitata durante la configurazione iniziale dell'applicazione.

Attività standard:

Ripristino della password segreta

  1. Aprire Kaspersky Mobile Security > Menu > Ripristino password segreta.
  2. Copiare il codice di identificazione del dispositivo.
  3. Premere Vai oppure aprire il seguente sito Web http://mobile.kaspersky.com/recover-code-lang?lang=It nel dispositivo o in un computer.
  4. Specificare le seguenti informazioni nei campi appropriati:
    • L'indirizzo e-mail specificato durante la configurazione iniziale dell'applicazione per il ripristino della password segreta
    • Codice di identificazione del dispositivo
  5. All'indirizzo e-mail specificato verrà inviato un messaggio e-mail con la password di ripristino.
  6. Aprire Kaspersky Mobile Security > Menu > Immettere la password di ripristino.
  7. Immettere la password di ripristino contenuta nel messaggio e-mail.
  8. Confermare la nuova password segreta.
