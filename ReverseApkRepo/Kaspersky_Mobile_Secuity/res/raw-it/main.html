Kaspersky Mobile Security

Anti-Virus

Anti-Virus consente di rilevare e neutralizzare le minacce nel dispositivo utilizzando i database anti-virus dell'applicazione ed eventualmente il servizio cloud Kaspersky Security Network. Anti-Virus include le seguenti funzioni: protezione, scansione e aggiornamento.

Protezione rileva le minacce nei file aperti, esamina le nuove applicazioni e protegge in tempo reale i dispositivi dalle infezioni.

Scansione esegue scansioni su richiesta dell'intero file system, della memoria operativa o delle cartelle. Scansione completa verifica la presenza di oggetti dannosi nell'intero file system, mentre Scansione cartella esamina una cartella specifica. Scansione completa e Scansione cartella rilevano le minacce presenti nei file installati ma non ancora aperti, nonché le minacce presenti nei file attualmente aperti. Scansione memoria rileva solo le minacce presenti nei file attualmente aperti.

L'aggiornamento consente di scaricare i nuovi database anti-virus e le nuove versioni dell'applicazione.

Attività standard:

Scansione completa del file system del dispositivo

Nella finestra principale di Kaspersky Mobile Security fare click su Anti-Virus >Scansione>Scansione completa.

Scansione cartella

  1. Nella finestra principale di Kaspersky Mobile Security fare click su Anti-Virus >Scansione>Scansione cartella.
  2. Selezionare le cartelle da esaminare.

Scansione memoria

  1. Nella finestra principale di Kaspersky Mobile Security fare click su Anti-Virus >Scansione>Scansione memoria.

Modifica della modalità di Protezione

  1. Nella finestra principale di Kaspersky Mobile Security premere Anti-Virus > Impostazioni.
  2. Selezionare la modalità Protezione. A tale scopo, selezionare un valore per l'impostazione Modalità Protezione.

Scansioni pianificate

  1. Nella finestra principale di Kaspersky Mobile Security fare click su Anti-Virus > Impostazioni > Impostazioni di scansione.
  2. Selezionare la frequenza per l'avvio delle scansioni. A tale scopo, selezionare un valore per l'impostazione Pianificazione.
  3. Selezionare un giorno e l'ora per l'avvio delle scansioni. Selezionare i valori per le impostazioni Giorno di inizio e Ora di inizio.

Aggiornamenti dei database anti-virus e della versione

Nella finestra principale di Kaspersky Mobile Security fare click su Anti-Virus > Aggiornamento.

Aggiornamenti pianificati dei database e della versione

  1. Nella finestra principale di Kaspersky Mobile Security fare click su Anti-Virus > Impostazioni > Impostazioni di aggiornamento.
  2. Selezionare la frequenza per l'avvio degli aggiornamenti. A tale scopo, selezionare un valore per l'impostazione Pianificazione.
  3. Selezionare un giorno e l'ora per l'avvio degli aggiornamenti. Selezionare i valori per le impostazioni Giorno di inizio e Ora di inizio.

Protezione privacy

Protezione privacy consente di nascondere i contatti riservati e la relativa cronologia di chiamate e messaggi SMS.

I contatti riservati vengono salvati nell'elenco Contatti da nascondere.

L'applicazione è in grado di nascondere le seguenti informazioni per i contatti riservati:

  • Voci della rubrica
  • Messaggi SMS letti o inoltrati e i messaggi in Bozze
  • Elenco cronologia chiamate
  • Chiamate in entrata e notifiche sui nuovi messaggi SMS

Quando Protezione privacy è abilitato, l'applicazione blocca il segnale e le notifiche alla ricezione di un nuovo messaggio SMS proveniente da un numero privati e nasconde il messaggio nella Posta in arrivo. L'applicazione blocca le chiamate in entrata provenienti dai numeri privati e non visualizza le informazioni sulle chiamate in entrata sullo schermo. Come risultato, il chiamante riceve un segnale di occupato. Per visualizzare le chiamate e i messaggi SMS in entrata mentre Protezione privacy è abilitato, è necessario disabilitare l'applicazione. Non appena Protezione privacy viene abilitato nuovamente, le informazioni riservate tornano a essere nascoste.

È possibile abilitare Protezione privacy dall'applicazione, o in remoto inviando un comando SMS da un altro dispositivo mobile. È possibile inviare un messaggio SMS standard con una password speciale oppure inviare un messaggio SMS da Kaspersky Mobile Security se l'applicazione è installata nell'altro dispositivo.

È possibile consentire o bloccare l'avvio remoto di Protezione privacy nel dispositivo. Se l'avvio remoto di Protezione privacy è bloccato, la funzione non può essere avviata in remoto utilizzando un comando SMS.

È possibile disabilitare Protezione privacy solo per l'applicazione installata nel dispositivo.

Attività standard:

Occultamento delle informazioni relative ai contatti

  1. Nella finestra principale di Kaspersky Mobile Security fare click su Protezione privacy.
  2. Creare un elenco dei contatti da nascondere se non è già stato creato in precedenza. L'elenco contiene i contatti di cui si desidera nascondere i dettagli. Per eseguire questa operazione, premere Contatti da nascondere > Aggiungi.
  3. Tornare alla finestra principale di Kaspersky Mobile Security >Protezione privacy e spostare l'interruttore Nascondi informazioni su abilitato.

Avvio remoto di Protezione privacy

Eseguire una delle seguenti operazioni in un altro dispositivo mobile:

  • Se nell'altro dispositivo è installato Kaspersky Mobile Security, creare e inviare un comando SMS al proprio dispositivo.
  • Creare e inviare un messaggio SMS con il testo hide:<password>, dove <password> è la password segreta del dispositivo.

Per i messaggi SMS in uscita vengono applicate le tariffe dell'operatore di telefonia mobile dell'altro dispositivo.

Creazione di un elenco di contatti da nascondere

  1. Nella finestra principale di Kaspersky Mobile Security, premere Protezione privacy > Contatti da nascondere.
  2. Eseguire una delle seguenti operazioni:
    • Per aggiungere un contatto dalla rubrica del dispositivo, premere Aggiungi > Contatto.
    • Per aggiungere un numero di telefono, premere Aggiungi > Numero di telefono.

Antifurto

Antifurto protegge le informazioni memorizzate nel dispositivo dagli accessi non autorizzati e consente di localizzare il dispositivo in caso di furto o smarrimento.

Antifurto consente di bloccare il dispositivo mediante un comando SMS da un altro dispositivo mobile, eliminare le informazioni presenti o localizzare il dispositivo su una mappa. Antifurto consente inoltre di bloccare il dispositivo se la scheda SIM viene sostituita o se il dispositivo viene attivato senza una scheda SIM.

Le funzioni Antifurto vengono avviate mediante uno speciale comando SMS. È possibile inviare un messaggio SMS standard con un testo speciale da un altro dispositivo mobile oppure, se nel dispositivo è installato Kaspersky Mobile Security, inviare un messaggio SMS dall'applicazione. Per inviare un comando SMS, è necessario conoscere la password segreta di Kaspersky Mobile Security installato nel dispositivo.

È possibile consentire o bloccare l'avvio remoto delle funzioni Antifurto sul dispositivo. Se l'avvio remoto di Antifurto è bloccato, la funzione non può essere avviata mediante un comando SMS.

Attività standard:

Consentire il blocco del dispositivo in remoto

  1. Nella finestra principale di Kaspersky Mobile Security premere Antifurto >Blocco dispositivo.
  2. Selezionare la casella Consenti Blocco Dispositivo.
  3. Se necessario, nella sezione Testo quando bloccato, immettere il messaggio da visualizzare sullo schermo del dispositivo bloccato.

Consentire l'eliminazione dei dati in remoto

  1. Nella schermata principale di Kaspersky Mobile Security, premere Antifurto >Cancellazione Dati.
  2. Selezionare le informazioni da eliminare alla ricezione del comando remoto:
    • Per eliminare i dati selezionati in remoto, selezionare la casella Elimina i dati selezionati. Per eliminare i dati personali (come contatti, messaggi SMS e calendario), selezionare la casella Dati personali. Per eliminare le cartelle in remoto, premere Aggiungi cartella e selezionare le cartelle desiderate.
    • Per eliminare tutti i dati in remoto e ripristinare le impostazioni di fabbrica nel dispositivo, selezionare la casella Elimina tutti i dati.

Consentire la localizzazione del dispositivo in remoto

  1. Nella finestra principale di Kaspersky Mobile Security premere Antifurto >Localizzazione GPS.
  2. Selezionare la casella Consenti Localizzazione.
  3. Se necessario, specificare un indirizzo e-mail per la ricezione delle coordinate del dispositivo.

Sostituzione della scheda SIM

  1. Nella finestra principale di Kaspersky Mobile Security, premere Antifurto >SIM-Watch.
  2. Selezionare la casella Abilita SIM-Watch.
  3. Specificare un numero di telefono aggiuntivo o un indirizzo e-mail per la ricezione di un nuovo numero di telefono via SMS o e-mail in caso di sostituzione della scheda SIM.
  4. Se necessario, selezionare la casella Blocco per bloccare il dispositivo in caso di sostituzione della scheda SIM o di attivazione del dispositivo senza una scheda SIM. Specificare il testo del messaggio da visualizzare sullo schermo del dispositivo bloccato.

Avvio remoto di Antifurto da un altro dispositivo

Eseguire una delle seguenti operazioni in un altro dispositivo mobile:

  • Se nell'altro dispositivo è installato Kaspersky Mobile Security, creare e inviare un comando SMS al proprio dispositivo. Per creare un comando SMS, è necessario conoscere la password segreta di Kaspersky Mobile Security installato nel dispositivo.
  • Inviare un messaggio SMS al dispositivo con il seguente testo speciale:
    • lock: <password> – per bloccare il dispositivo
    • wipe: <password> – per eliminare i dati personali selezionati dal dispositivo
    • fullreset: <password> – per eliminare tutti i dati dal dispositivo
    • locate: <password> – per localizzare il dispositivo

    <Password>  – è la password segreta di Kaspersky Mobile Security nel dispositivo.

Per i messaggi SMS in uscita vengono applicate le tariffe dell'operatore di telefonia mobile dell'altro dispositivo.

Filtro chiamate/SMS

Filtro chiamate/SMS consente di bloccare le chiamate e i messaggi SMS in entrata indesiderati. L'applicazione filtra le chiamate e i messaggi SMS in base agli elenchi dei contatti bloccati e consentiti, a seconda della modalità filtro selezionata.

Attività standard:

Blocco delle chiamate e dei messaggi SMS in entrata indesiderati

  1. Selezionare i contatti per i quali si desidera bloccare chiamate e messaggi SMS. Per eseguire questa operazione, scegliere la modalità di filtraggio. Nella finestra principale di Kaspersky Mobile Security premere Filtro chiamate/SMS > Modalità: <modalità di filtraggio corrente>.
  2. Selezionare il valore per l'impostazione Filtro chiamate/SMS:
    • Per bloccare solo le chiamate e i messaggi SMS provenienti dai contatti bloccati e consentire le chiamate e i messaggi SMS provenienti dagli altri contatti, selezionare Contatti bloccati.
    • Per bloccare le chiamate e i messaggi SMS provenienti da tutti i contatti e consentire solo le chiamate e i messaggi SMS provenienti dai contatti consentiti, selezionare Contatti consentiti.
    • Per bloccare le chiamate e i messaggi SMS provenienti dai contatti bloccati e consentire le chiamate e i messaggi SMS provenienti dai contatti consentiti, selezionare Standard.
  3. Se necessario, abilitare le impostazioni di filtraggio avanzate.

    Per esempio, se è stata selezionata la modalità di filtraggio Contatti consentiti o Standard, è possibile consentire automaticamente le chiamate e i messaggi SMS provenienti dai contatti della rubrica del dispositivo, indipendentemente dalla modalità di filtraggio abilitata. Per eseguire questa operazione, selezionare la casella Consenti contatti.

    Se è stata selezionata la modalità di filtraggio Contatti bloccati o Standard, è possibile bloccare automaticamente i messaggi SMS e le chiamate provenienti da mittenti non numerici, indipendentemente dalla modalità di filtraggio abilitata. Per eseguire questa operazione, selezionare la casella Blocca mittenti non numerici.

  4. A seconda della modalità selezionata, creare l'elenco dei contatti consentiti e quello dei contatti bloccati:
    • Per creare l'elenco dei contatti bloccati, tornare alla finestra principale. Nella finestra principale di Kaspersky Mobile Security > Filtro chiamate/SMS > Contatti bloccati.
    • Per creare l'elenco dei contatti consentiti, tornare alla finestra principale. Nella finestra principale di Kaspersky Mobile Security > Filtro chiamate/SMS > Contatti consentiti.

Aggiunta di un contatto all’elenco dei contatti consentiti

  1. Nella finestra principale di Kaspersky Mobile Security, fare click su Filtro chiamate/SMS > Contatti consentiti.
  2. Fare click su Aggiungi.
  3. Selezionare i tipi di eventi che si desidera ricevere dal contatto.
  4. Immettere il numero di telefono del contatto da cui si desidera ricevere informazioni. Per eseguire questa operazione, completare il campo Numero di telefono consentito.

    È possibile aggiungere un numero di contatto all'elenco in uno dei seguenti modi: specificando il numero manualmente nel campo di immissione, selezionando una chiamata del contatto nell'elenco delle chiamate o un messaggio SMS del contatto nell'elenco dei messaggi in entrata. Per aggiungere un numero dall'elenco delle chiamate o dei messaggi, fare click su .

    Il numero di telefono deve comprendere solo caratteri alfanumerici; può iniziare con una cifra, una lettera o essere preceduto dal simbolo "+". Per i numeri è inoltre possibile utilizzare i caratteri jolly "*" o "?" (dove "*" rappresenta un numero qualsiasi di simboli e "?" qualsiasi singolo simbolo). Ad esempio, se il numero *1234? viene aggiunto alla Lista consentiti, Filtro chiamate/SMS consente le chiamate o i messaggi SMS provenienti da numeri che contengono 1234 seguito da qualsiasi carattere.

  5. Se fra i tipi di eventi è stato selezionato SMS, immettere una frase chiave per indicare che un messaggio SMS in entrata è desiderato. Per eseguire questa operazione, completare il campo Testo consentito. Filtro chiamate/SMS consente gli SMS provenienti da questo numero solo se contengono la frase chiave, mentre blocca gli altri SMS provenienti da questo numero.

    Se si desidera ricevere messaggi SMS con qualsiasi contenuto da questo numero, lasciare vuoto il campo Testo consentito per questa voce.

Aggiunta di un contatto all’elenco dei contatti bloccati

  1. Nella finestra principale di Kaspersky Mobile Security, fare click su Filtro chiamate/SMS > Contatti bloccati.
  2. Fare click su Aggiungi.
  3. Selezionare i tipi di eventi provenienti dal contatto considerati indesiderati (spam).
  4. Immettere il numero di telefono del contatto bloccato. Per eseguire questa operazione, completare il campo Numero di telefono bloccato.

    È possibile aggiungere un numero di contatto all'elenco in uno dei seguenti modi: specificando il numero manualmente nel campo di immissione, selezionando una chiamata del contatto nell'elenco delle chiamate o un messaggio SMS del contatto nell'elenco dei messaggi in entrata. Per aggiungere un numero dall'elenco delle chiamate o dei messaggi, fare click su .

    Il numero di telefono deve comprendere solo caratteri alfanumerici; può iniziare con una cifra, una lettera o essere preceduto dal simbolo "+". Per i numeri è inoltre possibile utilizzare i caratteri jolly "*" o "?" (dove "*" rappresenta un numero qualsiasi di simboli e "?" qualsiasi singolo simbolo). Ad esempio, se il numero *1234? viene aggiunto alla Lista bloccati, Filtro chiamate/SMS blocca le chiamate o i messaggi SMS provenienti da numeri che contengono 1234 seguito da qualsiasi carattere.

  5. Se fra i tipi di eventi è stato selezionato SMS, immettere una frase chiave per indicare che un messaggio SMS in entrata è indesiderato. Per eseguire questa operazione, completare il campo Testo bloccato. Filtro chiamate/SMS blocca gli SMS provenienti da questo numero solo se contengono la frase chiave, mentre consente gli altri SMS provenienti da questo numero.

    Se non si desidera ricevere messaggi SMS con qualsiasi contenuto da questo numero, lasciare vuoto il campo Testo bloccato per questa voce.

Protezione Web

Protezione Web blocca i siti Web dannosi che distribuiscono codice dannoso e siti Web contraffatti (di phishing), i quali possono rubare le informazioni riservate dell'utente, come password di online banking, di aste online ed e-money, e accedere ai conti personali.

Protezione Web controlla i siti Web prima che vengano aperti utilizzando il servizio cloud Kaspersky Security Network (uno speciale servizio online di Kaspersky Lab che consente di ottenere informazioni sull'affidabilità di file, applicazioni e risorse Internet).

Protezione Web controlla solo i siti Web nel browser Android standard. Negli altri browser i siti Web non vengono controllati.

Quando è abilitato, Protezione Web controlla automaticamente il browser predefinito. Se il browser predefinito corrente non è supportato, Protezione Web richiede di passare al browser Android standard.

Per garantire che i siti Web vengano controllati costantemente, è consigliabile impostare il browser Android standard come browser predefinito.

Se non si desidera modificare il browser predefinito corrente, avviare il browser Android standard quando si desidera utilizzare la rete in modo sicuro.

È possibile avviare il browser Android standard da Kaspersky Mobile Security o dal menu dell'applicazione del dispositivo. Se Protezione Web è abilitato, bloccherà i siti Web contraffatti prima che vengano aperti.

Attività standard:

Abilitazione di Protezione Web per il controllo continuo dei siti Web

  1. Nella finestra principale di Kaspersky Mobile Security fare click su Protezione Web.
  2. Impostare il pulsante di opzione Abilita Protezione Web su abilitato.
  3. Premere Cambia browser (questo pulsante è visualizzato se Protezione Web è abilitato e se non esamina i siti Web nel browser predefinito corrente).
  4. Seguire le indicazioni della procedura guidata per l'impostazione del browser predefinito.

Utilizzo sicuro dei dati personali

Nella finestra principale di Kaspersky Mobile Security, fare click su Protezione Web > Apri browser.

Avanzate

È possibile eseguire le seguenti operazioni:

  • Configurazione della ricezione di notifiche e della visualizzazione di richieste dell'applicazione o del widget.
  • Modifica della password segreta.
  • Visualizzazione di rapporti sul funzionamento dell'applicazione (ad esempio, rapporti sulle scansioni effettuate, sulle minacce rilevate, sui messaggi SMS e sulle chiamate o i siti Web bloccati).
  • Visualizzazione di informazioni sull’applicazione.
