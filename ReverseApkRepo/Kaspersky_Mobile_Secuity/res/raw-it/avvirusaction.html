Azione se viene rilevata una minaccia

È possibile selezionare le azioni che l'applicazione deve eseguire sui file in cui vengono rilevate minacce.

Disinfetta

L'applicazione tenta di disinfettare il file infetto rilevato. Se la casella è selezionata, l'applicazione tenta in primo luogo di disinfettare un file infetto rilevato. Se l'applicazione non è in grado di disinfettare il file infetto, esegue l'azione specificata nella sezione Se la disinfezione fallisce.

Se la casella è deselezionata e viene rilevato un file dannoso, l'applicazione eseguirà immediatamente l'azione specificata nella sezione Se la disinfezione fallisce.

Se la disinfezione fallisce

L'azione eseguita dall'applicazione sul file infetto se la disinfezione fallisce o se la casella Disinfetta è deselezionata. È possibile selezionare una delle seguenti azioni:

  • Chiedi all'utente: l'applicazione richiede all'utente di selezionare l'azione da eseguire sul file infetto.
  • Elimina: l'applicazione elimina il file infetto.
  • Ignora: l'applicazione non esegue nessuna azione sul file infetto e non lo elimina dal dispositivo.
