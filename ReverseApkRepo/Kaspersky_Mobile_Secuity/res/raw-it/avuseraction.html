Minaccia rilevata

L'applicazione richiede all'utente di selezionare un'azione da eseguire sui file in cui viene rilevata una minaccia.

Le minacce rilevate vengono segnalate all'utente nelle seguenti circostanze:

  • Se Anti-Virus rileva una minaccia in un nuovo programma installato durante l'esecuzione del componente Protezione o durante una scansione del dispositivo.
  • Se Anti-Virus rileva una minaccia in un file e Anti-Virus è impostato su Chiedi all'utente per Se la disinfezione fallisce (finestra principale di Kaspersky Mobile Security > Anti-Virus > Impostazioni > Impostazioni di scansione > Azione se viene rilevata una minaccia).

Elimina

L'applicazione elimina l'oggetto infetto.

Ignora

L'applicazione non esegue nessuna azione sul file infetto e non lo elimina dal dispositivo.
