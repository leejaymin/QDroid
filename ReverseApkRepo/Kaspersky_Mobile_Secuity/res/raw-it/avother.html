Anti-Virus: impostazioni

Anti-Virus garantisce la protezione del dispositivo da virus ed altri malware. Anti-Virus include le seguenti funzioni: protezione, scansione e aggiornamento.

Il componente Protezione è abilitato all'avvio del sistema operativo e viene eseguito costantemente nella memoria del dispositivo. Il componente Protezione protegge in tempo reale il dispositivo dalle infezioni, esaminando tutti i file aperti, salvati ed eseguiti e tutte le nuove applicazioni.

Scansione viene avviato su richiesta per rilevare ed eliminare le minacce nel dispositivo (come file infetti aperti, file non aperti o disinstallati in archivi). Kaspersky Mobile Security consente di eseguire una scansione completa o parziale del contenuto del dispositivo. È possibile scegliere di eseguire la scansione dell'intero file system del dispositivo, del contenuto della memoria interna o di una specifica cartella.

Gli aggiornamenti dei database anti-virus e della versione dell'applicazione di Kaspersky Mobile Security (di seguito denominati "aggiornamenti") assicurano che la protezione del dispositivo sia sempre aggiornata.

Kaspersky Mobile Security utilizza i database anti-virus durante la ricerca del malware. I database anti-virus dell'applicazione contengono le descrizioni di tutti i malware attualmente noti e dei metodi per la loro disinfezione, nonché le descrizioni di altri oggetti dannosi indesiderati. Per questo motivo, è consigliabile aggiornare periodicamente i database anti-virus. Se sono trascorsi più di 15 giorni dall'ultimo aggiornamento, i database vengono considerati non aggiornati. La protezione risulterà pertanto meno affidabile.

Oltre ai database anti-virus, Kaspersky Mobile Security consente di aggiornare la versione dell'applicazione. Gli aggiornamenti della versione dell'applicazione rimuovono le vulnerabilità da Kaspersky Mobile Security, aggiungono nuove funzioni migliorano quelle esistenti.

Per gli aggiornamenti periodici è richiesta una licenza attiva per l'utilizzo dell'applicazione. Se la licenza è scaduta, gli aggiornamenti non sono disponibili.

Kaspersky Mobile Security esegue gli aggiornamenti dai server degli aggiornamenti di Kaspersky Lab (speciali siti Web che contengono gli aggiornamenti per tutti prodotti di Kaspersky Lab).

Modalità Protezione

Il componente Protezione è abilitato all'avvio del sistema operativo e viene eseguito costantemente nella memoria del dispositivo. Questo componente consente di eseguire la scansione di tutti i file aperti, salvati e avviati e di tutte le applicazioni installate.

Modalità Protezione

La modalità utilizzata dal componente Protezione per tenere traccia in background delle modifiche nel file system e per eseguirne la scansione alla ricerca di oggetti dannosi. Sono disponibili le seguenti modalità:

  • Estesa. Il componente Protezione verifica la presenza di eventuali oggetti dannosi quando su tali oggetti viene eseguita un'azione (ad esempio, quando vengono aperti, copiati, eseguiti, spostati, salvati o modificati). Il componente Protezione esamina le nuove applicazioni al momento dell'installazione.
  • Consigliata. Il componente Protezione verifica la presenza di oggetti dannosi solo nei nuovi programmi installati e non esegue la scansione di altri file o delle azioni ad essi relative. Il componente Protezione esamina i nuovi programmi al momento dell'installazione.
  • Disabilitato. Il componente Protezione è disabilitato.

    È possibile disabilitare temporaneamente il componente Protezione per ridurre il carico sul dispositivo durante l'esecuzione di diverse attività contemporaneamente.

    Gli specialisti di Kaspersky Lab sconsigliano di disabilitare il componente Protezione, in quanto ciò può causare l'infezione del dispositivo e perdite di dati.

    La disabilitazione del componente Protezione non influisce sull'esecuzione delle altre attività di Anti-Virus, come la scansione del file system alla ricerca di virus o l'aggiornamento dei database anti-virus.

Cloud Security Scan

Viene eseguita un'ulteriore scansione dei nuovi programmi installati utilizzando il servizio cloud Kaspersky Security Network (uno speciale servizio online di Kaspersky Lab che contiene informazioni sull'affidabilità di file, applicazioni e risorse Internet). La scansione tramite il servizio cloud consente di proteggere i dati dalle minacce nuove e sconosciute.

Se la casella è selezionata, Kaspersky Mobile Security esegue ulteriori controlli delle nuove applicazioni installate prima che vengano avviate per la prima volta utilizzando il servizio cloud Kaspersky Security Network. Se la casella è deselezionata, Kaspersky Mobile Security non esegue ulteriori controlli.

Questa impostazione è disponibile se Modalità Protezione è impostata su Consigliata o Estesa.

Adware, auto-dialer, altri 

È stato aggiunto il rilevamento di adware e altre applicazioni potenzialmente pericolose per il dispositivo. Questa categoria include le applicazioni che tentano di accedere ai dati personali e di inviare messaggi SMS o le coordinate relative alla posizione.

Se la casella è selezionata, Kaspersky Mobile Security rileva tali applicazioni e informa l'utente in caso di rilevamento. Se la casella è deselezionata, Kaspersky Mobile Security ignora le applicazioni di questa categoria.

Impostazioni del componente

È possibile configurare singolarmente ciascun componente Anti-Virus.

Impostazioni di protezione

È possibile selezionare i tipi di file esaminati dall'applicazione alla ricerca di oggetti dannosi e specificare le azioni da eseguire sui file in cui vengono rilevate minacce.

Impostazioni di scansione.

È possibile specificare i tipi di file da esaminare. Inoltre, è possibile impostare un'azione da eseguire sugli oggetti dannosi (ad esempio, disinfezione o eliminazione) e configurare scansioni automatiche pianificate.

Impostazioni di aggiornamento

È possibile configurare gli aggiornamenti automatici pianificati.
