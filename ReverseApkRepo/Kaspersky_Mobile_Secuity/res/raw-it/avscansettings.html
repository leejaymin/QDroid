Impostazioni di scansione

Per la scansione su richiesta del dispositivo, è possibile selezionare i tipi di file da esaminare per verificare la presenza di eventuali oggetti dannosi, abilitare la decompressione degli archivi, selezionare un'azione da eseguire al rilevamento di una minaccia e configurare le scansioni pianificate del file system.

Ambito della scansione

È possibile scegliere i tipi di file da esaminare alla ricerca di oggetti dannosi.

Azione se viene rilevata una minaccia

È possibile selezionare le azioni che l'applicazione deve eseguire sui file in cui vengono rilevate minacce.

Scansioni pianificate

È possibile configurare una pianificazione per avviare automaticamente una scansione completa del dispositivo: scegliere la frequenza di avvio della scansione, nonché la data e il giorno di avvio (se necessario).

Perché la scansione venga avviata automaticamente, il dispositivo deve essere acceso.

L'applicazione avvia la scansione automatica pianificata in background. Quando viene rilevato un oggetto dannoso, l'applicazione esegue l'azione selezionata nelle impostazioni del dispositivo.

Pianificazione

La frequenza con cui viene eseguita la scansione completa del file system. Sono disponibili le seguenti opzioni:

  • Settimanale: l'applicazione avvia automaticamente la scansione una volta alla settimana. Per questa opzione è possibile scegliere il giorno e l'ora di avvio della scansione.
  • Giornaliera: l'applicazione avvia automaticamente la scansione ogni giorno all'ora specificata. Per questa opzione è possibile scegliere l'ora di avvio della scansione.
  • Disabilitata: l'applicazione non esegue automaticamente una scansione pianificata. Questo consente di eseguire la scansione in qualsiasi momento.

Giorno di inizio

Giorno selezionato per la scansione. Questa impostazione è disponibile se è selezionato Settimanale per l'impostazione Pianificazione.

Ora di avvio

Ora per la scansione. Questa impostazione è disponibile se è selezionato Giornaliero o Settimanale per l'impostazione Pianificazione.
