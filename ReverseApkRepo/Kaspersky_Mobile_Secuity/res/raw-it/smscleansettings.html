Cancellazione Dati

Cancellazione Dati consente di eliminare le informazioni presenti nel dispositivo in remoto mediante un comando SMS, per impedire l'appropriazione da parte di utenti malintenzionati. È possibile eliminare le informazioni personali (ad esempio, contatti, messaggi, dati di account Google) e le cartelle selezionate, oppure cancellare tutti i dati, inclusi quelli contenuti nella scheda di memoria.

Per eliminare i dati presenti nel dispositivo, è necessario inviare i seguenti comandi SMS:

  • wipe: <password> – per eliminare dati specifici.
  • fullreset: <password> – per eliminare tutti i dati.

<Password> – è la password segreta di Kaspersky Mobile Security nel dispositivo.

Per i comandi SMS in uscita vengono applicate le tariffe previste per l'altro dispositivo mobile.

È possibile abilitare o disabilitare l'eliminazione in remoto di dati specifici o di tutti i dati presenti nel dispositivo, nonché selezionare informazioni specifiche da eliminare.

Elimina i dati selezionati

Consente di abilitare o disabilitare l'eliminazione in remoto di dati specifici dal dispositivo (mediante un comando SMS). È possibile selezionare informazioni personali e cartelle specifiche da eliminare.

Se la funzione è consentita nel dispositivo, alla ricezione del comando SMS corrispondente, l'applicazione elimina i dati selezionati nel dispositivo e risponde con una notifica via SMS.

Per i messaggi SMS in uscita vengono applicate le tariffe previste dall'operatore di telefonia mobile.

Se la funzione è disabilitata nel dispositivo, l'applicazione non può eseguire il comando SMS per l'eliminazione dei dati.

Elimina tutti i dati

Consente di abilitare o disabilitare l'eliminazione remota di tutti dati dal dispositivo mediante il comando SMS fullreset: <password> (dove <password> è la password segreta di Kaspersky Mobile Security nel dispositivo).

Se la funzione è consentita nel dispositivo, alla ricezione del comando SMS corrispondente, l'applicazione esegue il comando e risponde con una notifica via SMS.

Per i messaggi SMS in uscita vengono applicate le tariffe previste dall'operatore di telefonia mobile.

Se la funzione è bloccata nel dispositivo, l'applicazione non può eseguire il comando SMS per l'eliminazione di tutti i dati.

Dati personali da eliminare

È possibile selezionare informazioni specifiche da eliminare alla ricezione del comando SMS wipe: <password> (dove <password> è la password segreta di Kaspersky Mobile Security nel dispositivo).

È possibile selezionare dati specifici da eliminare se la casella Elimina dati selezionati è selezionata.

Dati di identità

Eliminazione di contatti e voci della scheda SIM, cronologia SMS, report sulle chiamate, calendario, impostazioni di connessione a Internet e account utente (tranne gli account di Google) alla ricezione del comando SMS wipe: <password> (dove <password> è la password segreta di Kaspersky Mobile Security nel dispositivo).

Se la casella è selezionata, Kaspersky Mobile Security elimina i dati personali alla ricezione del comando SMS. Se la casella è deselezionata, l'applicazione non può eliminare i dati personali mediante un comando SMS.

Cartelle

Elenco delle cartelle da eliminare alla ricezione del comando SMS wipe: <password> (dove <password> è la password segreta di Kaspersky Mobile Security nel dispositivo).

È possibile aggiungere cartelle salvate nella scheda di memoria. È inoltre possibile utilizzare il menu di scelta rapida per l'eliminazione delle cartelle dall'elenco.
