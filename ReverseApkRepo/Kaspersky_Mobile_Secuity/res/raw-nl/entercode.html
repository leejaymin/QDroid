De geheime code invoeren

De geheime code wordt geïnstalleerd wanneer het programma voor het eerst wordt uitgevoerd. De code wordt gebruikt om onbevoegde toegang tot de programma-instellingen te voorkomen, het apparaat te blokkeren of te zoeken, gegevens op het apparaat te verwijderen, of vertrouwelijke gegevens op afstand te verbergen.

Als u de geheime code bent vergeten, kunt u deze herstellen.

De geheime code kan niet worden hersteld als de herstelfunctie voor de geheime code werd uitgeschakeld tijdens de initiële configuratie van het programma.

Standaardtaken:

Herstel van geheime code

  1. Open Kaspersky Mobile Security > Menu > Herstel geheime code.
  2. Kopieer de identificatiecode van het apparaat.
  3. Druk op Ga of open de volgende website http://mobile.kaspersky.com/recover-code-lang?lang=Nl op het apparaat of op een computer.
  4. Voer de volgende informatie in de gepaste velden in:
    • E-mailadres dat u hebt opgegeven tijdens de initiële configuratie van het programma voor een eventueel herstel van de geheime code
    • Identificatiecode van het apparaat
  5. Een e-mail met de herstelcode wordt verzonden naar uw e-mailadres.
  6. Open Kaspersky Mobile Security > Menu > Voer herstelcode in.
  7. Voer de herstelcode in die is opgegeven in de e-mail.
  8. Bevestig de nieuwe geheime code.
