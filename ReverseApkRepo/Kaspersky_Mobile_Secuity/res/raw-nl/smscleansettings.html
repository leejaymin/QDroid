Data Wipe

Met Data Wipe kunt u gegevens van uw apparaat op afstand verwijderen met een sms-opdracht zodat deze gegevens niet in de verkeerde handen vallen. U kunt persoonlijke informatie (bijvoorbeeld contacten, berichten, Google-accountgegevens) en geselecteerde mappen verwijderen, of alle gegevens, inclusief gegevens op de geheugenkaart, wissen.

Als u gegevens op uw apparaat wilt verwijderen, moet u de volgende sms-opdrachten verzenden:

  • wipe: <code> – om specifieke gegevens te verwijderen.
  • fullreset: <code> – om alle gegevens te verwijderen.

<Code> – dit is de geheime code van Kaspersky Mobile Security op uw apparaat.

Uitgaande sms-opdrachten worden volgens de tarieven van het abonnement voor het andere mobiele apparaat in rekening gebracht.

U kunt de verwijdering op afstand van specifieke gegevens of alle gegevens op uw apparaat in- of uitschakelen, alsook specifieke gegevens selecteren die moeten worden verwijderd.

Geselecteerde gegevens verwijderen

Schakel het op afstand verwijderen van specifieke gegevens op het apparaat (per sms-opdracht) in of uit. U kunt persoonlijke gegevens en specifieke mappen selecteren die moeten worden verwijderd.

Als de functie is toegestaan op het apparaat, verwijdert het programma de geselecteerde gegevens op het apparaat bij ontvangst van de desbetreffende sms-opdracht en stuurt het een reactie in een sms.

Uitgaande sms-berichten worden volgens de tarieven van uw mobiele serviceaanbieder in rekening gebracht.

Als de functie is geblokkeerd op het apparaat, kan het programma de sms-opdracht voor de verwijdering van gegevens niet uitvoeren.

Alle gegevens verwijderen

Schakel de optie in of uit die alle gegevens op het apparaat op afstand verwijdert met de sms-opdracht fullreset: <code> (waarbij <code> de geheime code van Kaspersky Mobile Security op uw apparaat is).

Als de functie is toegestaan op het apparaat, voert het programma deze uit bij ontvangst van de desbetreffende sms-opdracht en stuurt het een reactie in een sms.

Uitgaande sms-berichten worden volgens de tarieven van uw mobiele serviceaanbieder in rekening gebracht.

Als de functie is geblokkeerd op het apparaat, kan het programma de sms-opdracht voor de verwijdering van alle gegevens niet uitvoeren.

Te verwijderen persoonlijke gegevens

U kunt specifieke gegevens selecteren die moeten worden verwijderd bij ontvangst van de sms-opdracht wipe: <code> (waarbij <code> de geheime code van Kaspersky Mobile Security op uw apparaat is).

U kunt specifieke gegevens selecteren die moeten worden verwijderd als het selectievakje Geselecteerde gegevens verwijderen is geselecteerd.

Identiteitsgegevens

Verwijdering van Contacten en vermeldingen op de simkaart, geschiedenis van sms-berichten, log met oproepen, agenda, instellingen van de internetverbinding, gebruikersaccounts (behalve Google-accounts) bij ontvangst van de sms-opdracht wipe: <code> (waarbij <code> de geheime code van Kaspersky Mobile Security op uw apparaat is).

Als dit selectievakje is geselecteerd, verwijdert Kaspersky Mobile Security uw persoonlijke gegevens bij ontvangst van de sms-opdracht. Als het selectievakje niet is geselecteerd, kan het programma uw persoonlijke gegevens niet verwijderen per sms-opdracht.

Mappen

De lijst met te verwijderen mappen bij ontvangst van de sms-opdracht wipe: <code> (waarbij <code> de geheime code van Kaspersky Mobile Security op uw apparaat is).

U kunt mappen toevoegen die op de geheugenkaart zijn opgeslagen. U kunt ook het contextmenu gebruiken om mappen uit de lijst te verwijderen.
