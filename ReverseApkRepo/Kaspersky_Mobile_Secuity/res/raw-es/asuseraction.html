Llamada entrante/SMS entrante

La aplicación solicita al usuario que seleccione la acción que se realiza cuando se recibe un mensaje SMS o cuando finaliza una llamada entrante.

La aplicación le solicita que seleccione una acción si se cumplen las siguientes condiciones de manera concurrente: El modo Estándar está establecido en el modo Filtro de llamadas y SMS y el número no se encuentra en la lista de contactos permitidos ni bloqueados.

Bloquear siempre

La aplicación marca las llamadas (o mensajes SMS) de los contactos como no deseados. La aplicación agrega el número o el contacto que emite la llamada (o mensaje SMS) a la lista de contactos bloqueados.

Si una llamada se recibe de un contacto, la aplicación bloquea todas sus llamadas subsiguientes. Pero, si se recibe un mensaje SMS del contacto, la aplicación le solicita que seleccione la acción que se realizará con el mensaje entrante.

Si se recibe un mensaje SMS del contacto, la aplicación bloquea ese mensaje SMS y los subsiguientes de dicho contacto. Pero, si se recibe una llamada del contacto, la aplicación le solicita que seleccione la acción que se realizará con la llamada entrante.

Permitir siempre

La aplicación marca esta llamada (o mensaje SMS) del contacto como deseados. La aplicación agrega el número o el contacto que emite la llamada (o mensaje SMS) a la lista de contactos permitidos.

Si una llamada se recibe de un contacto, la aplicación permite todas sus llamadas subsiguientes. Pero, si se recibe un mensaje SMS del contacto, la aplicación le solicita que seleccione la acción que se realizará con el mensaje entrante.

Si se recibe un mensaje SMS del contacto, la aplicación permite ese mensaje SMS y los subsiguientes de dicho contacto. Pero, si se recibe una llamada del contacto, la aplicación le solicita que seleccione la acción que se realizará con la llamada entrante.

Omitir

La aplicación omite las llamadas (o mensajes SMS) de este contacto. La aplicación no agrega el número o el contacto que emite la llamada (o mensaje SMS) a ninguna de las listas.

Cuando se reciba la siguiente llamada entrante (o mensaje SMS) del contacto, la aplicación le solicita que seleccione la acción que se realizará con dicho evento.
