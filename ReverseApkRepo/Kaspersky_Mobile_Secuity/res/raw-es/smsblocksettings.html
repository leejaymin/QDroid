Bloqueo del dispositivo

Bloqueo del dispositivo le permite utilizar el comando SMS lock: <código> para bloquear su dispositivo remotamente y prevenir un acceso no autorizado a sus archivos (<código> es el código secreto para Kaspersky Mobile Security en su dispositivo).

Los comandos SMS enviados se facturan de acuerdo con el plan de precios del otro dispositivo.

Puede activar o desactivar remotamente el bloqueo del dispositivo.

Activar el bloqueo del dispositivo

Activar/desactivar el bloqueo remoto del dispositivo mediante un comando SMS.

Si la función está permitida en el dispositivo, cuando reciba el correspondiente comando SMS, la aplicación bloquea el dispositivo y contesta con una notificación por SMS.

Los mensajes SMS enviados se facturan de acuerdo con el plan de precios de su operador de servicios móviles.

Si la función está desactivada en el dispositivo, la aplicación no puede ejecutar el comando SMS para activar Bloqueo del dispositivo.

Texto en caso de bloqueo

Texto del mensaje que aparece en la pantalla cuando el aparato ha sido bloqueado. De forma predeterminada, aparece un mensaje genérico.
