Borrado de datos

La función Borrado de datos le permite eliminar remotamente información de su dispositivo mediante un comando SMS para que no caiga en las manos equivocadas. Puede eliminar información personal (por ejemplo, contactos, correspondencia o datos de cuentas de Google) y carpetas seleccionadas o borrar todos los datos, incluidos los de la tarjeta de memoria.

Para eliminar datos de su dispositivo puede enviar los siguientes comandos SMS:

  • wipe: <código>: Para eliminar datos específicos.
  • fullreset: <código>: Para eliminar todos los datos.

<código>: Este es el código secreto para Kaspersky Mobile Security en su dispositivo.

Los comandos SMS enviados se facturan de acuerdo con el plan de precios del otro dispositivo.

Puede activar o desactivar el borrado remoto tanto de datos específicos como de todos los datos de su dispositivo y seleccionar la información específica que se debe eliminar.

Eliminar los datos seleccionados

Activa/desactiva el borrado remoto de datos específicos desde el dispositivo (mediante un comando SMS). Puede seleccionar la información personal y las carpetas que se deben eliminar.

Si la función está permitida en el dispositivo, cuando reciba el correspondiente comando SMS, la aplicación elimina los datos seleccionados del dispositivo y contesta con una notificación por SMS.

Los mensajes SMS enviados se facturan de acuerdo con el plan de precios de su operador de servicios móviles.

Si la función está bloqueada en el dispositivo, la aplicación no puede ejecutar el comando SMS para eliminar datos.

Eliminar todos los datos

Activa/desactiva el borrado remoto de todos los datos desde el dispositivo mediante el comando SMS fullreset: <código> (donde <código> es el código secreto para Kaspersky Mobile Security en su dispositivo).

Si la función está permitida en el dispositivo, cuando reciba el correspondiente comando SMS, la aplicación lo ejecuta y contesta con una notificación por SMS.

Los mensajes SMS enviados se facturan de acuerdo con el plan de precios de su operador de servicios móviles.

Si la función está bloqueada en el dispositivo, la aplicación no puede ejecutar el comando SMS para eliminar todos los datos.

Datos personales que se deben eliminar

Se pueden seleccionar datos específicos para su borrado si la casilla Eliminar datos seleccionados está seleccionada.

Datos de identidad

Eliminación de contactos y entradas de la tarjeta SIM, SMS, historial de llamadas, calendario, parámetros de conexión Internet, cuentas de usuario (excepto cuentas de Google) cuando se reciba el comando SMS wipe: <código> (donde <código> es el código secreto para Kaspersky Mobile Security en su dispositivo).

Si la casilla está seleccionada, Kaspersky Mobile Security elimina sus datos personales al recibir el comando SMS. Si la casilla no está seleccionada, la aplicación no podrá eliminar sus datos personales mediante el comando SMS.
