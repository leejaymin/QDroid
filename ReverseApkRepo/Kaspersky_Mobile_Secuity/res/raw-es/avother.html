Configuración de Antivirus

El componente Antivirus protege su dispositivo contra virus y software malintencionado. Antivirus incluye los siguientes componentes: protección, análisis y actualización.

La función de protección se inicia al mismo tiempo que el sistema operativo y está siempre residente en la memoria del dispositivo. La función de protección evita en tiempo real que se infecte el dispositivo: analiza todos los archivos abiertos, guardados y ejecutados y todas las aplicaciones nuevas.

Puede ejecutar un análisis cuando desee para detectar y eliminar amenazas en su dispositivo (como al abrir archivos infectados, archivos sin abrir y archivos desinstalados). Kaspersky Mobile Security le permite llevar a cabo un análisis completo o parcial del contenido del dispositivo. Puede elegir analizar todo el sistema de archivos del dispositivo, el contenido de la memoria interna o una carpeta determinada.

Las actualizaciones de las bases de datos de virus de Kaspersky Mobile Security y de la versión de la aplicación (de ahora en adelante, "actualizaciones") garantizan que la protección de su dispositivo esté actualizada.

Kaspersky Mobile Security utiliza bases de datos de virus cuando realiza búsquedas de software malintencionado. Las bases de datos de virus de la aplicación contienen descripciones de todo el software malintencionado que se conoce actualmente y de los métodos para neutralizarlo, además de descripciones de otros objetos malintencionados no deseables. Por este motivo, se recomienda encarecidamente que actualice las bases de datos de virus de forma regular. Si se superan más de 15 días desde la actualización anterior, las bases de datos se consideran desfasadas. La protección será cada vez menos fiable.

Al igual que con las bases de datos de virus, Kaspersky Mobile Security le permite actualizar la versión de la aplicación. Las actualizaciones de la versión de la aplicación eliminan vulnerabilidades de Kaspersky Mobile Security, agregan nuevas funciones o mejoran las actuales.

Para usar la aplicación y llevar a cabo las actualizaciones regulares, se necesita una licencia activa. Si la licencia ha caducado, no se podrán realizar actualizaciones.

Kaspersky Mobile Security ejecuta las actualizaciones desde los servidores de actualizaciones de Kaspersky Lab (sitios Web especiales que contienen actualizaciones para todos los productos de Kaspersky Lab).

Modo de protección

La función de protección se inicia al mismo tiempo que el sistema operativo y está siempre residente en la memoria del dispositivo. Esta función permite analizar todos los archivos que se abran, guarden y ejecuten, así como todas las aplicaciones instaladas.

Modo de protección

Se trata del modo que utilizará la función de protección para realizar el seguimiento de los cambios en el sistema de archivos en segundo plano y para analizar objetos malintencionados. Están disponibles los siguientes modos:

  • Completo. Protección analiza en busca de objetos malintencionados cuando se realiza cualquier acción con ellos (por ejemplo, cuando se abren, ejecutan, copian, guardan, mueven o modifican). Protección analiza las aplicaciones nuevas inmediatamente después de su instalación.
  • Recomendado. Protección analiza la presencia de objetos malintencionados solo en las aplicaciones recién instaladas; no analiza otros archivos u otras acciones que requieran estas aplicaciones. Protección analiza los programas nuevos inmediatamente después de su instalación.
  • Desactivado. La función de protección está desactivada.

    Puede desactivar temporalmente la función de protección para reducir la carga del dispositivo cuando se ejecuten varias tareas al mismo tiempo.

    Los especialistas de Kaspersky Lab no recomiendan desactivar esta función, ya que podría provocar la infección de su equipo y la pérdida de datos.

    Si se desactiva, no afectará a la ejecución de otras tareas del componente Antivirus, como el análisis del sistema de archivos en busca de virus o la actualización de las bases de datos de virus de la aplicación.

Cloud Security Scan

Existe un análisis adicional de las aplicaciones recién instaladas que utiliza el servicio en la nube de Kaspersky Security Network (un servicio en línea especial de Kaspersky Lab que contiene información sobre la fiabilidad de los archivos, las aplicaciones y los recursos de Internet). El análisis mediante el servicio en la nube lo ayuda a proteger sus datos frente a amenazas nuevas y desconocidas.

Si se activa la casilla, Kaspersky Mobile Security realizará comprobaciones adicionales mediante el servicio en la nube de Kaspersky Security Network en las aplicaciones recién instaladas antes de que se inicien por primera vez. Si se desactiva la casilla, Kaspersky Mobile Security no realizará comprobaciones adicionales.

Este parámetro está disponible si el modo de protección está establecido en Recomendado o Completo.

Software publicitario,marcadores automáticos y otros 

Se ha añadido la detección de software publicitario y otras aplicaciones que podrían dañar el dispositivo. Esta categoría incluye las aplicaciones que intentan acceder a su información personal, enviar mensajes SMS o sus coordenadas.

Si la casilla está seleccionada, Kaspersky Mobile Security detecta dichas aplicaciones y le alerta. Si se desactiva esta casilla, Kaspersky Mobile Security omite las aplicaciones en esta categoría.

Configuración del componente

Puede configurar cada componente Antivirus de manera individual.

Configuración de la protección

Puede seleccionar los tipos de archivos que la aplicación analizará en busca de objetos malintencionados, así como especificar la acción que llevará a cabo la aplicación en los archivos cuando se detecten amenazas.

Configuración del análisis.

Puede especificar los tipos de archivos que se deben analizar. Además, puede definir qué acción se realiza con los objetos malintencionados (por ejemplo, desinfectar o eliminar) y configurar análisis planificados automáticos.
