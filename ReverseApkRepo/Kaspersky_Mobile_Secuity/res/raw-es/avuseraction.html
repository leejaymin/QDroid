Amenaza detectada

La aplicación solicita al usuario que realice una acción en los archivos cuando se detecta una amenaza.

La aplicación informa al usuario de las amenazas detectadas si de dan las siguientes circunstancias:

  • Si el componente Antivirus detecta una amenaza en una aplicación recién instalada durante las operaciones de la función de protección o durante el análisis del dispositivo.
  • Si el Antivirus detecta una amenaza en un archivo y el Antivirus tiene el valor de configuración Si falla la desinfección establecido en Preguntar (ventana principal de Kaspersky Mobile Security > Anti-Virus > Configuración > Configuración del análisis > Acción al detectar una amenaza).

Eliminar

La aplicación elimina el objeto infectado.

Omitir

La aplicación no ejecuta ninguna acción sobre el archivo infectado y no lo elimina del dispositivo.
