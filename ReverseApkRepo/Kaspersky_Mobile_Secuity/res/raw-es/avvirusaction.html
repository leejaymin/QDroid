Acción al detectar una amenaza

Puede seleccionar las acciones que la aplicación llevará a cabo en los archivos en los que se detecten amenazas.

Desinfección

La aplicación intenta desinfectar el archivo infectado que se ha detectado. Si la casilla está seleccionada, la aplicación intenta primero desinfectar el archivo infectado que se ha detectado. Si la aplicación no puede desinfectarlo, realizará la acción que se haya especificado en la sección Si falla la desinfección.

Si la casilla no está seleccionada y se detecta un archivo malicioso, la aplicación responde de inmediato con la acción definida en la sección Si falla la desinfección.

Si falla la desinfección

La acción que la aplicación llevará a cabo en el archivo infectado si falla la desinfección o si la casilla Desinfectar no está seleccionada. Puede seleccionar una de las acciones siguientes:

  • Preguntar: La aplicación le solicita que seleccione la acción que llevará a cabo en el archivo infectado.
  • Eliminar: La aplicación elimina el archivo infectado.
  • Omitir: La aplicación no lleva a cabo ninguna acción en el archivo infectado y no lo elimina del dispositivo.
