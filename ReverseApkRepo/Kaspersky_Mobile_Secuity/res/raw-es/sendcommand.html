Envío de comando

Puede ejecutar funciones Antirrobo de manera remota u ocultar la información desde cualquier dispositivo con Kaspersky Mobile Security instalado. Para ello, tiene que enviar un comando SMS especial.

El comando SMS se envía en un formato cifrado y contiene el código secreto de la aplicación instalada en el otro dispositivo. El otro dispositivo no reconoce la recepción del comando SMS.

La aplicación ejecuta el comando SMS recibido en el otro dispositivo si se cumplen las siguientes condiciones: la aplicación se ha verificado correctamente, el dispositivo tiene cobertura, Kaspersky Mobile Security se ha instalado como administrador del dispositivo y el inicio remoto de esta función está permitido en el dispositivo.

Los comandos SMS salientes se facturan al precio de su operador de servicios móviles.

Número del teléfono receptor del comando SMS

El número de teléfono del dispositivo al que se envía el comando SMS.

Código secreto para la aplicación en el dispositivo receptor del comando SMS

El código secreto para Kaspersky Mobile Security en el dispositivo que recibe el comando SMS.
