Introducción del código secreto

El código secreto se instala cuando se inicia la aplicación por primera vez. Se utiliza para prevenir un acceso no autorizado a la configuración de la aplicación y para bloquear o encontrar el dispositivo, eliminar información u ocultar remotamente archivos confidenciales.

Si ha olvidado su código secreto, puede recuperarlo.

El código secreto no se puede recuperar si se ha desactivado la función de recuperación del código secreto durante la configuración inicial de la aplicación.

Tareas estándar:

Recuperación del código secreto

  1. Abra Kaspersky Mobile Security > Menú > Recuperar código secreto.
  2. Copie la información de identificación del dispositivo.
  3. Presione Ir o abra el siguiente sitio Web http://mobile.kaspersky.com/recover-code en el dispositivo o en un equipo.
  4. Especifique la siguiente información en los campos adecuados:
    • La dirección de correo electrónico que especificó durante la configuración inicial de la aplicación para la recuperación del código secreto.
    • El código de identificación del dispositivo.
  5. Se enviará un correo electrónico con el código de recuperación a su dirección de correo electrónico.
  6. Abra Kaspersky Mobile Security > Menú > Introducir código de recuperación.
  7. Introduzca el código de recuperación que se le proporciona en el correo electrónico.
  8. Confirme el nuevo código secreto.
