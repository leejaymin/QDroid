Configuración del análisis

Para analizar el dispositivo cuando desee, puede seleccionar los tipos de archivos que se analizarán en busca de objetos maliciosos, activar la descompresión de los archivos, seleccionar la acción que se realizará cuando se detecte una amenaza y configurar el análisis planificado del sistema de archivos.

Cobertura del análisis

Puede seleccionar los tipos de archivos que la aplicación analizará en busca de objetos maliciosos.

Acción al detectar una amenaza

Puede seleccionar las acciones que la aplicación llevará a cabo en los archivos en los que se detecten amenazas.

Análisis planificados

Puede configurar una planificación para que se inicie automáticamente un análisis completo del dispositivo: elija la frecuencia con la que se iniciará el análisis, así como el día y la hora (opcional).

Para que la aplicación inicie el análisis automáticamente, el dispositivo debe estar encendido.

La aplicación iniciará el análisis planificado automático en segundo plano. Cuando se detecta un objeto malicioso, la aplicación lleva a cabo la acción seleccionada en la configuración del análisis.

Planificación

Frecuencia con la que se ejecuta un análisis completo del sistema de archivos. Están disponibles las siguientes opciones:

  • Cada semana: La aplicación inicia automáticamente el análisis una vez a la semana. En esta opción, puede elegir el día y la hora de inicio del análisis.
  • Cada día: La aplicación inicia automáticamente el análisis cada día a la hora especificada. En esta opción, puede elegir la hora de inicio del análisis.
  • Desactivado: La aplicación no realizará un análisis planificado automático. Esta opción permite iniciar un análisis en el momento.

Día de inicio

Día del análisis. Este parámetro estará disponible si se selecciona Cada semana en el parámetro Planificación.

Hora de inicio

Hora del análisis. Este parámetro estará disponible si se selecciona Cada día o Cada semana en el parámetro Planificación.
