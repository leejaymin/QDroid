Detector de SIM

Detector de SIM le ayuda a obtener el número de teléfono vigente de la tarjeta SIM en caso de que sea reemplazado y bloquear el dispositivo si la tarjeta SIM se reemplaza o el dispositivo se activa sin tarjeta SIM.

Puede obtener un nuevo número de teléfono en un mensaje que se envía al número de teléfono o a la dirección de correo electrónico especificados.

Para enviar el correo electrónico, la aplicación envía un SMS al número especializado del operador móvil ruso MTS. El coste de envío de un SMS a este número especializado depende del contrato que tenga con su operador. El operador ruso MTS envía a continuación el número actual del teléfono a su dirección electrónica.

Puede activar o desactivar Detector de SIM, especificar un número de teléfono o a una dirección de correo electrónico para recibir un nuevo número de teléfono si la tarjeta SIM se reemplaza o el dispositivo se activa sin tarjeta SIM.

Activar Detector de SIM

Permitir/bloquear las acciones seleccionadas si la tarjeta SIM se reemplaza o el dispositivo se activa sin ella (por ejemplo, enviar un número de teléfono nuevo por SMS o correo electrónico o bloquear el dispositivo).

Si la función está permitida en el dispositivo, la aplicación llevará a cabo las acciones seleccionadas si la tarjeta SIM se reemplaza o el dispositivo se activa sin tarjeta SIM.

Si se bloquea la función en el dispositivo, la aplicación no controla las acciones sobre la tarjeta SIM.

Enviar nuevo número

Puede especificar un número de teléfono y una dirección de correo electrónico para recibir un mensaje con el nuevo número de teléfono si la tarjeta SIM se reemplaza.

Para enviar el correo electrónico, la aplicación envía un SMS al número especializado del operador móvil ruso MTS. El coste de envío de un SMS a este número especializado depende del contrato que tenga con su operador. El operador ruso MTS envía a continuación el número actual del teléfono a su dirección electrónica.

Número de teléfono

El número al que la aplicación envía el mensajes SMS con el número del teléfono si la tarjeta SIM se reemplaza. Este número de teléfono debe comenzar por un dígito o por "+" y solo puede incluir dígitos.

Dirección de correo electrónico

La dirección de correo electrónico a la que la aplicación envía el mensajes con el número del teléfono si la tarjeta SIM se reemplaza.

Bloqueo del dispositivo

Puede activar el bloqueo automático del dispositivo en caso de que se reemplace la tarjeta SIM o el dispositivo se active sin ella.

Bloquear

Bloquea el aparato si se intenta encenderlo después de cambiar o incluso retirar la tarjeta SIM. Si la casilla está seleccionada, la aplicación bloquea el dispositivo si la tarjeta SIM se reemplaza o el dispositivo se activa sin ella. El código secreto de la aplicación que debe introducirse para desbloquear el aparato. Si no selecciona la casilla, la aplicación no bloquea el terminal tras cambiar la tarjeta SIM.

Texto en caso de bloqueo

Texto del mensaje que aparece en la pantalla cuando el aparato ha sido bloqueado. De forma predeterminada, aparece un mensaje genérico.
