Configuración de actualizaciones

Puede configurar una planificación para que se inicien automáticamente las actualizaciones: elija la frecuencia con la que se iniciarán, así como el día y la hora.

La aplicación únicamente realiza las actualizaciones automáticas planificadas cuando el dispositivo está encendido.

Planificación

Frecuencia con la que la aplicación ejecuta las actualizaciones de forma automática. Están disponibles las siguientes opciones:

  • Cada semana: La aplicación inicia automáticamente la actualización una vez a la semana. En esta opción, puede elegir el día y la hora de inicio de la actualización.
  • Cada día: La aplicación inicia la actualización cada día. En esta opción, puede elegir la hora de inicio de la actualización.
  • Desactivado: La aplicación no inicia automáticamente una actualización planificada. Esta opción le permite iniciar una actualización en el momento.

Día de inicio

Día de inicio de la actualización. Este parámetro estará disponible si se selecciona Cada semana en el parámetro Planificación.

Hora de inicio

Hora de inicio de la actualización. Este parámetro estará disponible si se selecciona Cada día o Cada semana en el parámetro Planificación.
