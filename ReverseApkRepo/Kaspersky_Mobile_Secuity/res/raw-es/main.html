Antivirus

El antivirus le permite detectar y neutralizar amenazas en su dispositivo mediante el uso de las bases de datos antivirus de la aplicación y del servicio en la nube de Kaspersky Security Network. Antivirus incluye los siguientes componentes: protección, análisis y actualización.

Protección detecta amenazas en archivos abiertos, analiza las nuevas aplicaciones y evita la infección del dispositivo en tiempo real.

La función Análisis ejecuta, a petición, un análisis de todo el sistema de archivos, de la memoria operativa o de las carpetas. Análisis completo analiza el sistema de archivos en busca de objetos malintencionados, Análisis de carpetas analiza una carpeta en particular. Análisis completo y Análisis de carpetas detectan amenazas en archivos que se han instalado pero no se han abierto todavía, así como amenazas en archivos que están abiertos. Analizar memoria detecta amenazas solamente en los archivos que están abiertos.

Mediante la función de actualización, puede descargar nuevas bases antivirus y nuevas versiones de la aplicación.

Tareas estándar:

Análisis completo del sistema de archivos del dispositivo

En la ventana principal de Kaspersky Mobile Security, presione Antivirus > Analizar > Análisis completo.

Análisis de carpetas

  1. En la ventana principal de Kaspersky Mobile Security, presione Antivirus > Analizar > Análisis de carpetas.
  2. Seleccione una carpeta para analizarla.

Análisis de memoria

  1. En la ventana principal de Kaspersky Mobile Security, presione Antivirus > Analizar > Análisis de la memoria.

Cambio del modo de protección

  1. En la ventana principal de Kaspersky Mobile Security, haga clic en Antivirus > Configuración.
  2. Seleccione el modo de protección. Para ello, seleccione un valor para el parámetro Modo de protección.

Análisis planificados

  1. En la ventana principal de Kaspersky Mobile Security, presione Antivirus > Configuración > Configuración del análisis.
  2. Seleccione la frecuencia del inicio de análisis. Para ello, seleccione un valor para el parámetro Planificación.
  3. Seleccione un día y una hora para el inicio de análisis. Para ello, seleccione los valores para los parámetros Día de inicio y Hora de inicio.

Actualizaciones de la base de datos antivirus y de la versión

En la ventana principal de Kaspersky Mobile Security, presione Antivirus > Actualizar.

Actualizaciones planificadas de la base de datos antivirus y de la versión

  1. En la ventana principal de Kaspersky Mobile Security, presione Antivirus > Configuración > Configuración de actualizaciones.
  2. Seleccione la frecuencia del inicio de actualizaciones. Para ello, seleccione un valor para el parámetro Planificación.
  3. Seleccione un día y una hora para el inicio de actualizaciones. Para ello, seleccione los valores para los parámetros Día de inicio y Hora de inicio.

Protección de privacidad

Protección de privacidad le permite ocultar los contactos confidenciales y todo lo relacionado con el historial de llamadas y la correspondencia SMS.

Los contactos confidenciales se guardan en la lista Contactos que se deben ocultar.

La aplicación puede ocultar la siguiente información de los contactos confidenciales:

  • Entradas de la libreta de teléfonos.
  • Mensajes SMS leídos, enviados y borradores.
  • Registros del historial de llamadas.
  • Llamadas entrantes y notificaciones sobre nuevos mensajes SMS.

Cuando la función Protección de privacidad está activada, la aplicación bloquea la señal y las notificaciones cuando recibe un mensaje SMS de un número privado y oculta el mensaje en la bandeja de entrada. La aplicación bloquea las llamadas entrantes desde un número privado y no muestra ninguna información en la pantalla indicando que se ha recibido una llamada. Por su parte, el remitente escucha el tono ocupado. Para visualizar las llamadas y mensajes SMS entrantes de un período en el que la función Protección de privacidad estaba activada, debe desactivarla. Si reactiva la función Protección de privacidad, su información confidencial quedará oculta de nuevo.

Puede activar la Protección de privacidad dentro de la aplicación o de forma remota enviando un comando SMS desde otro dispositivo móvil. Puede enviar un mensaje SMS estándar con un código especial o enviar un mensaje SMS desde Kaspersky Mobile Security si la aplicación está instalada en el otro dispositivo.

Puede permitir o bloquear el inicio remoto de la función Protección de privacidad en su dispositivo. Si el inicio remoto de Protección de privacidad está bloqueado, la función no se puede iniciar remotamente mediante un comando SMS.

Puede desactivar Protección de privacidad solo en la aplicación instalada en su dispositivo.

Tareas estándar:

Ocultación de información relacionada con los contactos

  1. En la ventana principal de Kaspersky Mobile Security, haga clic en Protección de privacidad.
  2. Cree una lista de contactos que se deben ocultar si no lo ha hecho ya. La lista contiene contactos cuya información desea ocultar. Para ello, presione Contactos que se deben ocultar > Agregar.
  3. Vuelva a la ventana principal de Kaspersky Mobile Security > Protección de privacidad y cambie el conmutador Ocultar información a Activado.

Inicio remoto de Protección de privacidad

Realice una de las siguientes acciones en otro dispositivo móvil:

  • Si el otro dispositivo tiene Kaspersky Mobile Security instalado, cree y envíe un comando SMS a su dispositivo.
  • Cree y envíe un mensaje SMS con el texto hide:<código>, donde <código> es el código secreto definido en su dispositivo.

Los mensajes SMS salientes se cobrarán según la tarificación establecida por el operador móvil del otro dispositivo.

Creación de una lista de contactos que se deben ocultar

  1. En la ventana principal de Kaspersky Mobile Security, presione Protección de privacidad > Contactos que se deben ocultar.
  2. Realizar una de las siguientes acciones:
    • Para agregar un contacto desde la libreta de teléfonos del dispositivo, presione Agregar > Contacto.
    • Para agregar un número de teléfono, presione Agregar > Número de teléfono.

Antirrobo

El componente Antirrobo ofrece protección contra el acceso no autorizado a la información almacenada en el dispositivo. Además, lo ayuda a localizar su dispositivo en caso de pérdida o robo.

Antirrobo le permite bloquear su dispositivo mediante un comando SMS desde otro dispositivo móvil, eliminar información almacenada en él o visualizar su ubicación en un mapa. Además, Antirrobo le permite bloquear el aparato si se reemplaza la tarjeta SIM o si se activa sin una.

Las funciones Antirrobo se inician mediante un comando SMS especial. Puede enviar un mensaje SMS estándar con un texto especial desde otro dispositivo móvil o si el dispositivo tiene Kaspersky Mobile Security instalado, puede enviar un mensaje SMS desde la aplicación. Para enviar un comando SMS, necesita saber el código secreto del Kaspersky Mobile Security instalado en su dispositivo.

Puede permitir o bloquear el inicio remoto de las funciones Antirrobo en su dispositivo. Si el inicio remoto de Antirrobo está bloqueado, la función no se puede iniciar mediante un comando SMS.

Tareas estándar:

Permitir el bloqueo remoto del dispositivo

  1. En la ventana principal de Kaspersky Mobile Security, presione Antirrobo > Bloqueo del dispositivo.
  2. Active la casilla Activar Bloqueo del dispositivo.
  3. Si lo desea, en la sección Texto en caso de bloqueo, escriba el texto que desee que se muestre en la pantalla del dispositivo bloqueado.

Permitir eliminación remota de datos

  1. En la ventana principal de Kaspersky Mobile Security, presione Antirrobo > Autoborrado.
  2. Seleccione la información que se eliminará cuando se reciba el comando remoto:
    • Para eliminar datos seleccionados remotamente, seleccione la casilla Eliminar datos seleccionados. Para eliminar datos personales (como contactos, correspondencia SMS o calendario), seleccione la casilla Datos personales. Para eliminar carpetas remotamente, presione Agregar carpeta y seleccione las carpetas que desee.
    • Para eliminar todos los datos remotamente y revertir el dispositivo a la configuración de fábrica, seleccione la casilla Eliminar todos los datos.

Permitir buscador remoto de la ubicación

  1. En la ventana principal de Kaspersky Mobile Security, presione Antirrobo > Localizar.
  2. Active la casilla Activar la función Localizar.
  3. Si fuera necesario, especifique la dirección de correo electrónico en la que desea recibir las coordenadas de su dispositivo.

Sustitución de la tarjeta SIM

  1. En la ventana principal de Kaspersky Mobile Security, presione Antirrobo > Detector de SIM.
  2. Active la casilla Activar Detector de SIM.
  3. Especifique un número de teléfono o una dirección de correo electrónico adicionales para recibir un nuevo número de teléfono por SMS o correo electrónico cuando se sustituya la tarjeta SIM.
  4. Si fuera necesario, seleccione la casilla Bloquear para bloquear el dispositivo cuando la tarjeta SIM se sustituya o cuando el dispositivo se inicie sin una. Especifique el texto del mensaje que se visualizará en la pantalla del dispositivo bloqueado.

Inicio remoto de Antirrobo desde otro dispositivo

Realice una de las siguientes acciones en otro dispositivo móvil:

  • Si el otro dispositivo tiene Kaspersky Mobile Security instalado, cree y envíe un comando SMS a su dispositivo. Para crear un comando SMS, necesita saber el código secreto del Kaspersky Mobile Security instalado en su dispositivo.
  • Envíe un mensaje SMS a su dispositivo con el siguiente texto especial:
    • lock: <código>: Para bloquear el dispositivo.
    • wipe: <código>: Para eliminar los datos personales seleccionados del dispositivo.
    • fullreset: <código>: Para eliminar todos los datos del dispositivo.
    • locate: <código>: Para localizar el dispositivo.

    <código>: Este es el código secreto para Kaspersky Mobile Security en su dispositivo.

Los mensajes SMS salientes se cobrarán según la tarificación establecida por el operador móvil del otro dispositivo.

Filtro de llamadas y SMS

El Filtro de llamadas y SMS le permite bloquear las llamadas y mensajes SMS entrantes no solicitados. La aplicación filtra las llamadas y mensajes SMS basándose en las listas de contactos permitidos y bloqueados y de acuerdo con el modo de filtro seleccionado.

Tareas estándar:

Bloqueo de llamadas y mensajes SMS entrantes no solicitados

  1. Seleccione los contactos cuyas llamadas y mensajes SMS desea bloquear o permitir. Para ello, seleccione el modo del filtro. En la ventana principal de Kaspersky Mobile Security, presione Filtro de llamadas y SMS > Modo: <modo actual del filtro>.
  2. Seleccione el valor de la configuración del modo Filtro de llamadas y SMS:
    • Para bloquear llamadas y mensajes SMS solamente de los contactos bloqueados y permitir llamadas y mensajes SMS de los demás contactos, seleccione Contactos bloqueados.
    • Para bloquear llamadas y mensajes SMS de todos los contactos y permitir llamadas y mensajes SMS solo de los contactos permitidos, seleccione Contactos permitidos.
    • Para bloquear llamadas y mensajes SMS de los contactos bloqueados y permitir llamadas y mensajes SMS de los contactos permitidos, seleccione Estándar.
  3. Si fuera necesario, active la configuración de filtrado adicional.

    Por ejemplo, si ha seleccionado los modos de filtrado Contactos permitidos o Estándar, puede permitir automáticamente llamadas y mensajes SMS de los contactos de la libreta de teléfonos del dispositivo sin tener en cuenta el modo de filtrado activado. Para ello, seleccione la casilla Permitir contactos.

    Si ha seleccionado los modos de filtrado Contactos bloqueados o Estándar, puede bloquear automáticamente los mensajes SMS de los números que contienen letras sin tener en cuenta el modo de filtrado activado. Para ello, seleccione la casilla Bloquear remitentes no numéricos.

  4. Dependiendo del modo seleccionado, cree la lista de contactos permitidos y bloqueados:
    • Para crear la lista de contactos bloqueados, vuelva a la ventana principal. En la ventana principal de Kaspersky Mobile Security > Filtro de llamadas y SMS > Contactos bloqueados.
    • Para crear la lista de contactos permitidos, vuelva a la ventana principal. En la ventana principal de Kaspersky Mobile Security > Filtro de llamadas y SMS > Contactos permitidos.

Agregación de un número a la lista blanca

  1. En la ventana principal de Kaspersky Mobile Security, haga clic en Filtro de llamadas y SMS > Contactos permitidos.
  2. Haga clic en Agregar.
  3. Seleccione el tipo de eventos que desea recibir del contacto.
  4. Introduzca el número de teléfono del contacto del que desea recibir información. Para ello, complete el campo Número de teléfono permitido.

    Para agregar un número de abonado utilice cualquiera de estos métodos: especifique manualmente el número en el campo de entrada, seleccione las llamadas de un abonado en el registro de llamadas, o seleccione el SMS de un abonado en la lista de mensajes entrantes. Para agregar un número desde la lista de llamadas o mensajes, haga clic en .

    El número de teléfono solo debe incluir caracteres alfanuméricos; puede comenzar por un dígito, una letra o ir precedido por el símbolo "+". En lugar de dígitos, también es posible utilizar los caracteres genéricos "*" ó "?" (donde "*" es cualquier secuencia de caracteres y "?" cualquier carácter único). Por ejemplo, si el número *1234? ha sido incluido en la lista blanca, el Filtro de llamadas y mensajes permite las llamadas o mensajes SMS de los números en los que cualquier carácter siga a la secuencia 1234.

  5. Si ha seleccionado SMS como un tipo de evento, introduzca una frase clave que indique que el mensaje SMS es deseado. Para ello, complete el campo Texto permitido. El Filtro de llamadas y SMS solo entrega un SMS cuando contiene esta frase clave y bloquea todos los demás SMS de este número.

    Si no desea recibir ningún SMS, con cualquier contenido desde este número, deje el campo Texto permitido en blanco para esta entrada.

Agregación de un número a la lista negra

  1. En la ventana principal de Kaspersky Mobile Security, haga clic en Filtro de llamadas y SMS > Contactos bloqueados.
  2. Haga clic en Agregar.
  3. Seleccione los tipos de avisos de este contacto que considera no deseado (spam).
  4. Introduzca el número de teléfono del contacto bloqueado. Para ello, complete el campo Número de teléfono bloqueado.

    Para agregar un número de abonado utilice cualquiera de estos métodos: especifique manualmente el número en el campo de entrada, seleccione las llamadas de un abonado en el registro de llamadas, o seleccione el SMS de un abonado en la lista de mensajes entrantes. Para agregar un número desde la lista de llamadas o mensajes, haga clic en .

    El número de teléfono solo debe incluir caracteres alfanuméricos; puede comenzar por un dígito, una letra o ir precedido por el símbolo "+". En lugar de dígitos, también es posible utilizar los caracteres genéricos "*" ó "?" (donde "*" es cualquier secuencia de caracteres y "?" cualquier carácter único). Por ejemplo, si el número *1234? ha sido incluido en la lista negra, el Filtro de llamadas y SMS bloquea las llamadas o SMS de un número con la secuencia 1234 seguida de cualquier carácter.

  5. Si ha seleccionado SMS como un tipo de evento, introduzca una frase clave que indique que el mensaje SMS es no deseado. Para ello, complete el campo Texto bloqueado. El Filtro de llamadas y SMS solo prohíbe un SMS cuando contiene esta frase clave y permite todos los demás SMS de este número.

    Si no desea recibir ningún SMS, con cualquier contenido, desde este número, deje el campo Texto bloqueado en blanco para esta entrada.

Protección Web

Protección Web bloquea sitios Web malintencionados que distribuyen código malintencionado y sitios web falsos (phising) que pueden robar sus datos confidenciales, como contraseñas de bancos en línea, sistemas de pago y subasta en línea, así como acceder a sus cuentas financieras.

Protección Web comprueba los sitios Web antes de que se abran mediante el uso del servicio en la nube de Kaspersky Security Network (un servicio en línea especial de Kaspersky Lab que contiene información sobre la fiabilidad de los archivos, las aplicaciones y los recursos de Internet).

Protección Web solo comprueba sitios Web en el navegador estándar de Android y no en otros navegadores.

Cuando se active, el componente Protección Web comprobará automáticamente cuál es el navegador predeterminado. Si Protección Web no es compatible con el navegador predeterminado actual, este le invita a cambiar al navegador estándar de Android.

Para garantizar que los sitios Web se comprueben continuamente, se recomienda establecer el navegador estándar de Android como navegador predeterminado.

Si no desea modificar el navegador predeterminado actual, inicie el navegador estándar de Android para utilizar sus datos personales en la red de forma segura.

Puede iniciar el navegador estándar de Android desde Kaspersky Mobile Security o desde el menú de aplicaciones del dispositivo. Si Protección Web está activado, bloqueará los sitios Web falsos antes de que se abran.

Tareas estándar:

Activación de Protección Web para la comprobación continua de sitios Web

  1. En la ventana principal de Kaspersky Mobile Security, presione Protección Web.
  2. Cambie el conmutador Activar Protección Web a Activado.
  3. Presione Cambiar navegador (este botón se muestra si el componente Protección Web está activado y no se pueden analizar los sitios Web en el navegador predeterminado actual).
  4. Siga los pasos del asistente de establecimiento de navegador predeterminado.

Uso de los datos personales con seguridad

En la ventana principal de Kaspersky Mobile Security, haga clic en Protección Web > Abrir navegador.
