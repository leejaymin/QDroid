Localizar

Localizar le permite utilizar el comando SMS locate: <código> para localizar su dispositivo de manera remota y recibir sus coordenadas geográficas en un mensaje SMS o de correo electrónico (<código> es el código secreto de Kaspersky Mobile Security en su dispositivo).

Para enviar el correo electrónico, la aplicación envía un SMS al número especializado del operador móvil ruso MTS. El coste de envío de un SMS a este número especializado depende del contrato que tenga con su operador. El operador ruso MTS envía entonces las coordenadas actuales del dispositivo a su dirección de correo electrónico.

La respuesta SMS se factura de acuerdo con el plan de precios de su operador móvil.

La aplicación puede localizar el dispositivo por medio del GPS o de las redes móviles.

Para proporcionar la ubicación mediante el GPS, debe permitir el uso del GPS en la configuración del dispositivo. El módulo GPS se activará cuando la aplicación reciba un comando SMS especial. Si la aplicación no puede encontrar las coordenadas del dispositivo mediante el sistema GPS, utiliza las redes móviles para determinarlas.

Puede permitir o bloquear la función de localizar el dispositivo remotamente y especificar una dirección de correo electrónico para recibir las coordenadas geográficas del dispositivo.

Activar la función Localizar

Permite/bloquea la localización remota del dispositivo mediante un comando SMS.

Si la función está permitida en el dispositivo, cuando reciba el correspondiente comando SMS, la aplicación envía una respuesta con las coordenadas del dispositivo por SMS al dispositivo que envío el comando y a la dirección de correo electrónico especificada.

Los mensajes SMS enviados se facturan de acuerdo con el plan de precios de su operador de servicios móviles.

Si la función está bloqueada en el dispositivo, la aplicación no puede ejecutar el comando para localizar el dispositivo.

Enviar coordenadas del dispositivo

De forma predeterminada, la aplicación envía las coordenadas del dispositivo en un SMS al número que envió el comando SMS. También puede especificar una dirección de correo electrónico para recibir la ubicación del dispositivo.

Dirección de correo electrónico

La dirección de correo electrónico a la que la aplicación debe enviar las coordenadas de su dispositivo.
