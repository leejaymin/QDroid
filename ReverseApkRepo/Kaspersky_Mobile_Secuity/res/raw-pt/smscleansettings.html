Limpeza de Dados

A Limpeza de Dados permite eliminar informações do dispositivo remotamente por comandos de SMS para que os seus dados não sejam acedidos indevidamente. Pode eliminar informações pessoais (por exemplo, contactos, correspondência, dados da conta Google) e pastas seleccionadas ou limpar todos os dados, incluindo informações no cartão de memória.

Para eliminar dados no dispositivo, é necessário enviar os seguintes comandos de SMS:

  • wipe: <code> – para eliminar dados específicos.
  • fullreset: <code> – para eliminar todos os dados.

<Code> – o código secreto do Kaspersky Mobile Security no dispositivo.

Os comandos de SMS enviados são cobrados de acordo com o tarifário do operador móvel.

Pode activar ou desactivar a eliminação remota de dados específicos ou de todos os dados no seu dispositivo, e seleccionar informações específicas para serem eliminadas.

Apagar dados seleccionados

Permite activar/desactivar a eliminação remota de dados específicos do dispositivo (por comando de SMS). Pode seleccionar informações pessoais e pastas específicas para serem eliminadas.

Se a função estiver activada no dispositivo, ao receber o comando de SMS correspondente, a aplicação elimina os dados seleccionados no dispositivo e envia uma mensagem SMS de notificação.

As mensagens SMS enviadas são cobradas de acordo com o tarifário do operador móvel.

Se a função estiver bloqueada no dispositivo, não é possível a aplicação executar o comando de SMS para eliminar dados.

Apagar todos os dados

Permite activar/desactivar a eliminação remota de todos os dados do dispositivo utilizando o comando de SMS fullreset: <code> (em que <code> corresponde ao código secreto do Kaspersky Mobile Security no seu dispositivo).

Se a função estiver activada no dispositivo, ao receber o comando de SMS correspondente, a aplicação executa o mesmo e envia uma mensagem SMS de notificação.

As mensagens SMS enviadas são cobradas de acordo com o tarifário do operador móvel.

Se a função estiver bloqueada no dispositivo, não é possível a aplicação executar o comando de SMS para eliminar todos os dados.

Dados pessoais para eliminação

Pode seleccionar informações específicas para serem eliminadas ao receber o comando de SMS wipe: <code> (em que <code> corresponde ao código secreto do Kaspersky Mobile Security no seu dispositivo).

Os dados específicos podem ser seleccionados para eliminação se a caixa de verificação Eliminar dados seleccionados estiver seleccionada.

Dados de identidade

Permite eliminar as entradas de Contactos e do cartão SIM, o histórico de mensagens SMS, o registo de chamadas, o calendário, as definições da ligação à Internet e as contas do utilizador (excepto contas Google) ao receber o comando de SMS wipe: <code> (em que <code> corresponde ao código secreto do Kaspersky Mobile Security no seu dispositivo).

Se a caixa de verificação estiver seleccionada, o Kaspersky Mobile Security elimina os seus dados pessoais ao receber o comando de SMS. Se a caixa de verificação não estiver seleccionada, a aplicação não elimina os dados pessoais por comando de SMS.
