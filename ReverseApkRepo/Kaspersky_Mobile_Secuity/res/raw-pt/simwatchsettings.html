Prot. Cartão SIM

A Prot. Cartão SIM ajuda a obter o número de telefone actual do cartão SIM, caso este seja substituído, e bloqueia o dispositivo se o cartão SIM for substituído ou se o dispositivo foi activado sem um cartão SIM.

Pode obter o novo número de telefone numa mensagem enviada para o número de telefone ou o endereço de e-mail que especificar.

Para enviar uma mensagem de e-mail, a aplicação envia uma mensagem SMS para o número dedicado do operador MTS (Rússia). O custo de envio da mensagem SMS para o número dedicado depende do tarifário do operador. O operador MTS (Rússia) envia em seguida o número de telefone actual para o seu e-mail.

Pode activar ou desactivar a Prot. Cartão SIM, especificar um número de telefone ou endereço de e-mail para receber um novo número de telefone, se o cartão SIM for substituído e activar o Bloqueio de Dispositivo.

Activar Prot. Cartão SIM

Pode permitir/bloquear as acções seleccionadas se o cartão SIM for substituído ou se o dispositivo for activado sem o mesmo (por exemplo, enviar o novo número de telefone por SMS ou e-mail ou bloquear o dispositivo).

Se a função for permitida no dispositivo, a aplicação efectua as acções seleccionadas, caso o cartão SIM seja substituído ou se o dispositivo for activado sem o mesmo.

Se a função estiver bloqueada no dispositivo, a aplicação não controla as acções do cartão SIM.

Enviar novo número

Pode especificar um número de telefone e um endereço de e-mail para receber uma mensagem com o número de telefone novo se o cartão SIM for substituído.

Para enviar uma mensagem de e-mail, a aplicação envia uma mensagem SMS para o número dedicado do operador MTS (Rússia). O custo de envio da mensagem SMS para o número dedicado depende do tarifário do operador. O operador MTS (Rússia) envia em seguida o número de telefone actual para o seu e-mail.

Número de telefone

O número para o qual a aplicação envia uma mensagem SMS com o número actual do telefone se o cartão SIM for substituído. Este número de telefone pode começar com um dígito ou com um “+” e tem de conter apenas dígitos.

Endereço de e-mail

O endereço de e-mail para o qual a aplicação envia uma mensagem com o número actual do telefone se o cartão SIM for substituído.

Bloquear o dispositivo

Pode activar o bloqueio automático do dispositivo se o cartão SIM for substituído ou se o dispositivo for activado sem o mesmo.

Bloquear

Bloqueia o dispositivo se o cartão SIM for substituído ou se o dispositivo for activado sem um cartão SIM. Se o caixa de verificação estiver seleccionada, a aplicação bloqueia o dispositivo se o cartão SIM for substituído ou se o dispositivo for activado sem um cartão SIM. Teria que ser introduzido o código secreto da aplicação para desbloquear o dispositivo. Se a caixa de selecção não estiver desmarcada, a aplicação não bloqueia o dispositivo se o cartão SIM for substituído.

Texto quando bloqueado

O texto apresentado no ecrã do dispositivo bloqueado. É apresentada uma mensagem padrão por defeito.
