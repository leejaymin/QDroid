Enviar um comando

Pode executar as função de Anti-Roubo remotamente ou ocultar informações confidenciais de qualquer dispositivo com o Kaspersky Mobile Security instalado. Para tal, é necessário enviar um comando de SMS especial.

O comando de SMS é enviado num formato encriptado e contém o código secreto da aplicação instalada no outro dispositivo. A recepção do comando de SMS não é identificada no outro dispositivo.

A aplicação executa o comando SMS recebido no outro dispositivo nas seguintes condições: a aplicação passou na verificação com êxito, o dispositivo consegue detectar um sinal de rede móvel, o Kaspersky Mobile Security foi instalado com a função de administrador do dispositivo e o início remoto desta função está activado no dispositivo.

Os comandos de SMS enviados são cobrados de acordo com o tarifário actual do operador móvel.

Número de telefone que recebe o comando de SMS

O número de telefone do dispositivo para onde o comando de SMS é enviado.

O código secreto da aplicação no dispositivo que recebe o comando de SMS

O código secreto do Kaspersky Mobile Security no dispositivo que recebe o comando de SMS.
