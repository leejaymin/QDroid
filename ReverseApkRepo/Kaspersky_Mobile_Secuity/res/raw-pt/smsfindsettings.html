Localizar

A função de localização permite utilizar o comando de SMS locate: <code> para localizar o dispositivo remotamente e receber as coordenadas geográficas numa mensagem SMS ou num e-mail (<code> é o código secreto do Kaspersky Mobile Security no seu dispositivo).

Para enviar uma mensagem de e-mail, a aplicação envia uma mensagem SMS para o número dedicado do operador MTS (Rússia). O custo de envio da mensagem SMS para o número dedicado depende do tarifário do operador. O operador MTS (Rússia) envia em seguida as coordenadas actuais para o seu e-mail.

A mensagem SMS de resposta será cobrada de acordo com o tarifário do operador móvel.

A aplicação consegue localizar o dispositivo através de GPS nas redes móveis.

Para fornecer posicionamento GPS, é necessário permitir a utilização de GPS nas definições do dispositivo. Quando a aplicação recebe um comando SMS especial, o módulo GPS será activado. Se a aplicação não identificar as coordenadas do dispositivo utilizando o sistema GPS, irá utilizar as redes móveis para determinar as coordenadas.

Pode activar ou bloquear a função de localizar o dispositivo remotamente e especificar um endereço de e-mail para receber as coordenadas geográficas do dispositivo.

Activar a função Localizar

Permite activar/desactivar a localização remota do dispositivo por comandos de SMS.

Se a função estiver activada no dispositivo, ao receber o comando de SMS correspondente, a aplicação envia uma resposta com as coordenadas do dispositivo por SMS para o dispositivo que enviou o comando e para o endereço de e-mail especificado.

As mensagens SMS enviadas são cobradas de acordo com o tarifário do operador móvel.

Se a função estiver bloqueada no dispositivo, não é possível a aplicação executar o comando para localizar o dispositivo.

Enviar as coordenadas do dispositivo

Por predefinição, a aplicação envia as coordenadas do dispositivo numa resposta por SMS para o número que enviou o comando de SMS. Adicionalmente, pode especificar um endereço de e-mail para receber a localização do dispositivo.

Endereço de e-mail

O endereço de e-mail para o qual a aplicação envia as coordenados do seu dispositivo.
