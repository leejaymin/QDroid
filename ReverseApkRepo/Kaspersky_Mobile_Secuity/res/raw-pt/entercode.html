Introduzir o código secreto

O código secreto é definido quando a aplicação é executada pela primeira vez. É utilizado para impedir o acesso não autorizado às definições da aplicação e para bloquear ou localizar o dispositivo, eliminar dados do mesmo ou ocultar informações confidenciais remotamente.

Caso se esqueça do código secreto, pode recuperá-lo.

Não é possível recuperar o código secreto se a função de recuperação do código secreto for desactivada durante a configuração inicial da aplicação.

Tarefas padrão:

Recuperação do código secreto

  1. Abra o Kaspersky Mobile Security > Menu > Recuperação do código secreto.
  2. Copie o código de identificação do dispositivo.
  3. Prima Ir para ou abra o website seguinte http://mobile.kaspersky.com/recover-code no dispositivo ou num computador.
  4. Introduza as seguintes informações nos respectivos campos:
    • O endereço de e-mail especificado durante a configuração inicial da aplicação para recuperação do código secreto
    • Código de identificação do dispositivo
  5. É enviado um e-mail com o código de recuperação para o endereço de e-mail do utilizador.
  6. Abra o Kaspersky Mobile Security > Menu > Introduzir o código secreto.
  7. Introduza o código de recuperação indicado no e-mail.
  8. Confirme o novo código secreto.
