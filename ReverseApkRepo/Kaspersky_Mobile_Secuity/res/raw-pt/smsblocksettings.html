Bloqueio do Dispositivo

O Bloqueio do dispositivo permite utilizar o comando de SMS lock: <code> para bloquear o dispositivo remotamente e impedir o acesso não autorizado aos seus dados (<code> é o código secreto do Kaspersky Mobile Security no dispositivo).

Os comandos de SMS enviados são cobrados de acordo com o tarifário do operador móvel.

Pode activar ou desactivar o bloqueio remoto do dispositivo.

Activar o Bloqueio do dispositivo

Pode activar/desactivar o bloqueio remoto do dispositivo por comandos de SMS.

Se a função estiver activada no dispositivo, ao receber o comando de SMS correspondente, a aplicação bloqueia o dispositivo e envia uma mensagem SMS de notificação.

As mensagens SMS enviadas são cobradas de acordo com o tarifário do operador móvel.

Se a função estiver desactivada no dispositivo, não é possível a aplicação executar o comando de SMS para activar o Bloqueio do dispositivo.

Texto quando bloqueado

O texto da mensagem apresentada no ecrã do dispositivo bloqueado. É apresentada uma mensagem padrão por defeito.
