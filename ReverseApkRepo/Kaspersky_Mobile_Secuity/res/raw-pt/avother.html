Definições de Antivírus

O Antivírus protege o seu dispositivo contra vírus e outros softwares maliciosos. O Antivírus inclui os componentes seguintes: protecção, verificação e actualização.

A Protecção fica activada quando o sistema operativo inicia e encontra-se sempre na memória do dispositivo. A Protecção impede, em tempo real, que o dispositivo seja infectado: verifica todos os ficheiros abertos, guardados e executados em todas as novas aplicações.

A verificação é iniciada a pedido para detectar e eliminar ameaças no dispositivo (tais como ficheiros infectados quando são abertos, ficheiros não abertos e ficheiros instalados em arquivos). O Kaspersky Mobile Security permite efectuar uma verificação completa ou parcial do conteúdo do dispositivo. Pode optar por verificar todo o sistema de ficheiros do dispositivo, o conteúdo da memória interna ou uma pasta específica.

As actualizações das bases de dados de antivírus do Kaspersky Mobile Security e da versão da aplicação (doravante, "actualizações") garantem que a protecção do dispositivo se mantém actualizada.

O Kaspersky Mobile Security utiliza as bases de dados de antivírus para pesquisar software malicioso. As bases de dados de antivírus da aplicação contêm descrições de todos os softwares maliciosos actualmente conhecidos e métodos para o tratamento dos mesmos, bem como descrições de outros objectos maliciosos indesejados. Por este motivo, é altamente recomendável actualizar as bases de dados de antivírus regularmente. Se tiverem passado mais de 15 dias desde a última actualização, as bases de dados serão consideradas desactualizadas. A Protecção será, então, menos fiável.

Para além das bases de dados de antivírus, o Kaspersky Mobile Security permite actualizar a versão da aplicação. As actualizações da versão da aplicação removem vulnerabilidades do Kaspersky Mobile Security e adicionam novas funções ou melhoram as funções existentes.

As actualizações regulares requerem uma licença activa para utilizar a aplicação. Se a licença tiver expirado, não será possível efectuar as actualizações.

O Kaspersky Mobile Security executa actualizações a partir dos servidores de actualização da Kaspersky Lab (sites especiais que contêm actualizações para todos os produtos da Kaspersky Lab).

Modo de protecção

A Protecção fica activada quando o sistema operativo inicia e encontra-se sempre na memória do dispositivo. A Protecção permite verificar todos os ficheiros que são abertos, guardados e iniciados, bem como todas as aplicações instaladas.

Modo de protecção

O modo utilizado pela Protecção para registar alterações no sistema de ficheiros, em modo de segundo plano, e verificar a existência de objectos maliciosos neste sistema. Estão disponíveis os seguintes modos:

  • Expandido. A Protecção verifica a presenta de objectos maliciosos quando qualquer acção é aplicada aos mesmos (por exemplo, ao abrir, executar, copiar, guardar, mover ou modificar). A Protecção verifica as novas aplicações logo depois de estas terem sido instaladas.
  • Recomendado. A Protecção verifica a presença de objectos maliciosos apenas nos novos programas instalados e não verifica outros ficheiros ou acções associadas aos mesmos. A Protecção verifica os novos programas logo após estes terem sido instalados.
  • Desactivado. A Protecção está desactivada.

    É possível desactivar temporariamente a Protecção para reduzir a carga no dispositivo ao carregar várias tarefas ao mesmo tempo.

    Os especialistas da Kaspersky Lab não recomendam desactivar a Protecção, dado que isto poderia resultar na infecção do computador e na perda de dados.

    Se a Protecção estiver desactivada, isso não irá afectar a execução de outras tarefas do Antivírus, como a verificação da existência de vírus no sistema de ficheiros ou a actualização das bases de dados de antivírus da aplicação.

Cloud Security Scan

Existe uma verificação adicional para aplicações recentemente instaladas, a partir do serviço cloud da Kaspersky Security Network (um serviço online especial da Kaspersky Lab que contém informações sobre a fiabilidade de ficheiros, aplicações e recursos da Internet). A verificação através do serviço cloud ajuda-o a proteger os dados de ameaças novas e desconhecidas.

Se a caixa for assinalada, o Kaspersky Mobile Security irá executar verificações adicionais das aplicações recentemente instaladas antes de estas serem iniciadas pela primeira vez, através do serviço cloud da Kaspersky Security Network. Se a caixa estiver desmarcada, o Kaspersky Mobile Security não irá executar as verificações adicionais.

Esta definição está disponível se o modo de Protecção estiver definido para Recomendado ou Expandido.

Adware, auto-dialers e outros 

Foi adicionada a função de detecção de adware e de outras aplicações que podem danificar o seu dispositivo. Esta categoria inclui as aplicações que tentam aceder aos dados pessoais, enviar mensagens SMS ou obter as coordenadas de localização.

Se a caixa de verificação estiver seleccionada, o Kaspersky Mobile Security detecta tais aplicações e alerta o utilizador quanto estas são detectadas. Se a caixa de verificação estiver desmarcada, o Kaspersky Mobile Security ignora as aplicações nesta categoria.

Definições de componentes

Pode configurar cada componente do Antivírus individualmente.

Definições de protecção

Pode seleccionar os tipos de ficheiros nos quais a aplicação verifica a existência de objectos maliciosos, bem como especificar a acção que será efectuada pela aplicação, nos ficheiros em que foram detectadas ameaças.

Definições de verificação.

Pode especificar os tipos de ficheiros que têm de ser verificados. Adicionalmente, pode definir uma acção a aplicar a objectos malicioso (por exemplo, desinfectar, ou eliminar), e configurar verificações agendadas automáticas.

Definições de actualização

Pode configurar actualizações agendadas automáticas.
