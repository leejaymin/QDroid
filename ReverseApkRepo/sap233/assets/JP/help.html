スマートロックをご利用頂き、誠に感謝しております。各機能の設定は下記の教程で説明します。

• 基本設定

  1. ロック状態

    2. スマートロック起動/停止 （アンロックパターンが設定されない場合、ロック機能を起動する時、アンロックパターンの設定が必要になります）。

  2. ロック遅延

    3. 一定のロック遅延時間以内に、保護されるアプリケーションは再びパターンロックアンロックを入力する必要がありません。

  3. 遅延時間

    4. ロック遅延の時間を設定します。

  4. 状態欄表示

    5. 状態欄でスマートロックの状態を表示するかどうかを設定します。タスクマネージャより強制に停止され、保護失効にならないようにするため、オンにするようにお勧めいたします。

  5. 状態欄表示透明

    6. 状態欄表示を透明にするかどうかを設定します。

• 安全設定

  1. ロックタイプ変更

    2. パターンロック、PINロック又はパスワードロックを選択します。

  2. アンロックパターンPIN又はパスワード変更

    3. アンロックパターンPIN又はパスワード変更。

  3. 見えるパターン使用（パターンロックを使用する時のみ見えます）

    4. パターンロックをアンロックする時にパターンを表示するかどうかを設定します。

  4. 振動フィードバック

    5. パターンロックを入力する時振動フィードバックが必要するかどうかを設定します。

  5. パスワードを忘れた

    6. パスワードを忘れた場合、アンロックパターンをリセットするため、アンロックリセット質問を設定します。

• 高級設定

  1. スクリーンロック壁紙

    2. 気に入ったスクリーンロック壁紙を選びます。

  2. スクリーンオフ時ロック

    3. 携帯スクリーンがオフする時、改めて保護されるアプリケーションをロックします。

  3. 自己保護

    4. システムプログラム管理及びandroid市場をロックするかどうかを選ぶことにより自己保護します（デフォルトとして全部ロックします）。
