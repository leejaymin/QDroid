Welcome to use Smart AppLock, this is the quick help for Smart AppLock setting

• Basic setting

  1. Lock Status

    2. Start/Stop the app lock (If you have not set the lock pattern, you need to set your pattern when you start the app lock).

  2. Delay Lock

    3. Protected unlocked apps do not require entering lock pattern for access during your defined "Delay Time".

  3. Delay Time

    4. Choose the relock delay time.

  4. Notifcation bar

    5. Whether to show the app lock status in notification bar. We suggest keeping it on to prevent Smart AppLock from being force-stopped by Task Manager.

  5. Notification Transparent

    6. Whether to make the notification bar icon transparent.

• Security setting

  1. Change lock Type

    2. Change your lock type to pattern, PIN or password.

  2. Change pattern, PIN or password

    3. Change your current pattern, PIN or password.

  3. Use visible pattern(Only visible when using pattern lock)

    4. Pattern visible or not when unlocking pattern lock.

  4. Vibration feedback

    5. Whether to use the vibration feedback when unlocking.

  5. Forgot Password

    6. Set the "authentication email" or "security question" to help you reset the pattern, PIN or password in case you forgot.

• Lock screen

  1. Lock Screen

    2. Choose your lock screen wallpaper.

  2. Screen OFF Lock

    3. Whether to re-lock when screen turns off. (only if Delay Lock is on).

  3. Self Protection

    4. Choose whether to lock the system app manager and android market in order for self protection. (All are locked by default)
