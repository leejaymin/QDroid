1. 팝업 플레이어

재생 화면에서 " 아이콘 "을 누르면 팝업 플레이 모드로 전환됩니다. 팝업 플레이 모드에서 더블탭으로 원래 재생화면으로 돌아갑니다.

2. HDMI 자막 출력 ( BETA )

일부 삼성 기기에서 외부 HDMI출력에서 자막 출력을 지원합니다. 홈 화면에서 "메뉴키"->"설정"->"HDMI출력"을 활성화 시켜주십시요.

3. 루비 태그 지원(BETA)

SMI자막에서 Ruby Tag를 지원합니다. 홈 화면에서 "메뉴키"->"설정"->"루비 태크 지원"을 활성화 시켜주십시요.

4. 기부 기능

홈 화면에서 "메뉴키"->"기부하기" 메뉴를 통하여 $2를 기부하실 수 있습니다. Diceplayer 개발에 도움을 주시고자 한다면 기부하여 주십시요. 지속적인 업데이트에 도움이 됩니다.
